# 나라카 밸리 — 개발 로드맵

> **비전:** 나라카(저승 컨셉카페) 세계관의 **진짜 스타듀밸리형** 게임 (농사·낚시·채광·관계). Godot · Steam/PC.
> **핵심 원칙:** ① 한 시스템을 100%로 끝내고 다음으로. ② **그레이박스 먼저, 에셋은 나중.** ③ 매 단계 "재밌나?"를 묻는 게이트.
> 상세 용어는 [CONTEXT.md](./CONTEXT.md), 핵심 결정은 [docs/adr/](./docs/adr/) 참조.

---

## Phase 1 — 그레이박스 재미검증 프로토타입 (목표: 약 8주 / 파트타임)

**가용 시간:** 평일 저녁 + 주말 = 주 ~20h. 아래 스프린트는 각 **약 2주** 분량.
**목표:** 회색 도형만으로 첫 수직 슬라이스(농사 코어 + 미호 + 14일)를 끝까지 굴려, **"재밌나?"를 그래픽 없이 검증.**
**게이트:** Phase 1 끝 = 직접 플레이테스트 → 재미 판정 → 통과 시 Phase 2, 실패 시 루프 재설계.
**파트타임 생존 규칙:** 매 스프린트 끝엔 **반드시 "직접 플레이되는 무언가"**가 남게 한다. 진척이 눈에 보여야 동력이 유지된다.

### Sprint 1 (~2주) — Godot 학습 + 기술 스파이크  ⚠️ 최대 리스크 구간
- [ ] Godot 설치, 기본 튜토리얼 1개 완주 (GDScript 감 잡기)
- [ ] 내부 해상도 320×180 정수배 스케일 뷰포트 설정 (ADR-0003)
- [ ] 캐릭터 그레이박스 이동 (탑다운 4방향) + 충돌
- [ ] 16×16 타일맵으로 밭/집/카페 더미 맵 1개
- [ ] 시간 흐름(하루 사이클) 회색 UI로 표시 → 취침 시 다음 날
- **체크포인트:** "Godot으로 이게 되는구나" 확신 서면 통과. 안 서면 일정 재조정.

### Sprint 2 (~2주) — 농사 코어 + 하루/혼력 + 세이브
- [ ] 밭 칸: 괭이질 → 심기 → 물주기 → 성장(일수) → 수확 (플레이스홀더)
- [ ] 작물 3종 데이터: 혼령초(빠름)/피안화(중간)/영혼 호박(느림) — 성장일수만 다르게
- [ ] 혼력(에너지): 행동당 고정 소모, 취침 시 풀 회복
- [ ] 세이브/로드 (Godot 리소스 직렬화)
- **체크포인트:** "심고-키우고-거두는" 한 사이클이 돌아간다.

### Sprint 3 (~2주) — 경제 + 미호 + 관계 루프
- [ ] 카페 출하대: 수확물 → 골드 → 씨앗 재구매 (작은 순환 폐쇄)
- [ ] 미호 NPC 배치 + 대화 시스템(텍스트 박스)
- [ ] 호감도(하트): 일일 대화 + 선물(영혼 호박 선호)
- [ ] 여우불 성장 촉진(A 방식): 호감도 단계↑ → 가속 폭/범위↑
- [ ] 수확 시 사연 한 줄(플레이버 텍스트) 표시
- **체크포인트:** "친해질수록 농사 잘 됨"의 이중 보상이 체감된다.

### Sprint 4 (~2주) — 온보딩 + 폴리시 + ★재미 게이트
- [ ] 온보딩: 도착 → 옥자 통보 → 미호 멘토 → 튜토리얼 → 첫 수확
- [ ] 14일 분량으로 묶어 시작~끝 플레이 가능하게
- [ ] 밸런싱 1차(작물 가격·성장일·혼력 소모) — 대충이라도 끝까지 플레이되게
- [ ] 버그 잡기 + 직접 플레이테스트
- **★ 재미 게이트:** 직접 14일 플레이 → "또 하고 싶나?" 정직하게 판정.

## Phase 2 — 에셋 입히기 (게이트 통과 후)

> Phase 1이 재밌다고 검증된 **다음에만** 시작. 순서: 도트 → 초상화 → 사운드.

- [ ] 라이선스 점검 ([docs/licensing-checklist.md](./docs/licensing-checklist.md)) 먼저 확인
- [ ] PixelLab/Retro Diffusion으로 인게임 도트(미호·플레이어·작물 단계별·타일셋) 생성 → Aseprite 보정
- [ ] 미드저니로 대화용 큰 초상화(미호·옥자) 생성
- [ ] BGM 1~2곡(Suno 상업 플랜) + SFX 에셋팩
- [ ] 그레이박스 → 실제 에셋 교체, 톤 일관성 정리

## Phase 3 — 확장 (코어가 검증되고 에셋이 붙은 뒤)

> "전부 동시에"가 아니라 하나씩. 각 캐릭터는 속죄 테마(ADR-0004)를 따른다.

- [ ] **낚시** — 삼도천 낚시 (망자의 혼·도깨비 물고기)
- [ ] **멜** — 카페 경제·회계 (도박/사채 → 역전)
- [ ] **바나** — 밤 경비·손님 응대 (주거침입/흡혈 → 역전)
- [ ] 카페 운영(가공·서빙) 본격화 — 키운 작물 → 메뉴
- [ ] 채광(업화 갱도), 전투(탈주 잡귀), 커스터마이징(팔레트+프리셋, ADR/커스터마이징 C→A)
- [ ] 계절·달력 확장

## 일정 메모
- **파트타임(주 ~20h) 기준 약 8주(2개월).** 처음 정한 "1개월"은 풀타임 추정이었음 → 현실화.
- **Godot 학습(Sprint 1)이 최대 리스크.** 막히면 전체가 밀리니, Sprint 1 체크포인트에서 정직하게 일정 재조정.
- 막힐 때 우선순위: **"끝까지 플레이되는 것" > "기능이 많은 것" > "예쁜 것".**
- **번아웃 방지가 1순위 리스크 관리.** 파트타임 장기전이므로 몰아치기보다 꾸준함. 막히는 날은 쉬어도 된다 — 멈추지만 않으면 된다.
