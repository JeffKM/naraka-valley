# godot-tutorial — T1.1 공식 튜토리얼 프로젝트

Godot 공식 **"Your first 2D game"(Dodge the Creeps)** 튜토리얼을 직접 진행한 학습용 프로젝트.
`art/`·`fonts/`는 공식 튜토리얼 다운로드 에셋(Kenney/Godot 제공)이다.

- 튜토리얼: https://docs.godotengine.org/en/stable/getting_started/first_2d_game/index.html
- 에디터로 열기: `godot --path godot-tutorial -e`

> ⚠️ **학습용 보관 전용.** 나라카 밸리 게임 본체가 아니다(본체는 Phase 1 Sprint 1에서 별도 프로젝트로 시작). 여기 에셋은 학습 목적의 튜토리얼 자료이며 게임 본체에 가져다 쓰지 않는다(ADR-0001). 개념 정리·그레이박스 재현은 [`../godot-sandbox/`](../godot-sandbox/)와 [`../docs/godot-learning-notes.md`](../docs/godot-learning-notes.md) 참조.
