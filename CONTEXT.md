# 나라카 밸리 (Naraka Valley) — 용어집

> 이 문서는 프로젝트의 **공유 언어(glossary)** 다. 구현 세부사항은 담지 않는다.
> 코드/기획 문서에서 쓰는 도메인 용어는 모두 여기서 정의된 뜻을 따른다.

## 세계관 (Setting)

**나라카(Naraka)**
: 저승·명부(冥府). 게임의 무대가 되는 사후 세계. 배경 자체는 지옥이지만 톤은 귀엽고 아기자기·코믹하다.

**나라카 컨셉카페 (Naraka Concept Cafe)**
: 나라카 안에 존재하는 카페. 옥자가 운영한다. **실재하는 오프라인 컨셉카페**가 원작 IP이며, 게임은 그 세계관·캐릭터를 옮긴 것.

**종신계약 (Eternal Contract)**
: 죄인이 나라카에 끌려와 카페 직원으로 영원히 일하게 되는 계약. 멜·바나·미호가 각자의 죄로 묶여 있다.

## 등장인물 (Characters)

**옥자 (Okja)**
: 카페의 **사장이자 마녀**. 죄인을 직접 잡아온다(철퇴·십자가 등 무기 사용). 검은 마녀모자 + 빨간 리본 + 안경.
: 성격은 **시크하지만 은근 챙기는** 타입(겉은 차갑고 속정 있음). 미호의 밝음과 대비된다. 첫 슬라이스에선 오프닝에서 종신계약을 통보하고 "밭 일부터"라며 떠넘기는 역할로 잠깐 등장.

**멜 (Mel)**
: **강시(僵尸)**. 죄목: 불법 도박 및 사채. 모티프: 돈자루, 부적, 강시 모자, 청록색 의상.

**바나 (Bana)**
: **뱀파이어**. 죄목: 주거침입 및 상해미수. 모티프: 박쥐·검은고양이 사역마, 고딕 드레스, 흡혈.

**미호 (Miho)**
: **구미호(九尾狐)**. 죄목: 방화 및 재물손괴. 모티프: 아홉 꼬리, 여우귀, 여우불(파란 불꽃), 노란 한복.
: 성격은 **밝고 긍정적·해맑고 여성스럽다.** 악의가 없으며, 생전에 여우불을 천진하게 다루다 **통제 못 해** 호박밭을 태운 것(고의 악행이 아닌 해맑음이 부른 사고). 나라카에서 그 불을 "태우는" 게 아니라 "**키우는**" 데 쓰는 법을 배워, **여우불로 작물 성장을 촉진**한다 — 파괴의 불 → 양육의 불. 첫 수직 슬라이스의 간판 캐릭터이자 농사 멘토.

**속죄 (Atonement) — 게임 관통 테마**
: 각 죄인 직원은 **생전의 죄의 힘을, 나라카에서 정반대로 써서 속죄한다.** 미호: 방화 → 작물 양육. 멜(도박·사채): 돈으로 망침 → 카페 경제·회계 담당(예정). 바나(주거침입·흡혈): 밤의 침입자 → 밤의 경비·손님 응대(예정). 각 캐릭터의 호감도 스토리가 곧 속죄 서사이며, 능력이 게임 시스템에 직결된다.

**여우불 성장 촉진 (Foxfire Growth)**
: 미호의 여우불로 작물 성장을 앞당기는 메카닉. 첫 수직 슬라이스의 나라카적 핵심 훅. **방식: 관계 보상형(A)** — 미호 호감도가 오를수록 여우불 도움이 강해진다(가속 폭·범위↑). 관계와 핵심 농사 루프가 한 몸이다. (불 양을 직접 조절하다 밭을 태우는 미니게임형(C)은 후속 확장 서랍에 보관.)

**손우앙 (Sonwoaeng)**
: 흰색 둥근 유령형 마스코트 캐릭터. (역할 추후 정의)

**플레이어 (Player)**
: 멜·바나·미호에 이어 새로 나라카에 끌려온 **4번째 죄인 직원**. 생전의 죄로 종신계약. "내 죄는 무엇이었나"가 메인 스토리 훅. 외형은 플레이어가 커스터마이징한다(범위 추후 정의). 기존 3인방과 동료이자 후배로 관계를 맺는다.

## 게임 시스템 (Game Systems)

**하루 사이클 (Day Cycle)**
: 스타듀밸리식 실시간 시간 흐름. 정해진 시각이 지나면 강제 취침, 취침 시 다음 날로 진행. "해 지기 전에 끝내야 한다"는 시계 압박이 핵심 긴장.

**혼력 (魂力, Soul Energy)**
: 에너지(스태미나)의 나라카판 명칭. 괭이질·물주기 등 행동마다 소모, 바닥나면 행동 불가. 취침 시 풀 회복. 첫 슬라이스에선 단순 규칙(행동당 고정 소모)으로 시작하고 밸런싱은 후속.

**온보딩 (Onboarding)**
: 첫 슬라이스의 도입 흐름 — 플레이어 도착 → 옥자가 종신계약 통보·밭 일 떠넘김 → 미호가 해맑게 농사 멘토로 합류 → 튜토리얼(괭이·심기·물주기) → 첫 수확 → 미호 여우불 도움 시작.

**호감도 (Affinity / 하트)**
: 미호와의 관계 수치. 올리는 법: **일일 대화**(하루 1회 소폭, 대사 변화) + **선물**(선호 작물/아이템, 예: 영혼 호박 선호). 하트 단계가 오를 때마다 ㉠ 여우불 도움 강화(A 방식), ㉡ 미호의 **속죄 서사 한 토막** 해금. 친해질수록 농사 효율↑ + 캐릭터 진짜 이야기↑의 이중 보상.

**저승 작물 (Underworld Crops)**
: 첫 슬라이스의 작물 3종. 성장 속도를 3단으로 나눠 리듬을 만든다.
: - **혼령초 (魂靈草)** — 푸르게 빛나는 영혼 풀. 빠름·저수익. 입문/튜토리얼 작물.
: - **피안화 (彼岸花)** — 저승을 상징하는 붉은 꽃. 중간 속도.
: - **영혼 호박 (Soul Pumpkin)** — 얼굴이 살짝 비치는 호박. 느림·고수익. 미호 호박밭 떡밥과 연결된 상징 작물.

**사연 한 줄 (Flavor Memory)**
: 작물 수확 시 그 영혼의 생전 사연 한 줄이 표시된다(예: "이 영혼은 생전에 핼러윈을 가장 좋아했대"). 텍스트 한 줄로 세계관 깊이를 더하는 저비용 장치.

## 아트 스타일 (Art Direction)

**인스타툰체**
: 플랫 컬러 + 두껍고 균일한 검은 윤곽선 + 치비(2~3등신) 비율 + 단순한 배경. SNS 웹툰 톤. 원작 IP의 룩이며, 컷신·홍보 일러스트가 이 톤을 따른다.

**인게임 도트체**
: 실제 플레이 화면(탑다운)에 쓰는 픽셀아트. 인스타툰체를 픽셀아트 특화 AI(PixelLab / Retro Diffusion)로 변환·생성하고 Aseprite로 손보정한 것. 게임 장르는 **진짜 스타듀밸리형(탑다운 픽셀 이동 + 농사·낚시·채광·관계)**.

## 에셋 파이프라인 (Asset Pipeline)

- **컨셉·컷신·홍보 일러스트** → 미드저니/제미나이 (큰 단일 일러스트가 강점)
- **인게임 스프라이트·타일셋·애니메이션** → PixelLab / Retro Diffusion (4방향 캐릭터, 격자 타일, 걷기 애니)
- **일관성 보정·레이어 분리·커스터마이징** → Aseprite (수작업)
- **자체 "도트화 스튜디오"는 만들지 않는다** (기성 도구가 이미 해결. 자체 제작은 야크 셰이빙).
- **BGM** → 생성형 음악(Suno 등 상업 플랜)으로 게임 고유 분위기 확보
- **효과음(SFX)** → 로열티프리 에셋팩 (괭이질·물소리·UI 등 흔한 소리는 팩이 압도적으로 빠름)
- 첫 슬라이스 사운드는 최소만 — BGM 1~2곡(낮 밭 / 밤·메뉴) + 기본 SFX 묶음. "분위기가 난다" 수준이면 충분.
- ⚠️ 모든 생성형 도구의 **상업적 사용·저작권 약관을 Steam 출시 전 반드시 확인** → `docs/licensing-checklist.md` 참조.

**플레이어 커스터마이징**
: 1차 버전은 **고정 외형 주인공**으로 출시(C). 코어 루프 검증 후 **팔레트 스왑 + 헤어/의상 프리셋**(A)으로 확장. 부위별 풀 커스터마이징은 하지 않는다(에셋 폭발).
