# Dear My Naraka — 용어집

> 이 문서는 프로젝트의 **공유 언어(glossary)** 다. 구현 세부사항은 담지 않는다.
> 코드/기획 문서에서 쓰는 도메인 용어는 모두 여기서 정의된 뜻을 따른다.

## 세계관 (Setting)

**Dear My Naraka (게임 타이틀)**
: 게임의 정식 명칭(2026-07-05 확정). 저승(나라카)에게·나의 나라카에게 보내는 편지 같은 어감으로, [속죄]·[해방]·"본다"의 정서(사랑하는 이를 마침내 바라봄)를 담는다. **옛 가제 "나라카 밸리(Naraka Valley)"는 폐기** — 문서에서 게임명을 가리킬 땐 이 이름을 쓴다. 로고 = 영문 주 "Dear My Naraka" + 한글 독음 "디어 마이 나라카". (리포명 `naraka-valley`·내부 경로는 히스토리로 유지.) ※단어 "**나라카**" 자체는 아래 세계관 용어로 계속 유효(게임명 ≠ 세계 지명).

**나라카(Naraka)**
: 저승·명부(冥府). 게임의 무대가 되는 사후 세계. 배경 자체는 지옥이지만 톤은 귀엽고 아기자기·코믹하다.

**나라카 컨셉카페 (Naraka Concept Cafe)**
: 나라카 안에 존재하는 카페. 옥자가 운영한다. **실재하는 오프라인 컨셉카페**가 원작 IP이며, 게임은 그 세계관·캐릭터를 옮긴 것.
: 단순 배경이 아니라 게임의 **운영 가능한 무대**다 — 플레이어가 직접 운영(손님 응대·서빙·체키)하는 루프('카페 운영')가 돌아가고, 직원(옥자·미호·멜·바나)이 출근해 모이며, 단골 손님이 드나든다. 현실 컨셉카페의 활동(체키·서빙·바)은 **저승 세계관 언어로 번역**해서 넣는다(날것으로 넣으면 판타지가 깨지므로). 컨셉카페와 저승 판타지는 새로 결합하는 둘이 아니라 **원작부터 한 덩어리**다.

**종신계약 (Eternal Contract)**
: 죄인이 나라카에 끌려와 카페 직원으로 영원히 일하게 되는 계약. 멜·바나·미호가 각자의 죄로 묶여 있다. 표면적으로 *영원*으로 통보되지만, 속죄 완성 시 그 **강제력(족쇄)이 풀릴 수 있다**([해방] 참조 — 단 풀려도 탈출이 아니라 머무름은 선택).

**해방 (Liberation) — 엔딩 모델**
: 종신계약은 *기본 형기(강제력=족쇄)*다. 플레이어가 [미결의 죄]를 스스로 깨달아 속죄를 완성하면 계약의 *강제*가 풀린다(족쇄 해제). 단 해방은 **저승 탈출이 아니라** — 플레이어는 사랑하는 이·카페를 떠나지 않고 *스스로 남기를 선택*한다("머무름은 선택"). ⚠️ **스토리 엔딩 ≠ 게임 엔딩**(룬 팩토리형): 메인 스토리(미결의 죄→옥자→해방)는 클라이맥스로 *해결*되지만, 농사·카페 명소 일구기([ADR-0009])·결혼 생활은 그 후로도 계속된다. 이로써 "해결되는 서사 + 끝없는 코지 목표"가 공존한다. ⚠️ **경로별 톤(2026-06-22 grill):** "본다"가 속죄라 **척추 해결(=플레이어 자신의 해방)은 결혼과 독립**이다. *옥자의* 해방·결합은 옥자 결혼 한정이고, *타인과 결혼해도 옥자는 (척추 해결 시) 마침내 보였기에 슬픔이 치유돼 축복*한다(**씁쓸한 축복**, 징벌 아님) — [서사 바이블 §6.4](./docs/design/narrative-bible.md). (이 모델은 ADR-0005의 '창발형·열린 결말'을 *해결되는 엔딩 + 결혼 분기*로 확장한다 — 새 ADR로 기록 예정.)

## 지리 (Geography / 세계 지도)

> 세계는 검증된 **단일 무대**(밭+집+카페)에서 **다중 구역**으로 확장된다 — **맵 확장**(낚시·채광·전투 앞에 끼우는 별도 Phase, [ADR-0015](./docs/adr/0015-multi-region-world-map-expansion.md)). 구조 = **분리 맵 + 가장자리/길 전환**(스타듀식). 아래는 각 구역의 *정체성*만 정의한다 — 배치·인접·전환은 [docs/design/world-map.md](./docs/design/world-map.md)가 다룬다. 상세 레이아웃은 거기서.

**안식 농원 (Anshik Homestead)**
: 플레이어의 집·밭이 있는 **홈베이스**(검증된 기존 무대에서 카페가 마을로 빠진 형태). 농사의 집 + **목축**([하늘 목장])의 집. 건물: 내 집(배우자 방 지원)·창고(저장)·축사. ⚠️ **공간 구조([ADR-0035](./docs/adr/0035-homestead-elevation-cliff-overgrown-redesign.md)):** 좌우 대칭 중앙 밭이 아니라 **비대칭 Overgrown 개간지**다 — 북동 저지대에 본가+창고 병렬(자재 동선)·바로 남쪽 앞마당에 [스타터 패치], **서쪽 1단 고지**(절벽 + 창고 옆 돌계단)에 [하늘 목장], 중앙-약간 서쪽에 [영혼빛 연못]. 저지 농사는 처음부터 가동, 고지는 도구 게이트 해금(개간 Slice 1). 카페(일터)는 마을, 집·농장은 여기 — 출근 동선이 생긴다.

**하늘 목장 (Sky Pasture)**
: 안식 농원 **서쪽 1단 고지대**의 [목축] 방목지(S1-1 콤팩트 재배치: 남단 고지 전체가 단일 방목 Zone, 절벽 perimeter + 계단 animal gate가 가축을 confine). **절벽 단면이 천연 울타리**(울타리 비용 면제) — 그 위에 **펜스·게이트는 옵션 빌드 콘텐츠**(계단 틈 봉인·구획·장식 수동 배치, 강제 아님). 초기엔 고지로 오르는 돌계단이 [업화석](곡괭이)·[석화 고목](도끼)으로 하드 게이팅돼 도달 불가 — 도구 업그레이드로 게이트를 부수고 올라가 해금하는 **롱테일 마일스톤 영토**([ADR-0035], 개간 Slice 1). "평평≠막힘"([ADR-0008](./docs/adr/0008-growth-model-relationship-multiplier.md)): 저지 농사·목축 토대는 막지 않고, *고지 방목 영토 확장*만 도구 티어로 게이트한다([ADR-0027](./docs/adr/0027-tool-tier-aoe-access-skill-efficiency.md)/[ADR-0033](./docs/adr/0033-foraging-discovery-loop-nonskill-quality-tree-work.md) 정합).

**영혼빛 연못 (Spirit-glow Pond)**
: 안식 농원 **중앙-약간 서쪽의 비정형 연못**. #60d8f0 영혼빛 윤슬이 일렁이는 **비주얼 중심점**(휑했던 중앙의 앵커). 기능 = **물뿌리개 수급처**(농사 물 보충) + **텐션 낚시 앵커**(스타듀 농장 연못 낚시 대응 — *안식 연못의 정식 낚시 스팟 편입은 Slice 3 낚시에서 결정*, [ADR-0030](./docs/adr/0030-fishing-relationship-neutral-dave-reel-fight.md)). 영혼빛(저채도 시안)은 세계 전역에서 *영혼 깃든 곳*을 짚는 액센트(tileset-ruleset.md §3.4b).

**나루 마을 (Naru Village) — 허브**
: 망자가 삼도천을 나룻배로 건너 닿는 **나루(나룻터)**를 둘러싸고 자라난 저승 마을. 카페가 자리한 명소이자 모든 길이 통과하는 허브. **내부를 강+다리로 동/서 분할**(펠리컨 강변로식 — 한 번 건너야 동편). **서편:** 카페(명소·옥자·미호·멜·바나 출근·서빙·밤 바·**축제=카페 이벤트 데이**, 별도 광장 없음) + 메인 캐릭터 집(미호·멜·바나). **동편:** 만물상(잡화)·혼백관(박물관) + **주민 집들**. ⚠️ **ADR-0005에서 개정**([ADR-0014](./docs/adr/0014-relationship-marriage-liberation.md)): 더는 "사귈 주민이 없는 곳"이 아니다 — 주민은 3티어다(T1 결혼 가능 풀 캐릭터 / T2 사귐만(점주, 퍼크) / T3 순수 단골). 단 **서사 조각·관계 활동 보상(당근)은 여전히 메인 4인 독점** — 주민 결혼은 넓이를, 메인 결혼·해방은 깊이를 맡는다.

**삼도천 (Sanzu River, 三途川)**
: 망자가 건너는 저승의 강. **강 낚시**(망자의 혼·도깨비 물고기)의 집([game-loops §2.2](./docs/design/game-loops.md)). 나루 마을과 붙어 있다(나룻터가 마을의 기원). 나룻터 곁에 **혼백관**(박물관 — 망자의 유품·기억 전시; 도착→유품 전시가 지리·서사로 정합).

**황천해 (Hwangcheonhae, 黃泉海)**
: 삼도천이 하구로 흘러드는 저승의 바다. **바다 낚시**의 집(강 낚시와 별개 어종·무대). 해변·부두에 **[생선가게]**([뱃사공]이 운영).

**뱃사공**
: 황천해 부두 **생선가게**를 운영하는 **T2 점주**(결혼 넓이 T1 아님 — 사귐·퍼크만, [네오] 만물상 선례와 동형). 삼도천을 건네는 사공(카론 결)이 저승 물길·물고기를 가장 잘 안다는 테마 — **모든 낚시 기어(낚싯대·미끼·태클) 판매 + 물고기 환전의 단일 창구**(삼도천 강·황천해 바다 어획 양쪽 취급, 스타듀 윌리 1:1 대응). 속죄-중립(직원이지 메인 4인 아님). 퍼크 = ♡ → 기어 할인(grill 2026-06-29). 이름·종(저승사자 계열/순수 망자 사공)은 서랍.
_Avoid_: 어부, 생선장수 (점주 위상은 '뱃사공'으로 통일)

**저승 숲 (Underworld Forest)**
: **채집**(foraging)의 집 — 절기·장소별 자생물을 줍는 **관계-중립 앰비언트 루프**(소유 메인 없음, 곱셈기 0; 낚시와 함께 은둔 활동파의 정직한 노동 루프 — 단 낚시=긴장형 ↔ 채집=*이완형 발견*, [ADR-0033](./docs/adr/0033-foraging-discovery-loop-nonskill-quality-tree-work.md)). **줍기=혼력 0**(혼력 안 먹는 유일 산출 루프), 나무 작업(벌목·수액)만 혼력 소모. 저승 숲 = **일반종 채집 본진**(통과형). 어귀에 **목공방**(집·농장 건물 업그레이드, 로빈 대응 — 나무 곁; 원목 공급).

**미혹의 숲 (Forest of Bewilderment)**
: 저승 숲보다 깊은 숨겨진 숲(비밀의 숲 대응) — **희소종 특수 채집**의 집. *진입은 자유*지만 **특수 채집지(희소종)는 도끼 티어로 접근**(메카닉 게이트, [ADR-0033]). 깊은 끝에 **마녀의 오두막 = 옥자 집**(*미결의 죄* 해결 후 열리는 스토리 게이트 — 옥자의 '궁극의 결혼 분기' 위상과 정합; 메카닉 게이트와 분리). "길 잃음"이 테마라 채집 스킬 [혼 감지]가 여기서 빛난다.

**혼 감지 (Soul Sensing)**
: 채집 [스킬]의 줍기 갈래 보상 — 레벨이 오를수록 근처/시야 밖 채집물과 희귀종을 인지한다(순수 *편의*, +가치 0; [ADR-0033]·[ADR-0019](./docs/adr/0019-physical-skill-relationship-multiplier-two-axis.md)의 "+%가치 배제" 정합). 미혹의 숲 '길 잃음' 테마를 메카닉으로 살리는 장치. 채집의 성장 손맛은 +양이 아니라 **+발견**으로 채널링된다.

**업화 갱도 (Hellfire Mine, 業火坑道)**
: 저승의 광산. **채광**(광물·자원·혼력)의 집이며, **채굴하며 [잡귀]와 싸우는 스타듀식 woven 무대**다 — 잡몹 전투가 층마다 섞여 채굴을 *방해하는 위험*으로 짜인다(이중 시계: 곡괭이=혼력 시계 ↔ 몹=체력 시계, [ADR-0011](./docs/adr/0011-combat-health-resource.md)). **깊이 = 스타듀 광산형**(영구 하강 + 엘리베이터 체크포인트, 도달 깊이 보존). **갱도 바닥 = [나락] 졸업 게이트**(채광 깊이로 나락 최초 해금). 입구에 **업화 대장간**(도구를 벼림·지오드 개봉, 클린트 대응)·**모험가 길드**(무기·전투 장비 판매+토벌, 말론 대응). ⚠️ **갱도 잡몹 전투 = 관계-중립 채광 곁다리**(전투 스킬 base XP만, 바나 도메인 아님 — 낚시와 대칭). 옛 "갱도 끝 전투 던전" 중간 층은 폐기, woven으로 개정([ADR-0031](./docs/adr/0031-mining-combat-woven-mine-graduation-naraku.md) — ADR-0015 #11 개정·ADR-0011 무대 프로필 확정).

**나락 (Naraku, 奈落) — 독립 전투 던전**
: 저승 최하 심연. **[잡귀]의 근원**(바나가 여기로 가서 막는 전투/보안의 무대 — ⚠️ [ADR-0021]로 개정 전엔 잡귀가 밤 바로 새어나왔으나, 이제 나락에 남고 바나가 근원에서 막는다). **강몹+보스 순수 전투 전용 구역(체력만), [업화 갱도] 졸업 무대**(폐기된 "끝 던전"의 역할을 승격 흡수 — 채광으로 갱도를 뚫고 내려가 나락으로 졸업, [ADR-0031]). **깊이 = 해골동굴형**(매 진입 리셋 + 무한/심층, 고위험-고보상 단판). **보스 = 깊이 마일스톤마다 디스크리트 관문 보스**(named, 보장 출현 — "최고 도달 보스" 마일스톤은 영구 보존 = 바나 매크로 진척 눈금, 보스 드랍 = 카페 프리미엄 공급). **나락만 바나 전투 도메인**(호감도·당근 — 갱도 관계-중립과 대비). 갱도와 같은 전투 엔진 공유([ADR-0011](./docs/adr/0011-combat-health-resource.md)).

## 등장인물 (Characters)

**옥자 (Okja)**
: 카페의 **사장이자 마녀**. 죄인을 직접 잡아온다(철퇴·십자가 등 무기 사용). 검은 마녀모자 + 빨간 리본 + 안경.
: 성격은 **시크하지만 은근 챙기는** 타입(겉은 차갑고 속정 있음). 미호의 밝음과 대비된다. 오프닝에서 종신계약을 통보하고 "밭 일부터"라며 떠넘긴 뒤, 이후로는 **카페에 상주하는 사장**으로 매일 카페를 지킨다(직원·플레이어가 매일 마주치는 존재). 단 옥자의 호감도/관계 트랙은 **미결의 죄를 해결한 *후에야* 열린다**([ADR-0014](./docs/adr/0014-relationship-marriage-liberation.md)) — 그 전까진 진척에 걸린 핵심 게이트 장면으로만 인연이 비친다(앵커는 옥자 하나, ADR-0005). 일반 하트 그라인드 대상이 아니라, **생전의 죄→속죄→화해→사랑**이 메인 스토리의 *궁극의 결혼 분기*다(가장 어려운 엔드게임).
: **메인 스토리의 앵커.** 플레이어가 생전에 죄지은 상대가 바로 옥자다(→ '미결의 죄'). 옥자는 그 인연을 알지만 **마녀로서 죄인을 잡는 직무**로 처리할 뿐, *사적 복수가 아니다.* 플레이어의 죄목을 봉인하고 안 알려주는 것도 원칙 때문 — 나라카에선 죄를 *스스로* 깨달아야 속죄가 되므로 말로 알려주면 속죄가 성립하지 않는다. "시크하지만 챙기는" 태도는 *놓지 못함*으로 재해석된다.
: ⚠️ **2026-06-22 시나리오 grill 구체화([그날 밤]·[ADR-0016](./docs/adr/0016-converging-night-ensemble.md)):** 생전엔 타인의 액운·저주를 삼키며 약을 짓던 **고독한 약사(무녀)**, 탁기로 죽어가던 플레이어의 **은인이자 연인**이었다. [그날 밤] 대화재에서 자기 존재·기억을 대가로 플레이어를 살리고 나라카 마녀로 귀속됐다. *놓지 못함*의 정체가 곧 이 사랑이다.

**멜 (Mel)**
: **강시(僵尸)**. 죄목: 불법 도박 및 사채. 모티프: 돈자루, 부적, 강시 모자, 청록색 의상.
: 속죄로 **돈·물자가 도는 카페를 굴린다**(돈으로 망침 → 환대로 돈을 돌림). ⚠️ **2026-06-24 개정([ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md)):** 멜은 카페 운영을 *전담*하지 않는다 — 옥자·미호·바나와 함께 **카페+바 서빙 직원**이다([출하대]는 무인화). 멜의 관계 보상은 **카페에 통일**된다([ADR-0029](./docs/adr/0029-cafe-relationship-stage-domain-aligned-rewards.md), 2026-06-29 — 옛 카지노 split 폐기): *일일* = **서빙 마진**(팁·단가, 가볍게) / *매크로* = **카페 경제·매출·자금 확장**(돈을 돌리는 자→카페를 키울 자금, [카페 일구기] 게이트의 한 축). 아침 본거지 [카지노]는 *선택적 [명부의 운] 베팅 무대 + 속죄 플레이버*로 격하(멜 호감도에 안 매임). 카지노는 죄(도박)와 글자 그대로 닿으므로, *도박장=죄의 반복*(ADR-0004 충돌)을 피하려 **"속죄한 도박꾼이 *공정한* 하우스를 운영한다(등치지 않고 절제를 가르침)"** 방향으로 재단한다(→ [카지노], grill 2026-06-24 해소). 멜=옥자의 '기록(사실)' 조각([메인 스토리] 참조).

**바나 (Bana)**
: **뱀파이어**. 죄목: 주거침입 및 상해미수. 모티프: 박쥐·검은고양이 사역마, 고딕 드레스, 흡혈.
: 속죄로 **[나락]의 전투/보안을 담당**한다 — 남의 집을 침입하던 자가 이제 *나라카의 위협을 근원에서 막는 수호자*가 된다([ADR-0004] 거울: 침입자 → 수호자). ⚠️ **바나 도메인 = [나락] 한정([ADR-0031](./docs/adr/0031-mining-combat-woven-mine-graduation-naraku.md)):** [업화 갱도]의 잡몹 전투는 *관계-중립 채광 곁다리*라 바나가 손대지 않는다(바나 호감도·당근은 나락 본격 전투에서만 — "함께 칼 든 파티원" 결은 나락에만). ⚠️ **2026-06-24 개정([ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md)):** 기존 [밤 경비] 루프는 폐기됐고(전투 공간이 던전으로 따로 확보됨), 바나는 던전 전투 담당으로 옮겨졌다. 관계 보상은 *보호 곱셈기*가 아니라 도메인 일치 당근이다([호감도]·[ADR-0029](./docs/adr/0029-cafe-relationship-stage-domain-aligned-rewards.md)): *일일* = **전투 경험치 가속** / *매크로* = **던전([나락]) 진척**(깊이·접근·전투력 — 카페 일구기를 직접 게이트하지 *않는* 자기 도메인). 단 **나락 희귀 산출이 카페 최상위 프리미엄 메뉴·장식으로 흐르는 *공급*** 으로 카페와 엮인다(미호 농산물의 심연판). 밤엔 다른 셋처럼 *바 서빙*도 한다([나라카 바]). 바나=옥자의 '행동(목격)' 조각([메인 스토리]).

**미호 (Miho)**
: **구미호(九尾狐)**. 죄목: 방화 및 재물손괴. 모티프: 아홉 꼬리, 여우귀, 여우불(파란 불꽃), 노란 한복.
: 성격은 **밝고 긍정적·해맑고 여성스럽다.** 악의가 없으며, 생전에 여우불을 천진하게 다루다 **통제 못 해** 호박밭을 태운 것(고의 악행이 아닌 해맑음이 부른 사고). 나라카에서 그 불을 "태우는" 게 아니라 "**키우는**" 데 쓰는 법을 배워, **여우불로 작물 성장을 촉진**한다 — 파괴의 불 → 양육의 불. 첫 수직 슬라이스의 간판 캐릭터이자 농사 멘토. 농사가 본분이지만 **카페에도 출근**한다(직원이므로) — 시간대에 따라 밭↔카페를 오가며, 카페가 직원이 모이는 무대가 되게 한다.
: 옥자와는 사장-직원을 넘어 **친구 사이**다(존칭 없이 서로 '옥자'/이름으로 부르는 동등 관계 — '옥자 언니' 아님). 그래서 옥자의 숨긴 속정을 가장 가까이 알며, 이것이 ADR-0005에서 미호가 쥔 **'감정' 조각**의 근거다.

**속죄 (Atonement) — 게임 관통 테마**
: 각 죄인 직원은 **생전의 죄의 힘을, 나라카에서 정반대로 써서 속죄한다.** 미호: 방화 → 작물 양육. 멜(도박·사채): 돈으로 망침 → 카페 환대로 돈을 돌림(+카지노 = *공정한 하우스*, 서랍 — [멜] 참조). 바나(주거침입·흡혈): 침입자 → 나라카의 위협을 근원에서 막는 던전 수호자([ADR-0021]로 밤 경비→던전 전투 개정). 각 캐릭터의 호감도 스토리가 곧 속죄 서사이며, 능력이 게임 시스템에 직결된다.

**메인 스토리 (Main Story) — A 모델(창발형)**
: 나라카의 메인 스토리는 강제 줄거리가 아니라 **스타듀밸리식 창발형**이다. 실제 "이야기 체험"은 미호·멜·바나의 **호감도 단계별 속죄 서사**(관계 보상)에서 나오고, 그 위에 플레이어 본인의 미스터리('미결의 죄')가 멀리 깔린 떡밥으로 흐른다. 즉 **메인 스토리 = 캐릭터 속죄 서사들의 총합 + 플레이어의 미결의 죄.**
: 플레이어의 죄는 곧 **옥자와의 생전 인연**이며, 옥자가 앵커다. 속죄 플레이 방식은 **하이브리드** — 일상의 관계 루프(대화·선물·호감도)는 동료들(우선 미호)이 지고, 옥자와의 인연은 진척·호감도에 걸린 **몇 개의 핵심 게이트 장면**으로만 열린다(옥자 전용 풀 관계 트랙은 만들지 않는다).
: 인연 조각의 **출처 규칙(하이브리드):** 서사 조각은 *사람(캐릭터)* 에게서만 나온다(미호 → 이후 멜·바나). 낚시·채광 같은 *활동*은 서사 텍스트를 짊어지지 않고, 장면을 여는 **기계적 열쇠/게이트**로만 참여한다.
: 세 동료가 쥔 조각은 **종류가 다르다(각도만 확정, 내용은 서랍).** 각자의 속죄 능력(ADR-0004)에서 자연 도출 — **미호=옥자의 마음(감정**: 숨긴 속정을 읽는다), **멜=옥자의 기록(사실**: 장부·계약을 다루니 봉인된 죄목의 존재를 안다), **바나=옥자의 행동(목격**: 밤늦은 바에서까지 일하며 무방비한 옥자를 본다). 플레이어는 *감정+사실+목격*을 모아 같은 진실에 다가간다. 한 명에게 인연을 몰지 않는 이유이자, 셋 다 친해질 동기. (플레이어와 셋의 *개별* 인연은 두지 않는다 — 앵커는 옥자뿐, ADR-0005.)
: ⚠️ **운영 규칙:** 메인 서사 텍스트는 **캐릭터에만 붙인다.** 새 활동 시스템(낚시·채광·카페 등)은 **메인 플롯 글쓰기에 의존하지 않고 독립적으로 완성**한다("한 시스템 100% 끝내고 다음" 원칙 보호).

**미결의 죄 (Pending Charge)**
: 플레이어가 잡혀 있는 이유이자 메인 훅의 핵심. 플레이어는 옥자가 잡아온 **확정된 죄인**이라 다른 셋과 똑같이 종신계약으로 영원히 묶여 있다. 단 사건 파일에서 **'죄목' 칸 하나만 봉인**돼 있다 — "미결"은 *유무죄*가 아니라 *무슨 죄였는지*의 미결이다. 그 죄는 플레이어가 생전에 옥자에게 지은 **조용한 죄(외면·망각·부재 계열)** 이며, 플레이어가 **옥자를 알아보지 못하는 것 자체가 그 죄의 증상**이다(옥자는 플레이어를 알아본다). 봉인을 푸는 길은 플레이어가 *스스로* 깨닫는 것. ⚠️ **2026-06-22 시나리오 grill에서 구체화([ADR-0016](./docs/adr/0016-converging-night-ensemble.md)·[서사 바이블](./docs/design/narrative-bible.md)):** 옥자는 플레이어의 *은인이자 연인*(약사·무녀)이었고, 죄는 옥자가 깎아 살려낸 목숨을 *일에 묻혀 인연을 못 돌본 외면·부재*다. 강림 계약이 옥자 기억을 *삭제가 아니라 봉인(가라앉힘)* 했기에 못 알아봄이 성립하되([그날 밤]·[기억 봉인]), 그 봉인은 플레이어가 *스스로 안을 들여다봐야* 풀린다(못 알아봄=죄의 증상 유지). 미스터리는 *해결된다*(영구 모호 아님 — ADR-0014).

**그날 밤 (대화재)**
: 메인 서사가 수렴하는 **단 하나의 밤**([ADR-0016](./docs/adr/0016-converging-night-ensemble.md)). 옥자의 약방에서 일어난 대화재로, **전 캐스트(메인 4인 + T1 주민 11인)가 인과로 묶인다** — 루카(탁기 옮김)→옥자(치료·연인)→미르·스칼렛(빚 압박)→멜(습격)→미호+깨비(여우불·도깨비불 공명=화재)→바나(플레이어 방문 잠그고 도주)→켄(구조 실패)·세레나(목격)→옥자(불 속 희생)→강림(금기 계약)→모찌·네오(그 밤 탄생)→프로스티(여파의 공허에서 형상화). 상세 수렴 지도는 [서사 바이블](./docs/design/narrative-bible.md) §3.

**기억 봉인 (Sealed Memory)**
: 강림의 금기 계약이 플레이어를 살려 이승으로 돌려보내며 *옥자에 대한 진실을 플레이어 안에 가라앉힌* 것([ADR-0016](./docs/adr/0016-converging-night-ensemble.md)). **기억 *삭제*가 아니다** — 플레이어는 옥자를 품고 있으나 *스스로 들여다보지 않아* 닿지 못한다(ADR-0005 "기억상실 아님"의 정신 보존, 문구만 개정). 잔해 = 평생의 *원인 모를 공허*(→ [프로스티]). 정확히는 기억이 *뜯겨 나가 나라카에 보관*되며(옥자가 짊어짐) 게임 내내 접근 불가라 주관적으론 "지워진" 것과 같다. **봉인→귀환 모델:** 메인 4인의 [세 조각]([메인 스토리])과 공허로 플레이어가 진실을 *스스로 재구성*(속죄 성립)하면, **[해방] 순간 봉인이 풀려 기억이 돌아온다**(계약이 되돌려지므로) — 재구성(성숙한 선택의 사랑) 위에 귀환("아, 당신이었군요")의 카타르시스가 이중으로. **봉인의 법칙(키스톤 세계 규칙):** *봉인은 타인의 말로 풀리지 않는다 — 봉인된 본인이 스스로 닿아야만 풀린다.* 작법 관습이 아니라 *세계 물리*라, 옥자도 조연도, **전부 아는 강림조차 이 법칙에 묶여 말 못 한다**(강림은 ♡3 자기 파편 + B5 자각 후 *끄덕임* 확증까지만 — [코러스 모델]·서사 바이블 §5.2). 이로써 "스스로 깨달음"(ADR-0005)이 메카닉으로 정당화된다.

**코러스 모델 (Chorus Model) — 조연의 그날 밤 참여 규칙**
: [그날 밤]에 조연 11인이 *직접 묶이되*, 각자 **자기 파편·자기 죄책감만** 안다([ADR-0016](./docs/adr/0016-converging-night-ensemble.md)). **누구도 중심 진실(옥자가 *이 플레이어*를 위해 희생·봉인했다)을 말로 풀어주지 않는다.** 미스터리 *해결 열쇠*(세 조각)·*관계 활동 보상*·*해방-결혼*은 여전히 **메인 4인 독점**(ADR-0014 정신 보존, "조연=옥자 무관"만 개정). 조연은 세계를 깊게·테마를 예고하는 *라쇼몽식 증인*. 각 조연은 **2층** — ㉠ 생전 죄(저승에 온 이유, [마을 주민] residents.md 유지) + ㉡ 그날 밤 역할(연결). ⚠️ **개정(2026-06-22):** 조연은 순수 옵션이 아니라 ♡3 *그날 밤 고백*이 **클라이맥스 재구성(B5)의 필수 증거**다(11인 전원·로스터 고정) — 메인 *죄의 열쇠* + 조연 *그날 밤 증거* = 상호보완. 단 조연은 여전히 *중심 진실을 밝히지 않는다*(증거일 뿐, 해답은 플레이어 자각). → [서사 바이블 §6.2](./docs/design/narrative-bible.md).

**여우불 성장 촉진 (Foxfire Growth)**
: 미호의 여우불이 농사를 거들어 플레이어가 **농사 [스킬] 경험치를 더 빨리 쌓게** 하는 메카닉. 미호 호감도가 오를수록 여우불 도움이 강해진다(경험치 당근 차등). 여우불 픽션·연출은 유지하되, 메카닉은 *작물 타이머 가속*이 아니라 *스킬 경험치 가속*이다. ⚠️ **2026-06-24 모델 교체([ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md)):** 이전엔 *작물 성장가속 곱셈기*(ADR-0008)였으나, 관계가 스토리 필수 척추가 되며(ADR-0016/0017) 기계적 보상을 *가벼운 당근*으로 내려놓음. (불 양을 직접 조절하다 밭을 태우는 미니게임형은 후속 확장 서랍.)
: 당근은 **게이트가 아니라 가벼운 편의**다 — ♡0에서도 농사는 base rate로 굴러가고(은둔 활동파도 막히지 않음, "평평 ≠ 막힘"), 호감도는 그 위에 *조금 더 빠른 숙련*을 얹을 뿐(만렙에서 증발 — 그 시점의 관계 보상은 [호감도]의 ♡4 서사·♡5 연애·해방이다). 관계의 진짜 당김은 메카닉이 아니라 **서사**가 진다([ADR-0021]).

**미련까마귀 (저승 흉조)**
: 밤에 밭으로 날아와 **덜 여문 혼백 작물을 쪼아먹는 저승 흉조**. 작물에 남은 [미결의 죄]의 [미련]에 이끌린다는 결. 무방비 작물은 밤새 **영구 소실**한다(농사의 진짜 스테이크). ⚠️ **[잡귀]와 별개다** — 전투 대상이 아니라 [허수아비]로 *접근을 차단*하는 순수 농사 해충이며, 바나·[나락]·전투(체력)와 무관하다(밤 지표 위협 삭제 원칙 불침범). ⚠️ **2026-07-05 재도입([ADR-0051](./docs/adr/0051-scarecrow-crow-crop-defense-revive-cozy-bounded.md)):** 2026-06-24에 "코지 톤 충돌"로 기각됐던 위협을, 3중 안전장치(작물 문턱 발동·손실 상한·허수아비 자명 예방)로 *일회성 넛지*로 길들여 되살렸다.

**허수아비 (Scarecrow)**
: 밭 곁에 세워 반경 안 작물을 [미련까마귀]로부터 지키는 **농사 방어 인프라**. 미호/농사 도메인이며 서사는 없다(캐릭터가 아님, [ADR-0005]). ♡0에서도 만드는 **무게이트 base 크래프팅**(관계로 안 잠금, "평평 ≠ 막힘") — 미호 호감도 당근과 직교(친해진다고 까마귀가 덜 오지 않음, 허수아비가 유일한 답). **레어크로우** = 기능 동일한 저승 테마 **수집 스킨 8종**([혼백관] 전시·기존 서비스 분산 획득)으로, 8종 완성 시 **디럭스 허수아비**(반경 2배)를 해금한다.

**플레이어 (Player)**
: 멜·바나·미호에 이어 새로 나라카에 끌려온 **4번째 죄인 직원**. 생전의 죄로 종신계약. 기존 3인방과 동료이자 후배로 관계를 맺는다. 외형은 1차 고정(아트 디렉션 '플레이어 커스터마이징' 참조).
: **"내 죄는 무엇이었나"가 메인 스토리 훅.** 단, 플레이어는 **생전의 기억을 다 갖고 있다(기억상실 아님).** 미스터리는 *무엇을 했는지*가 아니라 *그게 죄였음을 본인이 인지 못 한다*는 것 — 방화·도박·주거침입 같은 큰소리 나는 죄가 아니라 외면·망각·부재 등 **스스로 잘못이라 느끼지 못한 조용한 죄**다. 다른 셋은 이미 자기 죄를 인정하고 속죄 중이지만, 플레이어는 **아직 자기 죄를 인정조차 못 한 유일한 한 명** — 14일과 그 이후를 살며 그 죄를 *스스로* 깨닫는 게 메인 여정이다. (→ '메인 스토리', '미결의 죄')

## 게임 시스템 (Game Systems)

**하루 사이클 (Day Cycle)**
: 스타듀밸리식 실시간 시간 흐름. 정해진 시각이 지나면 강제 취침, 취침 시 다음 날로 진행. "해 지기 전에 끝내야 한다"는 시계 압박이 핵심 긴장.

**명부의 운 (Daily Luck)**
: 매일 바뀌는 운 보정값. **명부(冥府)=운명·심판 세계**라 운/점술이 세계관에 깊이 정합(스타듀 일일 행운의 번역, grill 2026-06-24, [stardew-systems-catalog](./docs/design/stardew-systems-catalog.md)). **효과 = 전역·광범위**(스타듀식) — 갱도 사다리 발견 확률·광석/지오드 드롭·벌목 방향과 보너스 원목/씨앗·잡귀 전투 드롭·채집 산출 등 *게임 전반 수십 확률에 가산·감산*. ⚠️ **[저승 작물] 성장은 제외**(결정론적이어야 페이싱 보장). **예보 = 옥자 점술, [예보 매개체]로 전달** — 무녀·약사인 옥자가 캐스팅한 아침 점괘가 집(안식 농원)의 저승판 TV에 떠 그날 운의 *등급*을 명확히 읽어준다(나쁜 날엔 갱도 깊이 자제 등 *계획*을 세우게 — "예보는 명확, 내부 연산은 숨김"). [저승 날씨] 예보와 *같은 채널*(운=오늘 등급 + 날씨=내일 1일치). ⚠️ **게이트 아님**("평평 ≠ 막힘", [ADR-0008] #1) — 나쁜 날도 막히지 않고 *덜 유리할* 뿐. **멜 [카지노]**는 이 운을 끌어쓰는 *한 무대*일 뿐 거기 국한되지 않는다(옥자=운을 *읽기* / 멜=운을 *걸기* 분업 — 카지노 하우스 엣지 0, 그날 주사위는 명부의 운만 기울임). 보정원(럭키 런치 대응 음식·행운 반지·카지노 호감도 보너스)은 각 시스템 도입 시 연결([서랍]).

**저승 날씨 (Weather)**
: 매일 바뀌는 날씨(스타듀 날씨의 저승 번역, grill 2026-06-24). 지상 날씨(비·눈)를 날것으로 안 들이고 **[저승 절기]에 어울리게 저승화**한다(예: 유화절=불비/잿가루, 성야절=잿빛 눈, 피안절=안개 — 구체는 서랍). **절기별 날씨 풀에서 무작위**(절기 정체성 강화). **핵심 효과(저비용):** ㉠ **비 = 작물 자동 물주기** → 그날 물주기 [혼력] 0(공짜 날, [저승식 스프링클러]와 같은 결) ㉡ 낚시 어종 날씨 제약(어종 다양성) ㉢ 분위기(BGM·비주얼). ⚠️ **게이트 아님**("평평 ≠ 막힘") — 나쁜 날씨도 활동을 *막지* 않고 *바꿀* 뿐. **예보 = [명부의 운]과 *같은 채널***(옥자 점술이 집의 저승판 TV로 전달 — 운=오늘 등급 + 날씨=내일 1일치, 무녀라 통합). 내일 비면 오늘 물주기 스킵 등 농사 계획에 쓰임. 폭풍 자원(번개→배터리 대응) 등 추가 효과는 [서랍].

**저승 절기 (Underworld Calendar) — 계절 시스템**
: 스타듀의 4계절×28일 달력을 저승 세계관 언어로 *번역*한 것(Phase 3 도입 예정, 2026-06-24 grill 확정). ⚠️ **번역 원칙 정밀화(2026-06-24):** 봄여름가을겨울을 *날것 이름으로* 안 들이고 **절기 이름으로 번역**하되, **계절 구조엔 1:1 대응**한다(작물·날씨가 계절 결을 따름). 즉 화면엔 "3월"이 아니라 **"피안절 1일"** 만 보이고, *월(月)은 현실 카페 레퍼런스로만* 남는다("지상 계절 매핑 없음"→"구조 대응 O, 날것 이름 노출 X"로 부드럽게 개정). **각 절기 28일 = 1년 112일, 게임 시작 = 피안절 1일(봄):**
: - **피안절 (彼岸節) — 봄결.** 저편 기슭. [저승 작물] 피안화가 상징(글자 그대로 彼岸의 꽃, 봄꽃 정합·고정점).
: - **유화절 (流火節) — 여름결.** 흐르는 불(여우불·혼불·도깨비불 결)=더위. 혼령초가 어울리는 절기.
: - **망연절 (茫然節) — 가을결.** 아득함. ⚠️ **이중 한자 장치:** 평소 표면 한자는 **茫然(아득한 절기)**이나, **미결의 죄 클라이맥스에서 真名이 忘緣(잊은 인연)**임이 드러난다 — 매년 무심코 지나친 절기 이름이 [해방] 후 다시 읽히는 *환경적 떡밥*([기억 봉인] 봉인 법칙 "스스로 닿아야 풀린다"와 정합, 누구도 말로 풀어주지 않음).
: - **성야절 (星夜節) — 겨울결.** 길고 잿빛인 별밤. 영혼 호박(핼러윈 결)이 어울림([저승 날씨] 잿눈이 가장 잦음).
: 절기마다 자생 작물이 다르고, **절기 전환 시 그 절기 작물은 사멸**(계절을 *의미* 있게 만드는 장치 — 코지 처벌이지만 유지). ⚠️ **예외(grill 2026-06-29):** 고대열매식 **희소 다절기 작물 1~2종**은 절기를 넘어 생존(프레스티지 — "죽지 않는 혼의 작물"). 일반 작물은 사멸 유지라 의식은 보존되고, 다절기는 *쟁취하는 예외*다. ⚠️ **사멸 사전 경고(grill 2026-06-29):** 사멸은 *기습이 아니라 예고된 의식*이다 — 절기 마지막 날 [예보 매개체](저승판 TV)가 **"내일 절기 전환 → 절기 작물 사멸"** 을 띄워 마지막 수확·작별을 준비시킨다(새 UI 0, 기존 예보 채널에 D-1 플래그 한 줄). 죽음을 단일화하고 [저승 날씨]를 비살상으로 둔 의도가 *"죽음은 예측 가능해야 한다"* 이므로, 그 단 하나의 죽음을 기습으로 두면 모순(스타듀의 조용한 사멸과 의도적 분기). **절기당 28일 = 1년 112일 확정**(2026-06-24, 스타듀 정합 — 90일/365일 안은 농사·관계 페이싱 늘어짐으로 기각). 작물↔절기 매핑은 Phase 3 밸런싱 변수. 현재 14/21일 슬라이스는 *재미 게이트용*이었지 최종 길이가 아니며 풀 절기 달력으로 확장된다. [축제(카페 이벤트 데이)]는 이 절기와 직교하는 별도 층(아래).

**저승 날씨 (Underworld Weather) — 일일 변동 층**
: [저승 절기] *안에서* 하루하루 변동하는 날씨(2026-06-24 grill 확정, Phase 3+ 곁다리). 스타듀 날씨를 저승 언어로 *번역*한 것으로, **절기와 직교**한다(절기=무슨 작물이 자라나 / 날씨=오늘 하루가 어떤 날이냐).
: **구조 = 절기-가중 랜덤 + 1일 예보.** 매일 랜덤이되 *확률 분포가 절기마다 다르고*(절기 정체성), **하루치 예보**로 *내일*만 미리 보여줘 "내일 비 온댔으니 오늘 물주기 스킵"식 계획의 손맛을 준다(스타듀 비날의 핵심 재미). 예보 매개체는 [예보 매개체]로 확정(grill 2026-06-24).
: **강제 평온 못(grill 2026-06-29, 스타듀 관습 채택):** **절기 1일 = 항상 평온 + [카페 이벤트 데이] 당일 = 항상 평온.** 절기 전환 직후(작물 사멸 정리·새 파종)와 테마 데이 정체성에 날씨가 끼어들어 분위기를 깨지 않게 한다(스타듀가 시즌 1일·축제일을 강제 맑음으로 두는 것과 동형 — 단 스타듀 1년차 봄 초반 고정 패턴까지 복제하진 않음, 그건 빌드 튜닝값).
: **타입(각 날씨 = 유리해지는 루프 하나 = 루프 라우터):**
: - **평온** — 기본(맑음 대응). 효과 없음.
: - **혼우(魂雨)=비** — **농사 자동물주기 + 낚시 입질↑**(비=물주기는 [ADR-0021]상 미호=농사 XP로 바뀌어 충돌 없음). 피안·망연절에 잦음.
: - **잿눈(灰雪)=눈** — **카페 손님↑(추위 피난) + 야외(낚시·채집) 둔화 + 노지 작물 그날 성장 정지(물도 마름, 재물주기)**. ⚠️ **가혹-A: 잿눈은 작물을 *죽이지 않는다*** — 죽음은 [저승 절기] *전환* 전용으로 몰아둔다(죽음 출처 단일화). 예보로 대비 가능. 성야절에 잦음.
: - **혼불 바람(던전 날씨)** — 혼불이 심연([나락])에서 불어 올라 지상엔 *전조*로 보이되(예보에 뜸) 효과는 **던전에서 발현**(grill 2026-06-24 해소): 그날 갱도·[나락] 잡귀 **스폰 밀도↑·드랍/희귀드랍↑ = 양·기회 기반 '파밍 대목'**, 위험은 *물량發 체력 압박*이지 개별 난이도 스파이크 아님(후퇴 언제든 자유 — 코지·번아웃 방지). 지상 농사·낚시엔 기계 효과 0(으스스한 연출만), ⚠️ **밤 바·카페엔 무효**(잡귀 나락 한정 [ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md) — 옛 밤 약탈 모델 잔재 차단). 유화절에 잦음(daily luck 드랍 보정과 직교 스택). ⚠️ 정밀 위험-보상 곡선은 채광·전투 grill로 보류. Phase 3 전투와 함께 추가.
: **스코프 못:** Phase 3+ 곁다리라 기능 타입은 최소(평온·비·눈 3종)로 시작하고, 화려한 연출(파티클·BGM 변주)은 에셋 후순위("한 시스템 100%" 정신).

**예보 매개체 (Forecast Medium)**
: 집(안식 농원)의 **저승판 TV**(스타듀 TV 대응 — 정확한 기물 외형·명칭은 가벼운 서랍: 점괘 거울/혼백 라디오류). 무녀 옥자가 캐스팅한 아침 점괘가 여기 떠 **[명부의 운] 오늘 등급 + [저승 날씨] 내일 예보를 한 화면에** 보여준다(두 예보 채널 통합, grill 2026-06-24 — 혼백관 게시판·단순 UI 기각). ⚠️ **두 채널의 신뢰성은 의도적으로 다르다(grill 2026-06-29):** **[명부의 운] = 확률적 *등급* 힌트**(내부 굴림 결과는 숨김, 좋/평/흉만 읽어줌) vs **[저승 날씨] = 내일 100% 확정 예보**(스타듀 TV처럼 빗나가지 않음). 날씨가 틀리면 "내일 비 → 오늘 물주기 스킵" 계획이 *처벌*로 바뀌어 계획 손맛이 도박이 되므로 — 무녀의 날씨 예언은 확정, 운수 점괘만 확률. ⚠️ *집에서* 확인하므로 출발 전 계획 손맛 보존(옥자는 카페 상주라 대면 전달이면 계획 가치가 깨짐 — 옥자는 *캐스터*지 대면 전달자 아님). 스타듀 TV처럼 가벼운 아침 리추얼 기물(상호작용). 멜 [카지노] 호감도 ㉢(운 예보 정밀화)가 이 채널을 더 세밀히 읽어준다.

**카페 이벤트 데이 (Theme Day) — 축제 층**
: 스타듀의 *축제(festival)*에 대응하는 나라카 버전이자 **실재 나라카 컨셉카페의 시그니처 활동.** [저승 절기] *안의* 특정 날에 열리며, 그날 카페 장식 + 직원·플레이어 **의상** + 메뉴 + [체키]가 한 컨셉으로 맞춰진다(확정 4종: 메이드·바캉스·힙합·크리스마스 데이 — 아래 로스터). 절기와 **직교**한다(절기=무슨 작물이 자라나 / 이벤트 데이=오늘 카페가 무슨 컨셉이냐). ⚠️ **번역 원칙의 의도된 예외 — 컨셉은 (대체로) 날것으로 둔다:** 기본 메카닉(체키·서빙·바)은 저승 언어로 번역하지만, 테마 데이는 IP의 *알아볼 수 있는* 시그니처라 팝컬처 컨셉을 그대로 살린다(저승 색만 살짝). *저승 카페가 메이드 데이·항공 데이를 한다*는 부조화 자체가 "지옥인데 귀엽고 코믹한" 톤의 개그·매력(좀비 데이는 이미 저승과 찰떡). 무대는 나루 마을 서편 카페(별도 광장 없음).
: **발동 규칙(grill 2026-06-24):** ㉠ *발동* = 절기 달력 **고정**(예고 배너 → 기대감·리듬, 스타듀 축제의 마법). ㉡ *참여* = **옵트인·고인센티브**(강제 아님 — 스타듀식 맵 잠금 없이 카페 루프 위의 *기회 오버레이*, 운영하면 손님↑·테마 프리미엄 메뉴·체키 수요↑, 안 해도 손실 0 — [나라카 바] 옵트인 결·"평평≠막힘"). ㉢ *로스터* = **[카페 일구기] 진행으로 해금**(초반 소박 → 명소로 클수록 더 화려·더 많은 테마 데이; 테마 프리미엄 메뉴가 매출 증폭 — 테마 데이 = 카페 성장의 눈에 보이는 결실). ㉣ 테마↔절기 매핑은 **계절 결 기준 작가 배치**([저승 절기]가 계절에 대응하므로). ㉤ 테마 데이는 밤 [나라카 바]까지 **물들인다**(테마 바).
: **확정 로스터(2026-06-24, 절기당 1개 = 총 4):** 피안절(봄)=**메이드 데이**, 유화절(여름)=**바캉스 데이**, 망연절(가을)=**힙합 데이**, 성야절(겨울)=**크리스마스 데이**. 각 테마는 그 절기 **후반(≈25일경)** 고정 슬롯(예고 배너 → 절기 끝물 기대 리듬; "12월 크리스마스·3월 메이드" 같은 현실 월 표기의 게임판). **해금 순서 = 절기 순(메이드→바캉스→힙합→크리스마스), [카페 일구기] 마일스톤 게이팅** — 게임 시작이 피안절이라 **메이드 데이가 첫 해금·첫 체험**. ⚠️ **자산 천장:** 테마 1개 = [의상 모델]상 초상화 변형 ~5장(메인 4 + 플레이어)이라, 첫 테마는 그레이박스로 루프 검증 후 하나씩 확장("한 시스템 100%").
: **의상 모델(grill 2026-06-24):** 테마 의상의 본체는 **테마별 초상화 변형**(매력·디테일이 읽히는 곳 — ADR-0003/0018 분업의 확장, P2.4 초상화 채널 확장). 인게임 스프라이트는 **가벼운 테마 신호만** — 움직이는 플레이어·미호는 *모자/소품 레이어*, 상주 NPC(옥자·멜·바나)는 *통짜 스프라이트 스왑*(ADR-0018 애니 티어 결). **풀 4방향×프레임 의상 세트는 만들지 않는다**(에셋 폭발 회피 = [플레이어 커스터마이징] 풀 커스텀을 미룬 그 이유 보존). ⚠️ **플레이어는 테마 데이에 한해 초상화가 등장**한다(평소엔 인게임 초상화 없음 — 직원이라 테마 코스프레·[체키] 순간에 얼굴을 비춤). 테마 의상은 [플레이어 커스터마이징]과 *별개 축*(작가 배치 임시 오버레이 vs 플레이어 선택 지속 외형)이라, 풀 페이퍼돌 커스터마이징 시스템을 선행하지 않는다.

**절기 행사 (Seasonal Event) — 축제 2층(가벼운 층)**
: [카페 이벤트 데이]를 보완하는 **저자산 달력 이벤트 층**(grill 2026-06-29 ⑧ 재확인 패스 신설). ⚠️ **왜 필요한가:** 테마 데이는 자산이 무거워(테마당 초상화 ~5장) 절기당 1개로 캡되어 있는데, *28일 중 목적지 이벤트 1개*는 코지 리듬상 빈약하다. 스타듀 축제는 사실 여러 직무(경연·특수 상점·한정 포획)를 겸했고 나라카 테마 데이는 그중 *IP 시그니처(카페 컨셉)+개인 체키*만 가져갔다 — 나머지 직무를 **새 초상화·새 의상 0**의 가벼운 층으로 분리해 채운다(테마 데이 4개 캡과 [의상 모델] 자산 천장을 건드리지 않고 리듬만 복원). [저승 절기]와 직교(절기당 1행사, 테마 데이와 별개 슬롯).
: **정체성 = 오버레이 전용(잠금 베이스라인):** 절기 행사는 **이미 있는 활동 루프 위의 한정 오버레이**다 — *새 미니게임 엔진·새 장소·새 초상화 0*. [카페 이벤트 데이]가 카페 루프 위 오버레이인 것과 동형이다. ⚠️ **행사별 전용 미니게임 = 빌드 서랍(승급, 출시 비게이트):** 출시는 오버레이 전용 베이스라인으로 충분히 뜨고(엔진 0), 스타듀 에그헌트류 *전용 미니게임*은 나중에 폴리시로 행사 하나씩 올리는 **명시적 서랍**으로 둔다([ADR-0028] 서랍 철학 — "기능보다 완주 우선").
: **3아키타입 로스터(절기당 1행사, 4절기 분산):** ① **경연/더비** — [낚시]·[곳간] 위 랭킹 오버레이(밑활동이 절기마다 달라 자연 반복형: 곳간 출품 / 황천해 낚시 더비) ② **특수 상인/창구** — 임시 상인 오버레이([떠돌이 상인]·[집 꾸미기] 가구 세트 입수와 연결; 스타듀 *야시장*을 여기 흡수 — 행방 정리) ③ **한정 포획 창구** — 특정 시간·날씨·절기의 한정 스폰(월광 해파리류; 놓친 어종은 내년 회귀, [물고기 도감]은 비게이트 서랍이라 막힘 0). 정확한 절기↔행사 매핑·날짜는 **Phase 3 작가 배치/밸런싱 서랍**.
: **발동·참여·해금 규칙:** ㉠ *발동* = 절기 달력 **고정**(예고 배너 → 기대 리듬, 테마 데이와 공통). ㉡ *참여* = **옵트인 기회 오버레이**(맵 잠금 없음, 안 해도 게이트 0 — "평평≠막힘"; 단 경연 상품·한정 어종은 *그 절기 한정* = 놓치면 내년, **기회비용이지 처벌 아님**). ㉢ *해금* = **처음부터**(밑활동만 열려 있으면 1년차부터; ⚠️ 테마 데이와 **비대칭** — 테마 데이는 [카페 일구기] 게이팅 = 카페 성장의 결실, 절기 행사는 카페 이벤트가 아니라 *세계 절기 이벤트*라 카페 진척에 안 묶음).

**혼력 (魂力, Soul Energy)**
: 에너지(스태미나)의 나라카판 명칭. 괭이질·물주기 등 행동마다 소모, 바닥나면 행동 불가. 취침 시 풀 회복. 첫 슬라이스에선 단순 규칙(행동당 고정 소모)으로 시작하고 밸런싱은 후속.
: 혼력은 **노동(농사·후일 낚시·채광) 전용 제한**이다 — 스타듀가 도구질에만 스태미나를 물리듯. 카페 서빙 같은 응대·장사는 혼력을 쓰지 않고 **시간(손님이 오는 낮 창)** 으로 제한된다([ADR-0008](./docs/adr/0008-growth-model-relationship-multiplier.md)). 그래서 멀티루프에서 혼력 풀을 부풀릴 필요가 없다.
: 혼력은 **행동 예산**이다 — 내가 *무언가를 할 때* 능동적으로 깎인다. 적에게 맞아 깎이는 **체력**과 *종류*가 다르며, 이 구분이 "전투에 체력을 붙여도 혼력=노동 전용을 안 깨는" 빗장이다([체력] 참조, [ADR-0011](./docs/adr/0011-combat-health-resource.md)).

**체력 (體力, Health)**
: 전투 전용 **생존 자원**. 혼력과 *종류가 다르다* — 혼력은 *행동 예산*(노동할 때 능동 소모), 체력은 *피해 자원*(적에게 맞을 때 수동 소모). 칼질 자체엔 혼력도 체력도 행동비용으로 물리지 않는다(물리면 전투가 '노동'이 되어 ADR-0008 위반). Phase 3 전투 도입과 함께 들어온다([ADR-0011](./docs/adr/0011-combat-health-resource.md)).
: **회복:** 취침 시 풀 회복(혼력과 같은 `day_advanced` 결) + 낮 중 **음식 소비로 회복**(농사·채광 산출물이 흘러드는 소비처 → 루프 직조의 한 매듭).
: **바닥(0) = 기절:** 그 무대에서 강제 퇴장 + *그 시간만 날림*(골드·아이템·진행 손실 0 — 그레이박스·번아웃 방지). 혼력 0(행동 차단)보다 무겁되 영구 판돈은 없다.
: **무대별 자원 프로필:** 전투가 도는 곳에서만 존재한다 — **갱도(채광)** = 혼력+체력 동시(채굴은 노동→혼력, 잡귀 전투→체력, 후퇴 판단이 '이중 시계'), **[나락](전투 전용)** = 체력만(노동 없음). (⚠️ 2026-06-24: 옛 "밤 경비 = 체력만" 프로필은 [밤 경비] 폐기로 삭제 — [ADR-0021]·ADR-0011 헤더.)

**온보딩 (Onboarding)**
: 첫 슬라이스의 도입 흐름 — 플레이어 도착 → 옥자가 종신계약 통보·밭 일 떠넘김 → 미호가 해맑게 농사 멘토로 합류 → 튜토리얼(괭이·심기·물주기) → 첫 수확 → 미호 여우불 도움 시작.
: ⚠️ **농장 초기 상태 = 스타듀식 overgrown 개간(grill 2026-06-29):** 안식 농원은 *미리 갈린 밭*이 아니라 **오래 버려져 거칠어진 땅**으로 시작한다 — 작은 스타터 패치(완전 청소·즉시 경작)만 트여 있고, *(★이 "거칠어진 땅"의 시각 표현 = **흙-지배 지면 맨흙 ~72% + 잔디 패치 ~28%**, ADR-0053 — 봉인 법칙상 앰비언트 의도 전용·환경이 말하지 않음)* 나머지는 저승화 debris(**이승의 미련**=잡초 / **업화석**=돌 / **석화 고목**=그루터기 — ⚠️ "기억" 어휘는 [기억 봉인] 키스톤 전용이라 쓰지 않는다)가 덮고 있다. 플레이어가 [채집] 벌목(도끼)·[채광](곡괭이)·낫으로 *치우며 경작지를 넓힌다*(집 농장에 초반 벌목·채광 *목적* 부여). debris 드랍 = **기존 싱크로만**(원목→목공/제작·저승 광물→대장간·잡초→가벼운 채집/비료 재료 — *새 "연금" 범주 발명 금지*). ⚠️ **속죄 라임(앰비언트 한정):** 버려진 땅을 *돌봐 되살리는* 개간이 플레이어 [미결의 죄]=*외면·부재(neglect)*의 정반대(돌봄)를 *말 없이* 연습하는 것이며([속죄]·ADR-0004 미호 방화→양육의 거울), "안식(安息)"의 아이러니(안 쉬는 쉼의 땅 → 손길로 진짜 안식)가 이름 앵커. **단 활동-독립 규칙 보존**(CONTEXT 라인 85): 개간 메카닉은 *서사 텍스트 없이 자급자족*하고, neglect→돌봄 라임은 미호 *멘토 결*(캐릭터 텍스트 허용)로만 가볍게 입힐 뿐 [미결의 죄] 단서로는 *말하지 않는다*(봉인 법칙 — [해방] 후 다시 읽히는 환경 떡밥). ⚠️ **"평평≠막힘"([ADR-0008] #1):** 스타터 패치+0티어로 닿는 debris로 농사는 처음부터 완전히 돌아가고, 더 단단한 '원망의 바위·고목'은 *접근*이 아니라 *영토 확장*만 [도구 티어]로 막는다([ADR-0027]·[ADR-0033] 정합). 5×5 스타터·75% 밀도 등 수치는 Phase 3 밸런싱 변수.

**재점령 (再占領 / Encroachment)**
: 방치된 **빈 맨땅**을 [이승의 미련](잡초)이 다시 뒤덮는 현상([ADR-0055](./docs/adr/0055-homestead-reclamation-differential-respawn-weeds-cozy.md), 2026-07-07 grill). ⚠️ **[개간](Reclaim)의 반대 방향이다** — 개간은 플레이어가 debris를 치워 땅을 *되찾음*(일방향·구조물 영구)이고, 재점령은 잡초가 빈 땅을 *다시 차지함*이다(영문 글로스 = **encroachment**, "Reclamation"이 **아님**). ⚠️ **차등선 = `solid` 플래그:** non-solid 잡초(**이승의 미련**)만 재점령하고(매일 1~2칸/밤·빈 맨땅에만·[잿눈] 겨울 정지), solid debris(**업화석**·**석화 고목**)는 치우면 **영구**(성역 — 다시 마이닝·벌목 없음). **밭·심긴 작물·길·건물·회분은 성역**(작물은 재점령으로 **절대 죽지 않는다** — 스타듀와 분기, [ADR-0051] cozy bounded 톤). **일군 경작지는 영원히 깔끔**하고 되뒤덮이는 건 아직 안 다듬은 여백뿐. ⚠️ **테마(봉인 법칙 준수):** 구조적 개간 = *영구 진보*(막힌 땅 열기는 안 되돌아옴, [ADR-0053] 영구 회복이 여기 산다) / 잡초 재점령 = *앰비언트 돌봄 리듬*(속죄 = 한 번의 정산이 아니라 *매일의 돌봄* — [ADR-0004] 강화). 인게임에서 "저승 기운" 운운 금지·미호 멘토 대사 결로만.

**호감도 (Affinity / 하트)**
: 캐릭터와의 관계 수치(미호·멜·바나·옥자 + [마을 주민]까지 공유하는 일반 시스템). ⚠️ **보상 모델([ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md), 2026-06-24):** 관계의 *기계적* 보상은 **가벼운 호감도 당근**(초·중반 편의)이고, *진짜* 보상은 **서사(♡4 조각·♡5 연애·해방)** 다. 기계적 당근은 캐릭터마다 — **미호 = 농사 경험치 가속**([여우불 성장 촉진]), **바나 = 전투 경험치 가속**(스킬 트랙 있는 활동에만), **멜 = 서빙 마진**(팁·단가 ↑, *가볍게* — ADR-0008 마진 당근의 가벼운 복원). ⚠️ **당근은 *자기 도메인*에 둔다([ADR-0029](./docs/adr/0029-cafe-relationship-stage-domain-aligned-rewards.md), 2026-06-29 — 멜 split 제거):** *호감도 자라는 곳 = 보상 받는 곳* 일치(미호=농사·멜=카페 서빙·바나=던전·옥자=카페). 그래서 **멜 [카지노] 보너스 폐기**(카지노는 선택 베팅 무대로 격하). *매크로 당근*도 도메인별 — 미호=카페 메뉴·농산물 / 멜=카페 경제·매출 / 옥자=카페 정점 / 바나=**던전([나락]) 진척**(카페 아님). **[카페 일구기] 발전 = 카페-도메인 3인(옥자·미호·멜) affinity 분산 게이트**(⚠️ *매크로 목표만* 관계 요구 — *일일 활동은 ♡0 base rate 불변*, "평평≠막힘" 부분 한정). 당근은 만렙에서 증발하나 그 시점 보상은 연애·해방이라 무방하며, ♡0에서도 base rate로 멀쩡히 플레이된다("평평 ≠ 막힘", 관계 외면은 처벌 아님 — 당김은 메카닉이 아니라 서사가 짐). 하트가 차면 [결혼]으로 이어진다.
: ⚠️ **올리는 법은 메인/서브 이원화(2026-06-23 grill, [서사 바이블 §6.1](./docs/design/narrative-bible.md)):** **서브(T1 주민)** = 스타듀식 '쉼터' — 동선에서 마주쳐 **일일 대화 + 선호 선물**로 부드럽게(게임 내 성과 요구 0). **메인 3인(미호·멜·바나)** = **스타듀 선물 머신 + 활동, 두 채널 다 하트에 기여**(⚠️ grill 2026-06-29, [ADR-0017](./docs/adr/0017-main-affinity-activity-gate.md) 개정 — *IP 주인 요청*). 하트 미터는 스타듀식으로 **선물·일일 대화·활동 실적이 *모두* 채우며, 선호/혐오는 캐릭터 뒤의 *실제 인물*의 진짜 취향을 반영**한다(옛 "메인은 선물 아닌 활동" 폐기). ⚠️ **단 '활동→관계'는 폐기가 아니라 *이동*:** 활동은 더는 하트의 *유일* 동력이 아니지만, **속죄 서사 진급(♡ 단계 관문 컷신·[미결의 죄] 조각)은 그 캐릭터의 활동 deed 문턱을 추가로 요구**한다 — 선물로 하트는 차도 *미호 농사/멜 카페/바나 전투 deed* 없이는 속죄 비트가 안 열린다. 이로써 **"통합 플레이 = 속죄"(척추 = 농사·카페·전투 모두 요구, ADR-0017 ★)가 보존**된다("하트=애정은 선물로, 속죄 서사는 행동으로만"). 관계가 활동을 막진 않는다(♡0에서도 base rate, [ADR-0008] 정합). 또 **♡0~3은 점수가 차도 *특정 장소 대화/컷신*(단계 관문 이벤트 — 척추 비트·그날밤 고백의 그릇)을 봐야 진급**하고, **♡4~5는 *의지 시험*(특정 행동/상징 정표, [결혼] §6.3)으로** 열린다.
: ⚠️ **옥자는 활동 게이트의 예외(grill 2026-06-29):** 위 도메인표의 "옥자=카페"·"옥자 deed 모델 확장"은 *서사 위치*일 뿐, 옥자에겐 *게임 중* 플레이어-구동 호감도 미터가 **없다**(미호·멜·바나처럼 활동 실적으로 크는 트랙이 아니다). "옥자와 관계 쌓기" = 곧 **[미결의 죄] 미스터리 진척**(미호·멜·바나 ♡4 도달이 게이트, 옥자는 *보이지 않는 앵커*). 스타듀식 호감도 쌓기·선물·하트 그라인드·옥자 [결혼]은 **메인 스토리 클라이맥스(미스터리 해결) 이후 + *미혼 완주자*에게만** 열리는 궁극의 엔드게임 트랙(미혹의 숲 옥자 오두막). 클라이맥스 시점에 이미 미호·멜·바나 중 누군가와 결혼했으면 옥자 트랙은 닫히고 [해방]의 *씁쓸한 축복*으로 귀결된다(ADR-0014와 양립 — 옥자 결혼은 클라이맥스 선택지가 아니라 *미혼 완주자 전용 진엔딩성 분기*).

**결혼 (Marriage)**
: 호감도 최상단의 캡스톤([ADR-0014](./docs/adr/0014-relationship-marriage-liberation.md)). 대상: 메인(미호·멜·바나) + 옥자(미스터리 해결 후) + [마을 주민] T1. **메인 결혼 = 그들의 속죄를 완성해 종신계약에서 *해방*시키고 맺어지는 서사**([해방]·ADR-0004 정합 — damned soul을 사랑·속죄로 풀어준다). 주민 결혼은 *넓이*(스타듀형 선택지), 메인 결혼·해방은 *깊이*. ⚠️ **2단계(2026-06-22 grill·스타듀식):** ♡ 최상단 = *연애 개시*(즉시 결혼 아님) → *연애 후 추가 조건*(저승식 정표·집 업그레이드(배우자 방)·[메인 한정] 속죄 완성=해방) 충족 시 결혼. 메인·옥자·T1 주민 공통. 조건은 **티어별**(주민=정표+배우자 방 / 메인=+속죄 아크 완료=해방 / 옥자=+척추 해결·deed-track). **정표 = 옥자의 소울-바인딩 혼례 부적**(마녀=영혼을 묶는 자 → 족쇄 결박을 *사랑의 결합*으로; 마녀 서비스라 스파인 불가침. 단 옥자 본인 결혼은 강림이 맺어줌 — 강림 속죄 거울 완성). → [서사 바이블 §6.3](./docs/design/narrative-bible.md). 서브시스템 범위(약혼→배우자 이주)는 [서랍]. 자녀=[동행 혼](아래, 채택).
: ⚠️ **배타성·질투(grill 2026-06-29 — ADR-0014 서랍 해소):** **♡0~4 우정엔 배타성·질투 0**(척추가 미호·멜·바나 셋 다 ♡4를 *요구*하므로 불가침). 배타성은 **연애(♡5) 슬롯이 단 1개**라는 *구조*로 강제 — 정표는 한 명에게만, 두 번째 연애는 [이혼/결별] 후에만. **질투 = 카페 가족의 일회성 서사 반응**(메인들이 서로 친구·한 무대 출근·[그날 밤]으로 묶여 *연애를 안다* — 스타듀에 없는 디제시스 장치): ㉠연애 개시 순간 안 뽑힌 메인들의 **작고 회복되는 하트 찌릿함**(며칠 뒤 자동 회복 — 영구 손실·적대 0) + 일회성 "서운함" 비트(곧 응원으로 정착 = 옥자 *씁쓸한 축복*의 소형 미러) ㉡이미 연애 중 두 번째 정표 시도 = 침묵 아닌 *인-픽션 거절*. **일상 선물엔 질투 0**(선물 머신 보존 — 질투는 *로맨틱 정표*에만 반응). 스타듀 "다중 꽃다발→하트 폭락 그라인드"는 기각.
: **이혼([ADR-0022](./docs/adr/0022-divorce-liberation-irreversible.md)):** 전원(주민·메인·옥자) 가능하되 **[해방]·화해·진실은 불가역, 해소되는 건 *혼인(곁의 형태)*뿐**이다(서사 바이블 §6.4 "본다=속죄(불가역)/맺는다=선택" 정합). 메인은 *재결박 없이* 해방된 영혼으로 카페에 잔류한다. **푸는 주체 = 묶은 자의 거울**(메인 혼인=옥자가 부적을 풂 / 옥자 혼인=강림이 풂). 전배우자는 **적대 없음(씁쓸·우호적) + 호감도 0 리셋**, 재구애 가능하나 *이미 본 서사 조각·하트 이벤트는 재지급 없음*(기계적 당근만 정상). 재혼(메인)은 해방·배우자 방 면제라 *♡5 재도달+정표 재수여*만으로 가볍다. 비용 = 고액 골드. **Dark Shrine식 기억 삭제는 기각**([기억 봉인] 키스톤 보호). 자녀([동행 혼])는 이혼해도 *나라카에 깃든 존재라 잔류*(불가역·비극 없음, 가정/카페 가족 곁).

**동행 혼 (Companion Soul) — 자녀**
: [결혼] 후 부부의 [혼례 부적] 결속에서 안식 농원에 *깃들어 태어나는* 작은 새 혼(grill 2026-06-29 채택, ADR-0014 "자녀?" 서랍 해소). ⚠️ **생물학적 출산 아님** — [그날 밤]의 모찌·네오 "탄생"·프로스티 "형상화"와 같은 *혼의 형상화* 법칙의 **가정·소형판**이다(죽은 영혼이 아기를 낳지 않는다는 망자 설정 정합). 결속에서 깃들므로 **성별·생물학 무관**(동성 부부 포함 모든 부부 동일 — 스타듀 입양 분기 불요). 속죄 테마: *파괴*(그 밤의 불)에서 났던 모찌·네오의 대구 — 이번엔 *사랑의 결속*에서 깃드는 혼("파괴→양육"의 가족판). **스타듀 구조 차용:** 최대 1~2 혼, 아기→유아로 자라다 멈춤, 저상호작용 코지 존재(집 nursery + 가벼운 대사). 옵션 엔드게임 코지 롱테일(코어 아님). 룸메이트(아래)는 *동행 혼 없음*.

**룸메이트 (Roommate) — 비연애 동거**
: [결혼]을 원치 않는 플레이어를 위한 *플라토닉 동거* 분기(크로버스 대응, grill 2026-06-29 채택 — ADR-0014 서랍 해소). **대상 = 비인간 요괴 3인(모찌·네오·프로스티)만** — 표정만·인간화 ✕인 이들이 무연애 동행의 결에 맞는다(인간형 남5·여3 요괴는 결혼 전용). 같은 비인간 요괴를 **로맨스(결혼) OR 플라토닉 룸메이트로 플레이어가 택1**(♡ 트랙·반전 연출은 ♡4까지 공유하다 *서약 시점*에 갈림). 룸메이트 = *집 공유 + 동행*만 — **연애·[혼례 부적]·[동행 혼]·[해방] 없음**(혼례 결속이 아니므로, 크로버스가 자녀 없는 것과 동형). **가정 동거 슬롯은 단일** = 배우자 XOR 룸메이트(택1, [결혼] 배타성과 동형).

**목축 (Animal Husbandry)**
: 안식 농원 축사에서 **저승 가축(혼의 짐승)**을 기르는 새 루프. 농사(미호)와 같은 **양육 결**(방화→양육의 미호 테마 확장, 새 캐릭터 아님). ⚠️ **채택 확정(grill 2026-06-29):** 우정(0~5)·기분 → 산물 품질·대형 산물 확률 + 먹이(건초/사일로)·청결·야간 보호(스타듀 번역). 농사와 안 겹치게 **산물 = 카페 전용 고유 재료**(작물론 못 만드는 메뉴) + *매일 손 가는 돌봄 리듬*(작물=며칠 / 가축=매일)이 차별점. 타이밍은 로드맵 재배치에서 결정.

**집 꾸미기 (Home Decoration)**
: 플레이어가 **집(안식 농원) 내부**를 자유 배치로 꾸미는 코지 표현 시스템(grill 2026-06-29 채택 — 스타듀 최대 콘텐츠군 "가구·장식"의 나라카판). ⚠️ **범위 = 집 내부 한정:** *카페*는 플레이어가 안 꾸민다 — 카페 룩은 [카페 일구기](작가 배치 성장)·[테마 데이](그날 드레싱)가 짐(자유 배치를 카페에 열면 카페 일구기의 *명소로 자라는* 판타지와 충돌). 집 내부가 *진짜 빈 코지 캔버스*라 거기만 연다. 야외(농장 마당) 장식은 *같은 엔진의 자연 확장 서랍*(런치는 실내 우선; 기능성 울타리·길은 [목축]·농장 쪽 별 트랙).
: **메카닉 = 자유 그리드 3레이어** — 바닥재 + 벽지 + 가구를 칸 단위 자유 배치·회전(스타듀식 표현 캔버스; 슬롯 채우기 아님). 경계는 배치 제약이 아니라 *집 내부 평수*가 진다([집 업그레이드]로 평수↑=칸↑).
: ⚠️ **순수 코스메틱(버프 0·게이트 0):** 장식은 혼력·운·XP 등 *어떤 기능도 안 준다*([ADR-0019] "+가치 배제"·[ADR-0008] "평평≠막힘" 정합 — "잘 꾸민 집=더 강함"은 꾸미기를 *의무*로 만들어 코지를 죽임). 동기 = *자기표현 + 완주감 + **디제시스 반응*** (배우자([결혼])·[동행 혼]·방문 NPC가 집 상태에 *한 줄 반응* — 버프가 아니라 세계가 내 캔버스를 알아봐 주는 코지 만족, [체키 앨범]의 "기계 보상 0·감상" 결).
: **카탈로그 = "테마 세트" 모델(에셋 폭발 통제, [ADR-0001] 자체 창작):** 낱개 다수가 아니라 *일관된 저승 가구 세트* 단위로 만든다 — 세트 1개 = 전 카테고리(침대·탁자·의자·러그·벽장식·화분·조명)를 한 결로 커버해 그 세트만 깔아도 *완성된 룩*. 표현 다양성은 세트 간 조합으로(혼불 세트 + 피안화 바닥 믹스). 스타듀 *낱개 수(600+)*가 아니라 *응집된 세트 수*를 쫓는다. ⚠️ **세트 = 카탈로그 = 무한 배치:** 세트를 *해금*하면(골드 구매 마일스톤 / [카페 일구기] / [테마 데이] 짝) 그 세트 전체가 **무한·무료 배치 팔레트**로 열린다 — 해금된 세트 안에선 낱개 골드 비용 0(순수 캔버스, 표현 마찰 제거). 스타듀의 *단일 후반 카탈로그(20만 골드 벽)* 대신 *세트별 일찍·여러 번* 푸는 모델(자유 + 동기 드립 양립).
: **입수(저마찰·무제작게이트):** 세트 해금은 **[목공방] 가구 카탈로그(골드 — 로빈 대응, 건축+가구 판매 겸)**가 주, **[떠돌이 상인]**(희귀·로테이션 발견)·**[카페 일구기]·관계 마일스톤 보상 조각**(특별 기념·디제시스 의미)이 보조. *제작(재료) 게이트는 안 둔다*(코스메틱을 노동에 묶으면 의무화). 런치 1세트는 초반 저가/무상으로 캔버스를 바로 체험시키고, 추가 세트가 진척 드립을 진다(테마 데이 해금 리듬과 동형). ⚠️ 빌드 워크플로우 = 배치 *로직*은 그레이박스, 세트 *에셋*은 아트-주도([ADR-0028] "가구/장식=아트 주도"·[ADR-0025] 스펙 카드). 타이밍은 로드맵 재배치.

**저승 작물 (Underworld Crops)**
: 안식 농원의 작물. 입문 3종은 성장 속도를 3단으로 나눠 리듬을 만들고, 후속 2종은 스타듀식 메카닉 아키타입(재성장·트렐리스·다수확·다절기)을 각자 대표한다(합성 플래그 모델, [homestead-farming-greybox-spec §2.1](./docs/design/homestead-farming-greybox-spec.md)).
: - **혼령초 (魂靈草)** — 푸르게 빛나는 영혼 풀. 빠름·저수익. 입문/튜토리얼 작물. (유화절)
: - **피안화 (彼岸花)** — 저승을 상징하는 붉은 꽃. 중간 속도. (피안절)
: - **영혼 호박 (Soul Pumpkin)** — 얼굴이 살짝 비치는 호박. 느림·고수익. 미호 호박밭 떡밥과 연결된 상징 작물. 성숙 시 확률적 [거대] 합체. (성야절)
: - **황천포도 (黃泉葡萄)** — 황천(黃泉=저승)의 넝쿨 포도. **트렐리스**(격자 통과불가·인접 수확)이자 **재성장**(수확 후 되자람)·**다수확**(송이 2~3개). 투자→패시브 결. (망연절=가을, 유일 공백 절기를 메움)
: - **불사과 (不死果)** — 미혹의 숲 심층 채집으로만 얻는 고대열매. **[다절기 프레스티지]** — 절기 전환 사멸을 넘어 살아남는 "죽지 않는 혼의 작물"(쟁취하는 예외, [저승 절기] 참조). 만물상 미판매(채집 전용).

**혼의 나무 (Spirit Tree · 저승 과수)**
: 안식 농원의 **영속 과수** — 묘목을 심어 **1절기(28일) 성숙** 후 **제철마다 매일 결실**하고, 절기 전환에도 죽지 않는다([다절기 프레스티지]의 나무판, [저승 절기] 사멸의 예외). [작물]과 별개 시스템이다: **3×3 자리**를 차지하고, 물주기·비료 같은 일일 돌봄이 없는 *패시브 생산자*다. ⚠️ **품질 = 나무의 나이**(키운 절기 수 — 1년(112일) 키우면 최고 등급). 이는 [작물]의 *품질=비료*와 의도적으로 갈린 별개의 품질 소스다. 저승 과일 로스터는 자체 창작하며, 그레이박스 정준 종 = [혼백도]. (상위 결정 grill 2026-06-29 [stardew-systems-catalog §60](./docs/design/stardew-systems-catalog.md), 구현 [homestead-farming-greybox-spec §7](./docs/design/homestead-farming-greybox-spec.md))
: - **혼백도 (魂魄桃)** — 저승 복숭아. [혼의 나무] 그레이박스 정준 과일. 피안절(봄) 결실(복숭아=봄꽃 정합).

**저승식 스프링클러 (Sprinkler)**
: 매일 인접 칸을 자동으로 물 주는 농사 자동화 장치(스타듀 스프링클러의 저승 번역 — 구체 명칭·외형은 서랍). **물주기만 자동이고 심기·수확은 손**(수확=보람 액션 보존, [Junimo식 자동 수확은 기각]). 덮은 칸은 물주기 [혼력]을 안 쓰며(아낀 혼력은 채광·낚시 등 다른 루프로 *이동* — 노동 전용 역할 유지), **제작으로 획득**(농사 [스킬](./docs/adr/0019-physical-skill-relationship-multiplier-two-axis.md) 레벨 해금 + 채광 재료, 단계 티어 — 농사×채광 루프 직조). ⚠️ *편의 해금이지 접근 게이트 아님*("평평 ≠ 막힘", [ADR-0008] #1) — 저스킬·초반엔 손 물주기(+스킬 도구 범위 3×3)로 멀쩡히 굴러간다. ADR-0021로 미호 자동화 곱셈기가 사라져(관계=가벼운 당근) 자동화 공간이 비었고, 그 자리를 *관계 비의존 제작 QoL*로 채운 것(grill 2026-06-24, [stardew-systems-catalog](./docs/design/stardew-systems-catalog.md)).

**사연 한 줄 (Flavor Memory)**
: 작물 수확 시 그 영혼의 생전 사연 한 줄이 표시된다(예: "이 영혼은 생전에 핼러윈을 가장 좋아했대"). 텍스트 한 줄로 세계관 깊이를 더하는 저비용 장치.

**옥자의 잃어버린 책들 (Okja's Lost Books)**
: 발견형 로어의 *보상* 층([비밀 노트]=떡밥 ↔ 책=보상, 한 파이프라인의 두 형태 — [ADR-0034](./docs/adr/0034-books-narrative-discovery-okja-lost-books-mechanic-track.md)). [그날 밤] 대화재로 흩어진 옥자의 것을 세계 곳곳에서 *되찾으며* 읽는 무거운 한 챕터들이다. **[옥자]가 *게임 중 호감도 미터가 없는* 보이지 않는 앵커([ADR-0032])라 비어 있던 "옥자/세계를 알아가는 채널"을 이 책들이 메운다** — 옥자(약사/무녀)·세계 규칙·조연 사연이 한 조각씩 드러난다. ⚠️ **봉인 법칙 불가침([기억 봉인]·[ADR-0016]):** 책은 *타인의 말*이라 플레이어 죄목의 *중심 평결*도, 미스터리 *해결 열쇠*(세 조각=메인 4인 독점)도 서술하지 못한다 — *증거·공명*만 더한다([해방] 후 다시 읽히는 망연절식 환경 떡밥). **순수 서사 수집품(메카닉 효과 0)** 이라 [2축 성장] 충돌 원천이 없고, 매력은 이야기에서 나온다. 입수=세계 발견 → 인라인 즉독 → [혼백관] 유품 전시·[집 꾸미기] 책장 보관·재읽기. 선물 불가(유품 키프세이크). *통제된 예외 하나* — 특수 하위셋 **"옥자의 잃어버린 레시피"** 는 옥자의 약사 설정이 한 접시로 현현해 카페 *히든* [융합 메뉴] 1종을 연다(콘텐츠 해금이지 곱셈기 아님; 메인 메뉴 진척은 [카페 일구기] 독점).

**저승 실용서 (Underworld Manuals)**
: [옥자의 잃어버린 책들]과 분리된 *메카닉 곁들이* 트랙(서사 0 — [ADR-0034]). **flat 일회성 범프 + 곱셈기-무관 편의만** 준다: 스킬 XP/최대 HP 통 일회성 범프·정보/QoL 영구 해금(판매가 표시·지도)·이동속도↑. ⚠️ **영구 % 곱셈(+수율·+가치·피해감소)은 무조건 관계 곱셈기의 몫이라 책으로 안 준다**([ADR-0019]·멜/바나 충돌 방지). XP 범프는 *유한·희소*라 관계의 우월한 경로([ADR-0021])를 안 가린다. 입수=[여행 상인] 재고 흡수(새 NPC 0)+활동 드랍. 선물 가능(멜 애호 — 멜=옥자의 *기록* 결).

**카페 운영 (Cafe Operation)**
: 카페+바 서비스 루프. **농사와 나란한 하루 루틴 중 하나**(게임을 카페 운영으로 격상하는 게 아니라, 스타듀형 멀티루프에 루프 하나를 더하는 것 — ADR-0007). 다른 활동(농사·낚시·채광)의 산출물을 [출하대]로 환전하거나 카페 재료로 삼고, 그 재료로 **손님을 응대·서빙**해 매출을 낸다. 플레이어가 직접 카페를 돌아다니며 하는 활동이지, 위에서 관리하는 매니지먼트 타이쿤이 아니다(ADR-0002 직접 조작 유지). ⚠️ **2026-06-24 개정([ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md)):** 단일 메인 주인이 없다 — **옥자(사장)·미호·멜·바나 넷 다 오후 3시~밤 12시 서비스 블록에 출근**하되 **역할이 갈린다 — [주방요괴]=기본 음식·음료 생산(백스테이지), 메인 4인=특별 음료·칵테일(낮 카페 숙련 메뉴), 플레이어=프론트(응대·체키·그림·편지)**(아침엔 각자 본거지: 미호=밭, 멜=카지노, 바나=던전; 옥자=카페 상주). 멜의 관계 보상은 카페에 통일된다([ADR-0029] — 카지노는 관계 보상에서 격하, [멜]·[카지노] 참조). ⚠️ 단 **밤 [나라카 바]에선 플레이어가 직접 바텐딩**한다([칵테일] 미니게임 — 낮 칵테일은 메인 4인 NPC가, 밤 칵테일은 플레이어가).
: 농사와 **대체재가 아니라 보완재(공급사슬)**다 — 밭(느린 상류)→재료(재고)→서빙(빠른 하류, 골드). 둘 다 골드를 주는 대체재로 두면 잘 버는 쪽만 하고 나머지가 죽으니, *농사 그리고 카페*로 엮는다. ⚠️ **이 재료→메뉴 연결은 [융합 메뉴](실제 카페 메뉴 × 저승 산출물)로 구현된다(grill 2026-06-29)** — 발명한 저승 메뉴를 따로 만드는 게 아니라, *실제 나라카 컨셉카페 메뉴를 저승 산출물로 번안*하는 것 자체가 세계관 융합 작업이다. 희소성은 혼력이 아니라 **시간**(손님이 오는 서비스 창)에 싣는다([ADR-0008] 보존부 — 서빙은 시간 제한).
: **서빙 메카닉(grill 2026-06-24): *단계 사슬* + *가벼운 압박(다)*.** 플레이어 프론트 일은 한 손님에 값·시간이 커지는 단계로 쌓인다 — **응대(대화) → 체키 촬영 → 체키 그림 → 편지**(아래로 갈수록 단가·팁·단골화↑, 소요 시간↑, 다른 손님 방치도↑). 손님은 *넉넉한* 인내심 타이머를 갖고(코지·번아웃 방지), 놓쳐도 *작은* 매출 손실(가혹 X). 핵심 결정 = **한 손님에 사슬을 깊이 쏟을까(↑수익) vs 회전 유지할까(다른 손님 안 놓침)** = 옛 [밤 경비] 기회비용의 새 그릇. ⚠️ **미니게임화 확정(grill 2026-06-24, "한둘만"=둘·전부 동시 X):** 단계는 *그레이박스 단순 상호작용*으로 먼저 굴리고, Phase 3+에서 **둘만 진짜 미니게임으로 심화 — 낮 [체키](창작·수집 시그니처) + 밤 [칵테일](스킬 믹싱/타이밍)**(낮=코지 창작 / 밤=액티브 스킬 대비). **편지는 단순 단계로 유지**(글쓰기는 *작성 콘텐츠*라 미니게임으로 약함). 셋(그림·편지·칵테일) 동시 구현 금지 — *순차*(낮 체키 100% 먼저 → 밤 칵테일, ADR-0001 야크 셰이빙 회피).
: **시간·역할 구조([ADR-0029](./docs/adr/0029-cafe-relationship-stage-domain-aligned-rewards.md), 2026-06-29):** ⚠️ **소프트 분할** — *노동(농사·낚시·채광)은 종일 가능*, 카페/바는 15~24시에 열리는 **옵트인 기회 창**(오전 노동/오후 서비스는 강제가 아니라 자연 리듬 — 아침엔 직원이 각자 본거지라 카페가 비어 오후에 열릴 뿐, 하드 분할 기각). ⚠️ **플레이어 역할 = 카페의 *프리미엄·성장 엔진*(운영 필수 인력 아님):** 베이스라인(기본 메뉴=[주방요괴]·[융합 메뉴]=메인 4인)은 플레이어 없이도 굴러가고(→ *자유*: 손님당 기여라 2~3명만 받고 빠져도 됨), 카페의 *성장*([카페 일구기]·관계·골드)이 주로 플레이어 출석으로 흐른다(→ *의무감 = 기회비용*). **만성 스킵 = 당근만·무처벌**(감쇠 없음 — 그날 안 챙길 뿐, 은둔 활동파 보호·"평평≠막힘").

**주방요괴 (Kitchen Youkai)**
: 카페+바 주방에서 **기본 음식·음료를 생산하는 백스테이지 NPC**(실재 컨셉카페의 주방 담당을 옮김). 특별 음료·칵테일은 메인 4인이 만들고([카페 운영] 역할 분담), 주방요괴는 일반 메뉴를 댄다. **관계·서사 트랙 없음**(깊이는 메인 독점 — 배경 직원 또는 [마을 주민] T3 결, 구체 정체는 서랍). 플레이어 프론트 사슬에 *재료를 공급*하는 상류.

**융합 메뉴 (Fusion Menu)**
: 카페 메뉴 = **실제 나라카 컨셉카페 메뉴를 저승 산출물로 번안한 것**(grill 2026-06-29). 발명한 순수 저승 메뉴도, 날것 실제 메뉴도 아닌 *둘의 융합* — 영혼 호박(저승 작물)이 실제 카페 메뉴로 번안돼 "영혼 호박 [메뉴]"가 되는 식. ⚠️ **이 융합 자체가 IP(실재 카페)와 저승 판타지를 자연스럽게 섞는 세계관 작업**이다 — 공급사슬을 "대체재 방지 기계 장치"([ADR-0008])에서 *세계관 융합*으로 격상한 것(대체재 방지는 덤). 동시에 **콘텐츠 생성 규칙**이다: 실제 메뉴 → 어울리는 저승 산출물 → 융합([ADR-0025] 스펙 카드의 입력).
: **융합 깊이 = 기계적·가볍게:** 융합 메뉴 1장당 **시그니처 저승 산출물 1종**을 곳간에서 차감한다(다중 재료 레시피 트리는 안 만듦 = ADR-0001 야크 셰이빙 회피). *화장(이름만) 융합은 기각* — 농사가 카페와 무관해져 [ADR-0008] 대체재 문제가 부활하고, 융합이 *체험*되지 않는다(길러서 메뉴가 되는 손맛이 핵심).
: **2단 메뉴 구조("평평 ≠ 막힘" 보존):** ㉠ **기본 메뉴**(플레인 커피·물·차 등 소수) = 무재료·항시 제공·낮은 베이스 골드 → 곳간이 비어도 카페가 *안 멈춤* / ㉡ **융합 메뉴**(시그니처·푸드·계절) = 시그니처 저승 산출물 1종 요구·높은 프리미엄 골드 → 농사→메뉴 결속의 본체. 곱셈기 철학과 동형(평평해도 안 막히고 투자하면 명백히 우월, [ADR-0008] #1·[ADR-0021]).
: **절기 로테이션(자동 결과):** [저승 절기] 작물이 절기 전환 시 사멸하므로 그 작물에 묶인 융합 메뉴도 **절기마다 로테이션**된다(피안절=피안화 메뉴 … 성야절=영혼 호박 메뉴) — *메뉴판이 절기마다 바뀌는* 스타듀식 계절감(코지 강점). 작물 사멸 시 그 메뉴가 내려가는 건 **계절 메뉴로 받아들인다**(자연스러움). 단 **다절기 프레스티지 작물**(이미 결정된 희소 1~2종, [저승 절기] 예외)에 묶인 메뉴는 *연중 유지*(새 시스템 안 늘리고 기존 결정으로 해결).
: **수요·재고(곳간):** 손님은 융합 메뉴를 *선호·주문*한다 — [곳간]에 해당 저승 산출물이 적재돼 있으면 **프리미엄 매출**, 없으면 **기본 메뉴로 폴백**(낮은 매출, *손님 안 떠남* — 하드 실패 없음, "평평≠막힘"). 충족은 **NPC 직원이 곳간 재고에서 자동 처리**(기본 메뉴=주방요괴 / 융합=메인 4인 특별 음료, [카페 운영] 역할 분담) — *플레이어는 단무지 요리사가 안 됨*(플레이어 손일은 프론트 상호작용 사슬). → *무엇을 길러 곳간에 쟁일지 = 소프트 수요 예측*(절기마다 인기 메뉴가 다름)이 농사에 텍스처를 준다.

**곳간 (Larder)**
: 카페의 **재고 적재처**([융합 메뉴] 재료 = 저승 산출물). ⚠️ **[출하대]와 구분된다(grill 2026-06-29):** 수확물을 *출하대(즉시 골드·낮음)* 로 보낼지 *곳간(적재 → 고마진 융합 메뉴 재료·나중)* 으로 보낼지가 **손에 잡히는 "농사 그리고 카페" 선택**이다(같은 영혼 호박을 *팔 것인가 / 메뉴로 키울 것인가*). NPC 직원([주방요괴]·메인 4인)이 서빙 시 여기서 재고를 자동 차감. 적재는 batch 단위(per-item 마찰 X, 코지). 용량 확장은 [카페 일구기] 결([집 업그레이드] 가방 칸 확장과 같은 결).

**출하대 (Shipping Counter)**
: 활동 산출물(수확물 등)을 골드로 환전하는 카페 안 거래 창구. 스타듀의 출하 상자 대응. ⚠️ **2026-06-24 개정([ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md)):** 멜 운영이 아니라 **무인 출하함**(넣으면 골드 — 멜은 서빙 직원으로, 출하대는 주인 없는 창구로 분리). 씨앗 구매는 만물상(네오, 매대 소매) 쪽. 입고된 산출물은 [카페 운영]의 재료로도 흐른다.

**저장 상자 (Storage Chest)**
: 집·창고에 두고 아이템을 **보관**하는 순수 컨테이너(스타듀 상자 대응). ⚠️ **[출하대](즉시 판매)·[매대](구매)와 구분된다** — 저장 상자는 **경제 0**(팔지도 사지도 않고, 넣고 빼기만). 가방(인벤토리)이 꽉 찰 때 넘치는 산출물을 쟁여두는 코지 저장소. 곳간([곳간])이 *카페 재고 적재처*라면 저장 상자는 *플레이어 개인 창고*다. UI는 인벤토리 프레임의 한 컨텍스트(CHEST)로 연다. ([ADR-0048](./docs/adr/0048-homestead-demo-completion-pass-ui-screens-interiors.md) 신설)

**카지노 (Casino)**
: 멜의 *아침 본거지*(나루 마을 서편 — 미호=밭·바나=던전과 대칭, 아침엔 여기 → 오후 서비스 블록엔 카페 출근). ⚠️ **2026-06-29 격하([ADR-0029](./docs/adr/0029-cafe-relationship-stage-domain-aligned-rewards.md)):** 옛 *멜 관계 보상 무대*에서 **선택적 [명부의 운] 베팅 무대 + 속죄 플레이버**로 내려왔다 — 멜 관계·보상은 카페로 통일([멜]·[호감도]). 카지노는 멜 호감도에 *안 매이는* 독립 콘텐츠로 존속. ⚠️ **Phase 3+ 후속**([ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md) — 출시 전까지 멜은 카페 보상·서사 보상만). 죄(불법 도박·사채)와 글자 그대로 닿아 *도박장=죄의 반복*([ADR-0004](./docs/adr/0004-atonement-theme.md) 충돌)을 피하려 **"속죄한 도박꾼이 *공정한* 하우스를 운영(등치지 않고 절제를 가르침)"** 으로 재단(grill 2026-06-24):
: - **코어 = 가벼운 공정 베팅 + 절제 층.** 플레이어가 골드를 건다(가벼운 미니게임 — 혼 하이-로우/주사위 1종; 독립 슬롯/카드 도박 엔진은 안 만듦 = ADR-0001 야크 셰이빙 회피).
: - **공정 불변식 = 하우스 엣지 0.** 멜은 *결코 안 기운다*(속죄 불변식). 그날 주사위를 기울이는 건 **[명부의 운]뿐**(운명이 굴림, 옥자 예보로 투명 — "옥자=운을 *읽기* / 멜=운을 *걸기*" 분업 완성). 운 좋은 날 베팅 = 절제 상한까지 *가벼운·상한 걸린 골드 드립*, 평·흉일엔 자제(계획 손맛). 카지노 = 운명 읽기를 *가벼운 골드로 환전*하는 무대(만렙에서 증발 = [ADR-0021] 당근).
: - **절제 = 하드 일일 상한 + 소프트 코칭.** 오늘 베팅 예산 초과 불가·연속손실 시 그날 잠금 → 파국 손실이 *구조적으로 불가*([체력] 기절처럼 코지 가드레일·번아웃 방지). 멜 대사("세 판 졌으면 멈춰 — 그게 규칙이야")는 그 위 연출. 약탈적 인에이블링(빚 권함) 죄의 거울 = 플레이어를 *중독 나선에서 보호*.
: - ⚠️ **호감도 보너스 폐기([ADR-0029](./docs/adr/0029-cafe-relationship-stage-domain-aligned-rewards.md), 2026-06-29):** 옛 '멜 카지노 호감도 보너스(절제 천장↑·컴프·운 예보 정밀화)'는 *멜 보상이 카페로 통일*되며 제거됐다 — 멜 호감도는 카페(서빙 마진·경제)로 보상하고 카지노는 멜 호감도와 *비연동*. **절제(하드 일일 상한·연속손실 잠금)는 *구조적 가드레일*로 남는다**(호감도 무관 — 파국 손실 구조적 불가, 번아웃 방지). 베팅 EV는 안 건드린다(공정 불변식 — 그날 주사위는 [명부의 운]만 기울임).

**마을 주민 (Village Residents) — 3티어**
: 나루 마을에 사는 망자 NPC들. **나라카의 모든 망자처럼 이들도 생전에 죄를 지어 저승에 왔다**(저승=죄인이 오는 곳). 단 *저승에서 옥자에게 또 죄를 지어* 붙잡혀 카페 종신계약에 묶인 메인 4인(+플레이어)과 달리, 주민은 마을에 **정착한 자유민**이다 — 종신계약이 아니므로 **주민 결혼엔 [해방]이 없다**(해방=메인 결혼 전용). 생전 사연/가벼운 죄는 플레이버 한 줄로 *가질 수 있으나*, **속죄 아크·관계 활동 보상·서사 조각은 없다**(메인 독점). **T1 주민 생전 죄 규칙:** 각 T1 주민은 [사연 한 줄] 무게의 생전 죄를 하나 갖되 — ① 그 요괴의 **설화 본질과 공명**하고(설화가 곧 죄의 씨앗 → 캐릭터 비용 0), ② 메카닉으로 **뒤집지 않으며**(속죄 능력·관계 활동 보상 없음 — ADR-0004 정반대-속죄는 메인 전용), ③ 그 죄가 **결혼 후 반전 로맨스의 축**이 된다(평소 매력적 겉면 ↔ 결혼 후 드러나는 죄에서 비롯된 진짜 면모). 죄는 **옥자와 무관**(옥자에게 지은 죄 아님, 지옥에 온 이유일 뿐 — 메인 4인과의 차이). **⚠️ 비인간 요괴 예외:** T1 중 **비인간 요괴**(슬라임 등)는 죽어서 온 망자가 아니라 **나라카에서 생겨난 크리처**라 *생전(前世)이 없다* → **생전 죄 면제.** 이들의 결혼 후 반전은 *죄의 되돌림*이 아니라 **크리처 본성의 사랑스러운 심화**다(예: 흡수만 하던 슬라임 → 자신을 내어주는 반려). 즉 T1은 두 갈래 — **망자 주민**(생전 죄+속죄 결 반전) / **나라카-출생 크리처**(죄 없음, 순수 크리처 로맨스). 맵 확장으로 마을이 생기며 [단골 손님]을 3티어로 분화한다([ADR-0014](./docs/adr/0014-relationship-marriage-liberation.md)) — **T1 결혼 가능**(풀 캐릭터: 초상화·하트·하트 이벤트·결혼; 목표 남요괴 5+·여요괴 3+·비인간 요괴 3+, 맵엔 *집 자리만* 그레이박스로 확보하고 한 명씩 추가) / **T2 사귐만**(상점 점주류 — 가벼운 호감도로 할인·레시피 등 퍼크, 결혼 X) / **T3 익명 손님 볼륨**(이름·호감도 0, 분위기 군중 — *명명 단골은 [조연] 캐스트가 맡고 T3는 얼굴 없는 볼륨이다*, grill 2026-06-29 · [단골 손님] 참조). ⚠️ **서사 조각·관계 활동 보상(당근)은 어느 티어도 못 준다 — 메인 4인 독점**(주민은 *넓이*만; 깊이는 메인, ADR-0005 보존부).

**단골 손님 (Regular)**
: 카페에 드나드는 손님 군상. ⚠️ **2층 구조 확정(grill 2026-06-29):** ㉠ **명명 단골 = [조연] 캐스트** — 그날 밤 코러스(T1 주민 11인 + T2 점주)가 카페에 모인다(나루 마을 서편 *사회 허브*). *의미 있는 손님*(응대·체키·[단골화]·관계 대상)은 **전부 이 조연들**이다(새 캐릭터 발명 0 — 매일 커피를 내주는 손님 = 미스터리의 증인들, 코러스 모델 정합). ㉡ **익명 볼륨 = T3** — 이름·단골 지위·호감도 *없는* 얼굴 없는 엑스트라. 카페를 *붐비게* 보여주는 분위기 몸일 뿐(캐릭터 아님). ⚠️ **이 익명 볼륨이 [카페 일구기]로 늘어나** "카페가 명소로 붐빈다"(ADR-0009 성장 판타지)를 떠받친다 — 조연 캐스트는 ~11인 고정이라 *붐빔의 성장*은 T3 볼륨이 진다.
: **단골화(카페 당근):** 조연(T1/T2)이 *당신 카페의 단골이 되는 것* — 서빙·교류로 조연이 카페에 자주 들고 카페가 *살아난다*(명소화 기여). ⚠️ **단골화는 호감도 트랙이 아니다**(빈도·수집 결, ADR-0017 보호) — 조연의 풀 관계(서브 호감도·결혼)는 [마을 주민] T1/T2 트랙이 따로 진다.
: **카페 = T1 서브 관계 무대(결과):** 조연이 카페에 모이므로, *서빙하며 마주치는 동선*이 곧 [호감도] T1 서브(일일 대화·선호 선물)의 자리다. 단 **서빙(매출)과 대화·선물(호감도)은 별개 행동**(ADR-0017 경계 — 조연=넓이, 서사·곱셈기는 메인 4인 독점).

**체키 (Cheki)**
: 현실 컨셉카페에서 직원과 찍는 즉석 기념사진을 저승 세계관에 옮긴 [카페 운영] 프론트 활동(플레이어 도메인 — 멜 아님, ADR-0021로 카페 단일 주인 없음). **단어 "체키"는 IP 시그니처라 유지하고 결만 저승식**(실물 레퍼런스: 그을린 양피지 + 검은 나비 고딕 프레임). 아티팩트 = 합성 사진(포즈) + 손글씨 낙서 캡션 + 저승-고딕 프레임. **혼력 안 씀 — 시간 게이팅**(서빙과 같은 낮 손님 창, ADR-0008). 플레이어의 카페 역할 = **홀 막내 직원**(손님 응대·서빙·체키·바를 직접 담당 — 옥자(사장)/멜(카지노)/바나(던전 전투)/미호(농사)가 각자 본거지를 갖고 서비스 블록엔 다 같이 서빙하는 것과 대비). ⚠️ **체키 = 낮 카페의 심화 미니게임(grill 2026-06-24):** [카페 운영] 단계 사슬 중 *진짜 미니게임으로 키우는 둘 중 하나*(낮 시그니처 — 포즈·낙서로 *창작·수집*하는 결이지 스킬 스파이크 아님; 밤은 [칵테일]). 두 갈래(grill 2026-06-24):
: - **상업 체키** — 카페 영업 중 손님 서비스의 *프리미엄 단계*([카페 운영] 단계 사슬: 응대→촬영→그림→편지). 매출·팁·단골화↑, 테마 데이 수요↑. **남기는 아티팩트 없음**(손님이 가져감 — 수집 대상은 *개인 체키*만).
: - **개인 체키 (수집 앨범)** — [카페 이벤트 데이]를 운영(옵트인)한 날, 그날이 끝나고 **뒷정리 시간**에 플레이어가 코스프레 차림의 메인 캐릭터들을 *찍어주며* 얻는다(각자 개인 컷 + 플레이어도 들어가는 단체샷). 동료를 *수집*하는 게 아니라 *추억 정표를 남기는* 거라 직원 신분에 자연스럽다. 이벤트 데이마다 키프세임이 쌓여 **해방 후 옵션형 완주 트래커**(완벽함→해방 리스킨, ADR-0014)의 코지 그릇이 된다. **호감도를 올리지 않는다**(ADR-0017 보호 — 단 깊은 하트가 *더 특별한 체키 포즈를 여는* 보상은 가능).
: **앨범 UI(grill 2026-06-24):** Phase 2.8 탭 메뉴 셸을 공유하되 **앨범 탭만 *전체 화면 스크랩북***(아이템 맥락이 아니라 감상이라, 하단 *플레이어 가방* 모듈은 이 탭에서 숨김 — 프레임 §3.4의 작은 예외). **구성 = [카페 이벤트 데이]별 한 펼침면**(단체샷 1 + 동료 개인 컷 — 메이드 데이면 단체+미호·멜·바나·옥자 = 5칸), 절기 시간순 정렬. **완주 트래커 = 못 모은 칸은 실루엣**으로 비어 "메이드 데이 3/5" 식 진행률(해방 후 롱테일의 코지 그릇). **상호작용 = 사진 확대 감상 전용**(기계 보상 0, ADR-0017).

**나라카 바 (Naraka Bar)**
: 밤(예: 19시~24시)에 카페가 전환되는 밤 영업 모드 — [카페 운영] 서비스 블록(오후 3시~밤 12시)의 *밤 파트*. ⚠️ **2026-06-24 개정([ADR-0021](./docs/adr/0021-relationship-reward-light-affinity-narrative-primary.md)):** 옛 *바나 밤 경비*의 무대가 아니라 **멜/공동 밤 서빙**의 무대다(밤 경비·잡귀 약탈 리스크 삭제). 네 명 다 늦게까지 서빙한다. ⚠️ **밤 정체성 = 플레이어 [칵테일] 바텐딩(grill 2026-06-24):** 낮 카페가 [체키](창작·수집)면 밤 바는 플레이어의 **칵테일 믹싱/타이밍 미니게임**으로 구별된다(낮 서빙의 단순 연장 아님 — 밤에 독자 *액티브 스킬* 결). ## 아트 스타일 (Art Direction)

**인스타툰체**
: 플랫 컬러 + 두껍고 균일한 검은 윤곽선 + 치비(2~3등신) 비율 + 단순한 배경. SNS 웹툰 톤. 원작 IP의 룩이며, 컷신·홍보 일러스트가 이 톤을 따른다.

**인게임 도트체**
: 실제 플레이 화면(탑다운)에 쓰는 픽셀아트. 인스타툰체를 픽셀아트 특화 AI(PixelLab / Retro Diffusion)로 변환·생성하고 Aseprite로 손보정한 것. 게임 장르는 **진짜 스타듀밸리형(탑다운 픽셀 이동 + 농사·낚시·채광·관계)**.
