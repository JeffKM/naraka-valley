## 0. 라이브 전체 맵에서 실제 보이는 것

균일한 tan 평원 전면 + 그 위에 흩뿌린 오브젝트(풀tuft·twig·stone·잡초). 피처들:

- **연못:** 정사각 파란 박스 + 상단만 널판 테두리. 하드 직각.
- **절벽/고지:** 좌측 2개의 직사각 갈색 "구덩이"(grass top + dirt face). 구멍처럼 읽힘.
- **덤불 벽:** 완벽한 일렬 세로 덤불 12개(SOLID 경계). 극도로 인공적.
- **건물 발치:** 하드 직사각 tan 패드.
- **밭(SOIL):** 낱개 둥근 사각 타일이 격자 간격으로 뚝뚝(오토타일 미연결 — owner "따로따로 생성").

## 1. Crux — 베이스 지형: tan-단조 (가장 큰 격차, 결정 필요)

**스타듀 팜 = 잔디(초록) 지배 + 흙 패치.** 안식 = **흙(tan) 지배 + 잔디 스캐터**([ADR-0053] 봉인). 전체 덤프로 보면 이 선택이 맵을 **황무지/사막**처럼 만든다 — 스타듀의 lush farm과 정반대. owner의 반복된 "잔디가 거의 없다"가 사실 이 근본이다.

**긴장:** [ADR-0053]은 "overgrown 개간 황무지를 복원한다"는 속죄/개간 테마([ADR-0035/0055])로 흙-지배를 *의도적으로* 봉인했다. 그런데 스타듀 시작 팜도 잡초·돌 무성하지만 **베이스는 초록 잔디**다("overgrown"이 곧 "tan"은 아니다). 즉 "개간 전 방치"를 tan 사막으로 표현한 게 과했을 수 있다.

**crux 결정 — RESOLVED (owner 2026-07-17, 레퍼런스 `Desktop/naraka/map/스크린샷 2026-07-06 오후 9.40.44.png` = 스타듀 Forest Farm 지목):**

> **확정 = B/C(흙-지배 유지 + 유기 잔디뭉치 + 나무/절벽 경계).** 레퍼런스가 정확히: **① base=tan/흙 지배(ADR-0053 유효·A안 lush 피벗 기각) ② 흙 위에 채워진 잔디 "뭉치(패치)"가 불규칙 clump으로 분포(낱개 tuft 아님) ③ 사이드=전부 빽빽한 나무 + 바위 절벽 ④ 물=유기형 blobby 연못 ⑤ 나무 내부에도 밀집 ⑥ 돌·그루터기·잡초 overgrown 스캐터 유지.**
>
> **★정정:** 앞서(같은 세션) taxonomy를 Standard Farm 기준으로 읽어 "tuft-only·잔디패치 폐지"로 갔던 것은 과교정. Forest Farm 레퍼런스는 **잔디 패치(채움 clump)가 명백히 존재** → `_G16_GRASS_PATCHES` 재활성(유기적으로). 이전 패치 폐지 사유(잔디↔흙 하드 Wang 경계)는 **소프트 tuft 프린지 엣지**로 해소(레퍼런스도 경계가 tuft로 풀림 = owner "1자 경계" 직결).
>
> **슬라이스:** ①유기 잔디뭉치 부활(CA 마스크→`_bf_grass` 패치+소프트 엣지) ②경계=나무선+절벽(일렬 덤불 폐기) ③물 유기형 ④밭 오토타일 ⑤건물 접지 ⑥내부 나무 밀도. CA 클럼프 마스크(scatter 브랜치 Task 3)가 ①의 엔진.

## 2. 감사 — 우선순위별 개선 항목

### P1 · 하드 직선/직사각 경계 전면 (owner "1자 경계" 직격)
스타듀는 **오토타일 + 유기적 ragged 엣지**로 모든 경계를 녹인다. 안식은 전 피처가 하드컷:
- **길↔지면:** `_build_path_grass_fringe` 물결이 tan-tan에선 안 읽힘 → 길 양옆에 **명암/톤 대비 + raggedness** 강화, 잔디 base(A안)면 길↔잔디 프린지로 자연.
- **밭(SOIL) 외곽·타일간:** 오토타일(Wang) 미적용 → **밭 = 연속 오토타일 필드**로(내부 furrow 격자만 남기고 외곽·타일경계 매끈). owner "따로따로" 직결.
- **연못·절벽·건물 패드:** 아래 P3·P4·P7.

### P2 · 밭(SOIL) 오토타일 — "따로따로" 해소 (효과 큼·비교적 국소)
현재 SOIL 타일이 낱개 스프라이트로 격자 갭. **연속 tilled 필드**(Wang/오토타일)로 교체 — 내부는 furrow 라인, 외곽은 흙↔지면 유기 전환. 스타듀 hoe-dirt와 동형.

### P3 · 물 — 정사각 연못 → 유기형 + 오토타일 물가
현재 하드 직각 박스. **유기적 윤곽 + 물↔지면 Wang 물가**(반영·리플 엣지). [ADR-0044] 남향 물 토폴로지(강/개울)와 정합 검토. 스타듀 연못=곡선 shore.

### P4 · 절벽/고지 — 직사각 "구덩이" → 유기 절벽
좌측 2개 직사각 구덩이가 구멍처럼. **유기적 절벽 윤곽 + 초록 top lip + 그림자 접지**([ADR-0056]는 남단만 처리). 동단 "잔디 능선" 평평 seam은 dissolve로 완화됐으나, 절벽 자체 형태가 기하학적. 스타듀 절벽=불규칙 곡선.

### P5 · 경계 덤불 벽 — 일렬 → 유기 배치
완벽한 세로 일렬 덤불 = 인공. 스타듀 경계=불규칙 나무선/절벽. **덤불·나무를 불규칙 클럼프로 산개**(통행 차단은 유지하되 라인 깨기), 또는 절벽/울타리로 대체.

### P6 · 레이아웃/조닝 — 균일 빈 평원 → 목적 있는 zone
현재 광활한 빈 tan. 스타듀 팜=암시적 zone(농사·목축·과수·물). **의도적 구역화**(밭 블록·목초지·과수원·연못 주변)로 "빈 곳" 감소. C안(base 이원화)과 시너지.

### P7 · 건물 접지 — 하드 직사각 패드 → 유기 접지
`_g16_near_building` 패드가 직사각. **발치 유기 흙 번짐 + 그림자**로 "심어진" 느낌([ADR-0054] 정신 확장).

### P8 · 밀도·변주 잔여 — 스캐터 자체는 개선됨(이번 작업)
풀tuft 분포·clump은 이번 dissolve+CA로 나아짐. base(P1 crux) 해결 시 체감 급상승. 변종 확대(로스터 §)는 병렬 지속.
