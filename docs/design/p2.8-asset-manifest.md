## 4. 구역별 매니페스트

> 각 셀: **터레인 교체** = 그 구역에 깔린 터레인 중 신규 도색 대상(GROUND/PATH/SOIL은 T1 전역 재톤이라 구역 작업에선 *장식만*) · **장식(밀도 상한)** = 신규 `create_map_object` 장식 종수 상한 · **외관** = 서비스 건물 파사드.
> **밀도 원칙:** "구역이 *장소*로 읽힐 상한"(P2.3 계승, 스타듀 클러터 X). 상한은 *종수*이지 인스턴스 수가 아니다(같은 종을 여러 칸에 흩뿌리는 건 자유).

### T3 — 안식 농원 (80×65) · 데모 1

- **터레인 교체:** 없음(GROUND/PATH/SOIL = T1 피안절 재톤만). 물·나무·바위 없음.
- **장식(상한 4종):** 울타리(밭 경계)·농장 소품(물통/허수아비류)·길가 화분·꽃 패치. *과하지 않게 — 휑함보단 낫고 클러터보단 적게.*
- **외관(2건):** 창고(NE, 현재 enterable 그레이박스) · 축사(동편, 비-enterable 자리).
- **예상 gen:** 장식 ~4 + 외관 ~2 = **~6 gen**

> **★ T1 플레이테스트 피드백(2026-06-24, 옥자님 데모 사전 점검) — T3에서 해소:**
> 1. **집 외관이 맵·footprint 대비 너무 작다.** 집 facade가 어두운 WALL 박스(footprint) 한가운데 작게 떠 어두운 여백이 큼 + 80×65 맵에서 왜소. → 집 facade 스케일↑(footprint 1:1 채움) 또는 footprint 축소. 코지-와이드 재배치(C2)가 외부를 넓힌 뒤 코어 외관 크기를 안 키운 잔여.
> 2. **집↔밭 거리가 멀다 — "집에서 나오면 바로 밭"을 원함.** 현재 집(NW)·중앙 대형 밭이 떨어져 동선이 길다. 집 문 앞에 밭이 인접(또는 집을 밭 쪽으로)하도록 안식 농원 레이아웃 재배치(스타듀식 "현관 앞 농장"). C2 80×65 배치의 동선 튜닝.
> 3. **실내 가구가 벽에 안 붙고 일부 각도가 애매하다.** 침대·벽난로·책장이 상단 벽에서 떠 있고(flush 아님), 일부 prop은 생성 시 각도가 틀어짐. → 가구 벽 flush 재배치 + 각도 틀어진 prop 교정/재생성(P2.3② 가구 폴리시, 안식 농원 데모 품질).
> 4. **★ 실내 카메라 격리 깨짐(코지-와이드 회귀 — T3보다 광범위, 전 건물 영향).** 실내 cam rect(예: `HOME_HOUSE_CAM_RECT` 20×13타일)가 코지-와이드 뷰포트(960×540 = 30×17타일)보다 작아, 방 밖 외부 풀밭/VOID가 사방으로 샌다("집 안인데 집 밖이 보임"). ADR-0018이 뷰포트만 키우고 실내 방·cam rect를 코지-와이드로 안 키운 잔여(헤드리스 회귀는 렌더가 없어 미감지). → 실내 방 폭·높이 또는 cam rect를 30×17 뷰포트에 맞춰 재튜닝(카페·집·만물상·혼백관·생선가게·목공방·대장간·길드 전부). **별도 코지-와이드 실내 패스로 분리 검토**(T3 데모 전 선행 권장).

## 6. 공통 검증 (구역 task마다)

각 구역 작업(T3~T9)은 동일 게이트를 통과한다:

1. **`map_dump` 육안** — 구역이 의도한 '장소'로 읽히나(밀도 상한 내, 톤 일관).
2. **회귀 N스위트 PASS** — 좌표·충돌·동선·세이브 불변(아트만 변경 = 회귀 0).
3. **봇 21일 9/9** — 막힘 0(봇은 모델 레벨이라 맵 무관, soft-lock 회귀 감지).
4. **부팅 클린** — 헤드리스 import + 120프레임 구동, 에러·경고·orphan·leak 0.

> "도달 가능 구역 맵이 완성됐다"고 **직접 플레이로 확인**되면(옥자님 비주얼 데모 마일스톤=안식+나루, Phase 종료=7구역) → **Phase 3(활동 루프)**.
