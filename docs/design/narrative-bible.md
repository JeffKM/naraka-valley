## 2. 중심 정서 코어 — 미결의 죄

**옥자**는 생전에 타인의 액운·저주를 삼키며 약을 짓던 고독한 약사(무녀)였고, 루카의 저주(탁기)로 죽어가던 플레이어의 **유일한 생명줄 — 은인이자 연인**이었다.

플레이어의 **조용한 죄 = 외면·부재.** 옥자가 자기 피와 영력을 깎아 살려낸 목숨을, 플레이어는 *일에 묻혀 정작 소중한 인연을 돌보지 못하는 데* 썼다. 큰소리 나는 죄가 아니라, *곁의 사랑을 당연히 여기고 들여다보지 않은* 죄.

**봉인 재구성(결정 5):** 강림의 금기 계약은 플레이어를 살려 이승으로 돌려보내며 *옥자에 대한 진실을 플레이어 안에 가라앉혔다.* 기억이 마법으로 *삭제*된 게 아니라, **플레이어가 스스로 들여다보지 않기에 닿지 못한다** — 못 알아봄은 외부 마법이 아니라 *원래 인연을 안 보는 사람*이라는 **죄의 증상**이다(ADR-0005 정신 유지). 그 봉인의 잔해가 평생 가슴속 *원인 모를 공허*로 남았고, 플레이어는 **그 공허마저 외면**했다(→ 프로스티).

**속죄의 여정:** 농사(= 무언가를 *돌보는* 일)로 옥자가 *스스로* 시키고, 플레이어는 메인 4인의 세 조각(감정·사실·행동)과 자신의 공허를 통해 *스스로 안을 들여다보며* 봉인을 푼다. 옥자가 말로 알려주면 속죄가 성립하지 않는다(CONTEXT 불변).

### 2.1 세 죄의 번역 — 이승 옥자 약방에서 (결정 2026-06-22)

플레이어는 옥자의 희생으로 완치되어 *그 신비로운 약방 정원에서* 다시 눈을 뜬다(마을엔 여전히 요괴·영약이 드나드는 판타지 세계 — *오직 플레이어 머릿속의 옥자 기억만 봉인됐을 뿐*). 추상적 "외면·망각·부재"는 그 약방에서 벌어진 **세 가지 구체적 죄**로 번역되고, 각각이 **메인 3인의 조각과 1:1로 맞물린다.**

| 플레이어의 죄 (이승, 옥자 약방) | 계열 | 거울이 되는 조각 |
|---|---|---|
| **① 유품을 태움** — 옥자가 피로 꾹꾹 눌러쓴 약초 조합법 노트, 밤새 간호하며 깎아둔 나무 조각을 "누가 쓰다 버린 잡동사니? 방만 좁아지네"라며 아궁이에 땔감으로 태움. 은인의 흔적을 제 손으로 망각의 골짜기로 밀어넣음. | **망각** | **멜 = 사실/장부** — 옥자 장부의 *사본*을 쥐고 "네가 태운 게 무엇이었는지"를 들이댐 |
| **② 영혼의 통증을 '환각' 취급** — 옥자 기억이 뜯겨 나간 자리에 거대한 공허. 매일 밤 꿈속 검은 마녀모자 실루엣이 울고, 우물가만 가면 가슴이 찢어짐. "큰 병 앓다 나아 헛것이 보이나 보네"라며 술·독초 정화액으로 *감각을 마비시켜* 마지막 연결고리를 지움. 애절한 외침을 미친 헛소리로 외면. | **외면** | **미호 = 마음/감정** — 그 통증이 *진짜 옥자의 마음*이었음을 읽어줌 |
| **③ 정원을 방치하고 떠남** — 옥자가 목숨처럼 가꾸던 약초 정원이 주인을 잃음(옥자는 저승 창가에서 플레이어가 그 정원이라도 돌보며 자길 기억해 주길 바람). "주인도 없는 약방 밭을 내가 왜? 큰 도시 나가 떵떵거리며 살아야지"라며 잡초 구덩이로 방치하고 부유·편안한 삶으로 떠남. | **부재** | **바나 = 행동/목격** — 정원을 *지키지 못한* 죄의 거울(바나의 경비=지킴) |

**거울 치료(결정타):** 나라카에서 옥자가 *괭이를 쥐여주는 것* = 죄 ③의 정확한 반전이다. **이승에서 방치한 정원을, 저승에서 다시 돌보게** 한다 → "농사가 곧 속죄"가 *문자 그대로* 성립. (서랍: 나라카 농장이 옥자 약방 정원의 *메아리/재현*인지 — 정합성 높음.)

### 2.2 클라이맥스 모델 — 봉인→귀환 (결정 2026-06-22)

기억은 *삭제*가 아니라 **뜯겨 나가 나라카에 보관**된다(옥자가 짊어진 것). 게임 내내 플레이어는 *접근 불가*라 주관적으론 "지워진" 것과 같다.

> **봉인의 법칙 (키스톤 세계 규칙, 결정 2026-06-22):** *봉인은 타인의 말로 풀리지 않는다. 봉인된 본인이 스스로 닿아야만 풀린다.* → 옥자도 강림도 조연도 *말로 풀어줄 수 없고*(작법 관습이 아니라 *세계 물리*), 전부 아는 강림조차 이 법칙에 묶인다(§5.2). B5 자각만이 B6 귀환을 연다 — 이로써 "스스로 깨달음"(ADR-0005)이 메카닉으로 정당화된다.

1. **재구성(속죄):** 세 조각(망각↔멜·외면↔미호·부재↔바나) + 자신의 공허(프로스티)로 플레이어가 **스스로** 진실을 재구성한다. 옥자도 강림도 *말로 풀어주지 않는다*(코러스·ADR-0005).
2. **귀환(카타르시스):** 자각으로 속죄가 성립하면 **해방의 순간 봉인이 풀리며 기억이 *돌아온다*** — 계약이 되돌려지므로. → *성숙한 선택의 사랑(재구성)* **위에** *"아, 당신이었군요"의 홍수(귀환)*가 이중으로 터진다.
3. **해방·결혼:** 종신계약 강제 해제, "머무름은 선택", 궁극의 결혼 분기.

## 3. 그날 밤 — 수렴 지도 (대화재)

```
[뿌리]   루카(산 야수)가 플레이어에게 탁기(저승 불치병) 옮김
              │
         옥자(약사·연인)가 정화 약을 달이기 시작
           └ 조력자: 설화(냉기로 식어가는 열을 다스림)·세레나(심해 약초를 나루터로)·켄(우직한 조수)
              │
[탐욕]   미르(도박장 흑막) + 스칼렛(간판 타짜)이 멜의 전 재산을 말림
           └ 멜에게 압박: "오늘 밤 옥자 약방 털면 도박 빚 탕감"
              │
         멜(+부하) 약방 습격 — 채무 장부 소각·영약 강탈 (기름병 깨짐)
           └ 밖에선 루카가 원한(저주 약초 채취)으로 위협·소란
              │
[침입]   바나(흡혈)가 뒷문 잠입 — 옥자/플레이어의 피를 노리고 플레이어 방 앞까지
              │
[방아쇠] 미호(옥자의 친구)가 습격 공포로 여우불 폭주
           ✕ 깨비가 장난으로 숨겨둔 도깨비불과 공명 → 대화재(저주받은 화마, 10배 확산)
              │
[가둠]   바나 패닉 → 침입 은폐·혼자 살려고 플레이어 방문을 밖에서 잠그고 도주(복도 질식사)
         켄이 기둥 받치며 구하려 하나 잠긴 문 못 부숨 → 잔해에 깔려 사망
         세레나가 우물 안에서 실시간 목격하며 슬픈 노래 → 저승 기운에 우물째 황천해로 휩쓸림
              │
[희생]   옥자가 불 속으로 뛰어듦 → 강림(차사) 강림, 옥자의 눈물에 흔들려 율법을 어김
           └ 금기의 일방적 희생 계약:
               · 플레이어 완치·이승 귀환
               · 옥자에 대한 기억을 플레이어 안에 봉인(가라앉힘)
               · 옥자는 나라카 마녀로 종신 귀속
               · 강림은 차사직 박탈·나라카 유배
              │
[탄생]   모찌(옥자의 피눈물 + 저승 영력에서) · 네오(깨진 인과율로 튕긴 강림 명부 파편 + 태엽에서)
              │
[이후]   플레이어 2회차 삶: 원인 모를 공허(=봉인된 사랑의 잔해)를 품은 채 일에만 묻혀 인연 외면(조용한 죄)
           → 죽어 나라카로 끌려옴, 옥자를 못 알아봄
           → 프로스티(외면한 공허가 설산 괴수로 형상화)
```

> **그날 밤 가담 정리:** 메인 4(옥자·미호·멜·바나) + 남요괴 5(깨비·켄·미르·루카·강림) + 여요괴 3(설화·스칼렛·세레나) + 비인간 2(모찌·네오는 *그 밤에 태어남*). 프로스티는 *그 밤의 여파*(플레이어의 공허)에서 태어남. → **전원이 한 밤에 수렴.**

## 4. 앙상블 4링 + 코러스 모델

| 링 | 영화 비유 | 캐릭터 | 역할 / 독점 |
|---|---|---|---|
| **0 주인공** | 주인공 | 플레이어 + 옥자 | 척추(미결의 죄→봉인 해제→화해→해방). 중심 미스터리·앵커 |
| **1 주연급 조연** | 주연급 조연 | 미호·멜·바나 | **곱셈기 독점**(미호 자동화·멜 마진·바나 보호) + **해결 3조각 독점**(감정·사실·행동) + **해방-결혼** |
| **2 엮이는 조연** | 엮이는 조연 | T1 주민 11인 | 그날 밤 *파편 코러스* + 자기 반전 로맨스(작은 속죄) + 테마 복선. **곱셈기·해결 열쇠 없음** |
| **3 군중** | 엑스트라 | T3 단골·사연 한 줄 | 분위기·몰입 |

**코러스 모델(결정 6) — "메인이 특별한 이유"를 지키는 선:**
- 조연 11인은 각자 *그날 밤의 한 파편*과 *자기 죄책감*만 안다(깨비=불씨, 바나=잠근 문, 미르·스칼렛=빚 압박, 켄·설화·세레나=못 구한 부채감…).
- **누구도 중심 진실**(옥자가 *이 플레이어*를 위해 자신을 희생하고 기억을 봉인했다)**을 모르거나, 알아도 말로 풀지 못한다.** 조연은 세계를 깊게·테마를 예고할 뿐, *미스터리를 풀어주지 않는다.*
- 조연의 밤-역할은 *자기 반전 로맨스/속죄 아크*에서 고백으로 드러난다(횡적 콘텐츠).
- **합성은 메인 4인의 세 조각 종류 + 플레이어가 스스로 안을 들여다봐야** 완성된다(라쇼몽 구조). → ADR-0008(곱셈기 독점)·ADR-0005("스스로 깨달음") 보존.

## 5. 캐릭터 바이블 (15인)

> 메인·요괴 주민은 3층(**전생의 인연 / 그날 밤의 인연 / 나라카에서의 연결**). 비인간은 전생이 없어 2층(**탄생의 인연 / 나라카에서의 연결**). 조연은 *생전 죄*(저승에 온 독립적 이유, residents.md 유지)를 별도 보유.

### 5.1 메인 4인 (링 1 + 앵커)

#### 옥자 (Okja) — 앵커 · 궁극의 결혼 분기
- **전생의 인연:** 타인의 액운·저주를 독하게 삼키며 약을 짓던 고독한 약사(무녀). 탁기로 죽어가던 플레이어의 유일한 생명줄 — *은인이자 연인.* 자기 피·영력을 깎아 플레이어의 숨을 붙임.
- **그날 밤의 인연:** 불 속에 갇힌 플레이어를 구하러 화염 속으로. 강림과의 계약으로 자기 존재·기억을 대가로 바쳐 플레이어를 완치·이승 귀환시키고, 자신은 나라카 마녀로 종신 귀속.
- **나라카에서의 연결:** 알아보지만 규칙("스스로 깨달아야 속죄") 때문에 차갑게 괭이를 쥐여줌. 농사로 플레이어의 *외면의 죄*를 속죄시킴. 호감도 트랙은 *미결의 죄 해결 후에야* 열림(궁극의 엔드게임 결혼 분기).

#### 미호 (Miho) — 마음의 조각 (감정)
- **전생의 인연:** 영력 과잉으로 사고 치다 "사람 되는 법 배우러" 옥자 약방에 맡겨진 해맑은 영물(구미호). 옥자와 *이름을 부르는 동등한 친구*(언니 아님).
- **그날 밤의 인연:** 루카·멜의 약방 습격 공포로 여우불 폭주 → 깨비의 도깨비불과 공명해 대화재의 직접 불씨.
- **나라카에서의 연결:** 파괴의 불 → 양육의 불(농사 멘토, **자동화 곱셈기**). **마음의 조각** — 옥자의 숨긴 속정을 읽어 전함("옥자가 매일 밤 네 작물 영수증을 보며 운다").

#### 멜 (Mel) — 사실의 조각 (기록)
- **전생의 인연:** 도박 중독→사채업자. 약재 팔아 돈 모으던 옥자에게 거액의 도박 빚.
- **그날 밤의 인연:** 옥자의 독촉 장부에 맞서 빚 탕감·장부 소각하려 부하 끌고 약방 습격. 기름병 깨고 깽판 → 미호 폭주 유도. 불 속에서 돈자루 챙기려다 가장 먼저 타 죽어 강시가 됨.
- **나라카에서의 연결:** 돈으로 망친 죄 → 카페 경제·출하대 회계(**마진 곱셈기**). **사실의 조각** — 옥자의 숨긴 장부에서 플레이어의 생전 약값 내역 + 봉인된 죄목의 존재를 찾아 전함.

#### 바나 (Bana) — 행동의 조각 (목격)
- **전생의 인연:** 피의 갈망을 못 억제하던 흡혈귀(주거침입범).
- **그날 밤의 인연:** 화재 직전 약방 뒷문 잠입(옥자/플레이어의 피를 노림). 불 치솟자 패닉 → 침입 은폐·혼자 살려고 플레이어 방문을 밖에서 잠그고 도주 → 플레이어가 갇힘. 복도에서 질식사.
- **나라카에서의 연결:** "탐해 침입하고 문 잠그고 도망친 죄" → 밤 경비(**보호 곱셈기**). **행동의 조각** — "내가 문만 안 잠갔어도 사장님이 마녀가 안 됐을 것" + 밤마다 괴로워하는 옥자를 증언.

### 5.2 남요괴 5인 (링 2)

#### 깨비 — 도깨비
- **생전 죄(저승 이유):** "모든 걸 장난으로 만들어 진심을 한 번도 못 보인 죄."
- **그날 밤:** 약방 뒷산 야생 도깨비(메밀묵 얻어먹던 단골). 화재 몇 시간 전 장난으로 마당에 도깨비불을 숨김 → 미호 여우불과 공명해 화마를 10배로 키운 *증폭자.*
- **나라카 연결:** 장난이 대참사가 된 죄책감. 카운터 주변 서성, 장난치며 은근히 도움.
- **반전 로맨스 축:** 영원한 장난꾼 → 결혼=생전 처음 "진심"을 택함. ⚠️ *불은 메카닉 능력으로 쓰지 않음*(백스토리 한정, 미호 양육의 불과 중복 금지).

#### 켄 — 언데드 거인
- **생전 죄(저승 이유):** "겉모습 때문에 괴물 취급받다 단 한 번 터진 분노로 사람을 해친 죄."
- **그날 밤:** 약방 뒷마당에 쓰러진 걸 옥자가 안 무서워하고 치료해 준 인연 → 우직한 조수·친구. 괴력으로 기둥 받치며 갇힌 플레이어 구하려 했으나 바나가 잠근 문을 못 부숨, 옥자가 불 속으로 뛰어드는 걸 보며 잔해에 깔려 사망.
- **나라카 연결:** 은인들을 못 구한 거대한 부채감. 흉터 거구·슬픈 눈, 겉바속촉.
- **반전 로맨스 축:** 무서운 겉↔솜사탕 속 → 결혼=주인공만은 괴물로 안 보고 그 힘을 '지킴'으로 받아줌.

#### 미르 — 이무기
- **생전 죄(저승 이유):** "'용'에 집착해 곁의 인연을 내치고 홀로 천 년을 버틴 오만·고독의 죄."
- **그날 밤:** 멜의 전 재산을 말린 거대 불법 도박장의 *흑막.* 멜에게 "약방 털면 빚 탕감" 지시 → 습격의 실질적 원인 제공자. 멜 사후 인과율에 엮여 승천 실패·타락해 저승으로.
- **나라카 연결:** 자기 탐욕이 옥자·플레이어의 삶을 태웠음을 알아 도도함을 내려놓음. 카리스마 뒤 눈치 보는 츤데레 지배자, 마을 중심.
- **반전 로맨스 축:** 차가운 가면 → 마음 열면 오직 주인공에게 집착·응석.

#### 루카 — 늑대인간
- **생전 죄(저승 이유):** "끓는 충동을 다스리지 못하고 본능에 몸을 맡긴 채 곁을 거칠게 망가뜨린 죄."
- **그날 밤:** 깊은 산 야수 — *플레이어에게 탁기를 옮긴 원흉(모든 비극의 뿌리).* 옥자가 영역 침범·저주 약초 채취·자기 저주를 지움에 원한, 화재 당일 약방 밖에서 위협·소란. 진압 과정의 뒤틀린 인과 불길에 사망.
- **나라카 연결:** 자기 야성이 비극의 시초임을 알아 *지독하게 이성적인* 주민. 플레이어에게 묘한 부채감, 묵묵한 조력자.
- **반전 로맨스 축:** 보름달 본능 억누르기 → 결혼=오직 주인공 손길에만 얌전, 사랑이 짐승을 길들임.

#### 강림 — 폐직 저승사자 ⚠️ *코러스의 예외적 무게*
- **생전 죄(저승 이유):** "직무·규칙 뒤에 감정을 숨긴 채 한 영혼에게 마땅한 자비를 끝내 베풀지 않은 차가운 죄."
- **그날 밤:** 이승 원귀를 인도하던 최고의 차사. 화재 현장에 강림, 죽기 직전의 옥자·플레이어와 대면. 옥자의 눈물에 흔들려 율법을 어기고 *일방적 희생 계약*을 독단 체결 — 플레이어를 살리는 대신 옥자를 마녀로 귀속시키고 *옥자에 대한 기억을 플레이어 안에 봉인.*
- **나라카 연결:** 금기로 차사직 박탈·유배. 자기가 직장까지 잃고 살린 플레이어가 옥자를 잊고 내려온 것에 복잡미묘. 츤데레 가디언.
- **반전 로맨스 축:** 감정 없이 인도만 하던 자가 처음 사랑을 배움 → 서툴게 질투하고 고장 남.
- ⚠️ **확증까지만 — 봉인의 법칙 (결정 2026-06-22):** 강림은 봉인 계약의 *집행자*라 모든 진실을 안다. 그러나 **[봉인의 법칙](§2.2)에 *그 자신이 묶여* 입에 담을 수 없다.** 3단 경계:
  - 🟢 **허용(언제나):** 정서적 존재감·죄책감·복잡미묘한 눈빛, "너에게 빚이 있다"는 *막연한* 무게, ♡3 그날 밤 *자기 파편*("그 밤, 율법을 어긴 계약을 했다")까지.
  - 🟡 **확증선(B5 자각 후): 끄덕임만** — "…그래. 마침내 닿았구나." (밝힘이 아니라 *확인*.)
  - 🔴 **금지(영원히):** 사실 진술(옥자 희생 / 기억 봉인 / 마녀=연인 / 죄목) — 절대 강림 입으로 X.
  - **속죄 거울:** 생전 죄(규칙 뒤에 자비를 숨긴 차가움) ↔ 여기선 *말해주고 싶은 자비를 꾹 참는 것*이 진짜 사랑(말하면 속죄를 빼앗으니까). 차가운 규칙-숨기 → 따뜻한 진실-참기. (말할 뻔하다 법칙에 삼키는 연출 = 명장면 자리, 연출 디테일은 §8.)

### 5.3 여요괴 3인 (링 2)

#### 설화 — 설녀
- **생전 죄(저승 이유):** "따뜻함을 포기한 채 차가운 본질이 곁의 이들을 얼려 해치도록 내버려 둔 체념의 죄."
- **그날 밤:** 설산의 냉기 존재 — 옥자 부탁으로 탁기로 식어가는(타오르는) 플레이어의 열을 다스리던 오랜 조력자. 그날도 약방에 머물다 미호의 화염을 얼음으로 막으려 버텼으나 저주 화마에 영력 소진, 연기 질식 소멸.
- **나라카 연결:** 평소 차갑고 고고하나 플레이어·옥자 앞에선 "불 다 못 꺼 미안하다" 눈물 글썽.
- **반전 로맨스 축:** 마음은 열렸으나 몸이 막음 → 결혼=사랑 위해 생전 처음 "따뜻해지는 법"을 연습(순애보). *체념의 죄*가 *그날 밤 처음으로 막아보려 함*과 메아리침.

#### 스칼렛 — 메두사
- **생전 죄(저승 이유):** "치명적 매력으로 사람을 '돌처럼' 가둬두고 누구에게도 마음을 안 준 교만·소유의 죄."
- **그날 밤:** 미르 도박장의 간판 타짜 — 멜의 전 재산을 직접 말림. 미르 명령으로 멜에게 "약방 안 털면 파멸" 압박 → 도미노 공범. 멜 사후 인과율에 엮여 비극적 죽음.
- **나라카 연결:** 탐욕·손기술 부끄러워함. 도도한 독설 뒤 플레이어·옥자 눈치 보며 챙기는 츤데레 누님.
- **반전 로맨스 축:** 여유만만 ↔ 처음 페이스 잃고 빠짐 → 결혼=처음 자기를 내어줌.

#### 세레나 — 인어 (최고비용·마지막 추가)
- **생전 죄(저승 이유):** "세상을 흔드는 목소리를 나른한 무관심 속에 흘려, 노래에 파멸한 이들을 돌아보지 않은 방관의 죄."
- **그날 밤:** 목소리로 파도 잠재우던 인어 — 옥자가 필요로 한 심해 약초를 나루터로 전하던 동화 같은 인연. 그날도 새 약재 전하러 우물가에서 옥자를 기다리다 화재·옥자의 마녀화를 우물 안에서 실시간 목격, 저승 기운에 우물째 황천해로 휩쓸림.
- **나라카 연결:** 청순한 목소리로 카페 정화, 원흉(멜·바나) 보면 목격자로서 하악질. 플레이어에겐 *일반 생전 기억(바다·삶)*을 자극하는 슬픈 바다 레시피(옥자 봉인엔 직접 안 닿음).
- **반전 로맨스 축:** 평생 누워만 있던 그녀가 사랑 위해 처음 움직임 + 노래를 오직 한 사람 깨우는 데. 결혼은 옥자 비약 퀘스트로 다리(마녀 서비스 한정·옥자 게이트 불가침).

### 5.4 비인간 3인 (링 2 · 전생 없음 · 표정만)

#### 모찌 — 슬라임 (그날 밤 탄생)
- **탄생의 인연:** 옥자가 희생 계약서에 *피눈물로 수인을 찍을 때*, 그 눈물이 저승 영력과 결합해 태어남. → 태생이 *옥자의 사랑과 눈물.*
- **나라카 연결:** 플레이어만 보면 본능적으로 하트 모양으로 부풀어 찹쌀떡처럼 매달림. 맥스 시 홍조+행복 도트 눈입, 결혼 후 이불처럼 펴 껴안아주는 반려 슬라임.
- **본성 축:** 태생이 사랑이라 *받기보다 내어주는* 존재. (residents.md의 "흡수만→내어줌"을 *기원에서부터 사랑*으로 심화.)

#### 네오 — 오토마타 (그날 밤 탄생)
- **탄생의 인연:** 강림의 금기 계약으로 인과율이 깨질 때, 강림 차사 명부에서 튕긴 *차가운 법의 조각 + 부서진 태엽*이 나라카 백자 인형에 스며들어 태어남.
- **나라카 연결:** 깨진 인과율 데이터만 연산, 감정엔 "이해 불가" 뚝딱. 플레이어와 '사랑' 버그 발생 시 스크린 눈에 (>_<) + 볼 열선 과열(홍조). 결혼 후 "제 핵심 코어는 당신에게 종속되었습니다" 고백.
- **본성 축:** 순수 효율·논리 → 사랑이라는 '오류'로 처음 감정을 표현. (강림과 *기원으로 연결*되나, 강림=감정 숨김 / 네오=감정 부재→발생으로 결은 분리.)

#### 프로스티 — 예티 (그날 밤의 여파에서 형상화)
- **탄생의 인연:** 플레이어가 살아난 뒤 *옥자를 망각(봉인)한 대가로* 평생 가슴에 품은 '원인 모를 차갑고 시린 공허·외로움의 영혼 덩어리'가 사후 거대한 설산 털뭉치 괴수로 형상화. → **플레이어가 외면했던 슬픔의 본질 그 자체.**
- **나라카 연결:** 말 못 해도 눈망울·처진 눈썹으로 감정 표현. 지치고 추울 때 품에 안아 푹신함, 결혼 시 눈물 고인 눈·발개진 볼.
- **본성 축:** 거대·말 못 하는 괴수 외형 ↔ 가장 깊은 온기. *플레이어의 공허가 사랑으로 채워지는* 살아있는 은유.

## 6. 척추(spine) — authored 세트피스 골격

결정 1에 따라 옥자 척추는 *진척/호감도 게이트로 열리는 authored 세트피스*로 구성된다. 비트 구조·페이싱은 2026-06-22 grill에서 확정(아래 비트 + §6.1). 세트피스 *연출*과 정확한 도달 수치(밸런싱)는 후속(§8).

- **B0 (온보딩):** 옥자가 종신계약 통보·괭이 떠넘김 + *범용 복선*("난 평범하게 살았다고 믿는다") 한 줄. (이미 T4.1 온보딩에 존재)
- **B1~B3 (중반, 세 조각 = 세 죄 대질):** 메인 3인 호감도 진행으로 각 조각이 *플레이어 자신의 세 죄 중 하나*를 비춘다(§2.1 매핑) — **멜**(사실/장부)→네가 *태운 것*(망각)이 무엇이었는지 / **미호**(마음)→네가 *환각이라 마비시킨 통증*(외면)이 옥자의 진짜 마음이었음 / **바나**(행동)→네가 *방치한 정원*(부재)을 지키지 못한 죄. 각 조각은 *그날 밤 파편*이 아니라 *이승의 죄*를 겨눈다(조연 코러스는 그날 밤만, 메인 3인은 이승 죄까지).
- **B4 (공허 직면):** 프로스티/세레나 레시피 등으로 *플레이어 안의 공허*(봉인 잔해)가 표면화. "왜 이 빈자리가 있나."
- **B5 (재구성=자각):** 세 조각 + 공허가 맞물려 플레이어가 *스스로* 그날 밤·옥자를 **재구성**(속죄 성립). 옥자도 강림도 말로 안 풂(강림은 *정서적 확증*까지만).
- **B6 (봉인 해제·귀환):** 속죄 성립 → 옥자 호감도 트랙 개방 + **봉인 풀려 기억 귀환**("아, 당신이었군요") → 화해.
- **B7 (해방·궁극의 결혼 분기):** 종신계약 강제 해제, "머무름은 선택", 결혼.

### 6.1 페이싱 — 호감도와 게이트 (결정 2026-06-22)

**조각의 깊이 (캐릭터당 3단):**
- **♡3 = 씨앗(암시)** — 미호 "옥자가 운다" / 멜 "사장님 숨긴 장부가 있더라" / 바나 "옥자가 밤마다 괴로워해". *떡밥*만.
- **♡4 = 완전체 조각** — 그 암시가 *플레이어 자신의 죄*를 겨눔(멜→망각·미호→외면·바나→부재). 척추가 실제로 쓰는 열쇠.
- **♡5 (max) = 연애 개시** — 호감도를 다 채우면 *연애*가 열리고, *연애 후 추가 조건* 충족 시 결혼(메인=해방). 조각(♡4 우정)과 연애/결혼(♡5)을 분리 → *셋 다 연애 안 해도* 척추는 풀린다(결혼 시스템 §6.3).

**척추 해결 게이트 = AND(세 조각 전부 ♡4) + 공허 비트(B4) + 조연 11인 전원 ♡3 그날밤 고백(§6.2).** 라쇼몽 구조("셋이 모여야 진실")를 그대로 메카닉으로 — **메인은 *죄의 열쇠*, 조연은 *그날 밤 사건 증거*, 둘이 합쳐져야 재구성(B5).** (대안 OR/threshold는 한 명 스킵이 생겨 비채택.)

**스레딩:** 씨앗(♡3)·공허 암시는 *중반부터* 자연히 깔리고, 완전체 조각(♡4)·재구성·귀환·해방은 *풀 게임 엔드게임*(창발형 유지, 막판 갑툭튀 회피). **슬라이스(14/21일)는 척추 무관** — 미호 ♡1~3 + 범용 복선 한 줄까지만(ADR-0005).

**호감도 차는 법 — 메인 vs 서브 이원화 (2026-06-23 grill):**

척추 도달은 **순수 관계 상태 게이트**(날짜 무관 — 달력 시스템도 아직 없음, 룬팩토리·ADR-0008 "평평≠막힘" 정합)다. "막판 갑툭튀"는 *날짜 바닥*이 아니라 **호감도 진행 자체의 끈끈함**으로 막는다 — 점수가 차도 단계가 즉시 안 열린다:
- **♡0~3 = 단계 관문 이벤트.** 점수가 차도 *그 캐릭터와 특정 장소에서 대화/컷신*을 봐야 다음 칸으로 진급한다(스타듀 하트 이벤트 결). 플레이어가 동선에서 자연히 마주치며 서사를 소화 → **척추 비트(B1~B3 씨앗)·조연 그날밤 고백의 그릇.**
- **♡4~5 = 의지 시험.** 관계 종착지라 점수가 아니라 플레이어의 명확한 *의지* — 특정 행동 수행 or 상징적 정표(§6.3 정표·배우자 방·속죄 deed)를 가져다줘야 열린다.

**메인 3인 = 활동 성과 게이트 (활동→관계, ADR-0008 정합).** 호감도가 *대화/선물 누적*이 아니라 **각자의 속죄 활동 실적**으로 큰다 — 미호 = 작물 누적 수확(/특정 작물 성과), 멜 = 누적 출하 자금·카페 매출, 바나 = 밤 영업 방어 횟수·전투 성과. "함께 일해야 깊어진다"가 문자 그대로 — 옥자 deed 모델(아래)을 메인 3인으로 확장한 것. 곱셈기 종류(자동화·마진·보호)가 다르듯 *성과 게이트 종류*도 달라 그라인드를 피한다. ⚠️ **방향 주의:** 활동이 *관계를 게이팅*(활동→관계)하는 것이지, 관계가 활동을 게이팅(관계→활동)하는 게 아니다 — 후자만 ADR-0008 위반(♡0에서도 활동은 base rate). 대화/선물은 보조 가속(서브처럼 가능)이지 메인의 본령이 아니다.

**서브(조연 T1 11인) = 순수 스타듀식 "쉼터".** 게임 내 성과를 요구하지 않는다. 동선에서 마주칠 때 대화 + 선호 선물로 부드럽게 오른다(기존 `affinity.gd` 두 채널 — 일일 대화 하루 1회 +8 + 선물 일반 +15/선호 +40, 임계 `40 × RUN_DAYS/14`점/칸, ♡0~5). 활동 압박의 *쉼터*가 되어준다. ♡3 그날밤 고백·♡4 생전죄 고백·♡5 연애는 §6.2 4단 아크.

**★ 귀결 — 척추 = 세 활동 모두 (의도된 정합):** 척추 해결 게이트가 `AND(세 조각 ♡4)`(OR 기각)인데 메인 ♡4가 활동 성과 게이트이므로 → **척추(엔드게임 이야기)를 보려면 농사·카페·밤경비 세 활동을 모두 일정 수준 해야 한다.** 버그가 아니라 피처 — 세 조각=세 죄=세 활동의 1:1 거울(§2.1), **"통합 플레이 = 속죄."** 활동 자체는 ♡0에서도 굴러가므로(ADR-0008 "평평≠막힘") 이야기-게이팅 문구 내. ⚠️ *바나 성과 게이트의 "전투 성과"는 전투(Phase 3 맨 끝) 의존* — 1층(얇은 방어)에선 "밤 방어 횟수"로 ♡ 진행, 전투 도입 시 "전투 성과"가 상단 단계에 합류(구현 순서 메모).

**옥자 = 행동으로 갚는 모델 (메인 성과 게이트의 원형, 일반 그라인드 X).** 트랙은 척추 해결(B6) 후 열리고, *세 죄를 행동으로 되돌릴 때* 찬다:
  - *망각 반전* → 태운 유품(약초 노트·나무 조각)을 복원/재현(혼백관 전시 등)
  - *외면 반전* → 마비시키지 않고 공허·감정을 마주하는 비트
  - *부재 반전* → 옥자의 약초 정원(=농장)을 성실히 돌봄
  → 옥자의 사랑은 대화·선물이 아니라 *이승에서 저버린 것을 저승에서 갚는 deed*로 차오른다(세 죄 거울 치료가 곧 엔드게임 로맨스 연료). CONTEXT 옥자 "일반 하트 그라인드 아님"과 정합.

### 6.2 조연 그날 밤 고백 배치 — 전원 필수 (결정 2026-06-22)

조연 11인은 **4단 아크**로, 그날 밤에 *기계적으로* 엮인다(순수 옵션 아님 — 공들인 조연이 배제되지 않게).

- **♡1~2 평소 매력** (residents.md)
- **♡3 그날 밤 고백** — 자기 파편+죄책감만(중심 진실 X, 코러스). *바깥 향한 공유 비극.* → **B5 재구성의 필수 재료.** 11인 전원의 ♡3 고백이 모여야 "그날 밤 사건 전체"가 재구성된다(메인 *죄의 열쇠* + 조연 *그날 밤 증거* = 상호보완). 조연을 만나며 *"다들 그 밤에 있었어?"가 점층적으로 밝아옴.*
- **♡4 생전 죄 *고백*** — 조연이 *자기 생전 죄를 털어놓는 것까지만*. *안쪽 향한 개인 로맨스*가 가까워지는 단계(반전 아님).
- **♡5 연애** → (추가 조건) 결혼 (§6.3).
- **결혼 후 = 반전 로맨스** — residents.md 원안 그대로: *평소 매력적 겉면 ↔ 결혼 후 드러나는 죄에서 비롯된 진짜 면모.* 반전의 페이오프는 결혼 *후*에 온다(♡4 고백과 분리).

**변형:**
- 비인간 3인(모찌·네오·프로스티)은 ♡3 자리에 *탄생/본질 비밀*(죄책감 고백 대신) — 모찌=옥자 눈물 / 네오=강림 파편 / 프로스티=플레이어 공허 그 자체(가장 묵직).
- 강림은 예외(봉인 집행자) — ♡3 고백도 "확증까지만" 경계 적용(서랍 §8).

**안전장치 2개:** ㉠ **소프트 게이트** — ♡3 고백은 *플레이어가 대화재를 이미 접한 뒤* 개방(스포일러·순서 꼬임 방지). ㉑ **로스터 고정** — B5 요구 = *현 11인 로스터*(추후 추가 조연은 요구에서 제외, 척추 요구가 무한정 안 커지게).

**비용(눈 뜨고 수용):** ① 척추 엔딩이 *11인 전원 ♡3 콘텐츠*에 의존 → 전 조연 완성 후에야 엔드게임이 닫힘(번아웃 위험; 완화=척추는 엔드게임이라 어차피 전 조연 완성 후 출시·개발 중 비활성). ② 11인 ♡3 강제 = 무거운 관계 게이트(ADR-0008 "이야기만 호감도 게이팅" 문구엔 부합).

### 6.3 연애 → 결혼 (스타듀식 2단계, 결정 2026-06-22)

- **♡ max → 연애 개시.** 호감도를 다 채우면 *연애*가 열린다(즉시 결혼 아님). 메인·옥자·T1 주민 공통.
- **결혼 = 연애 + 추가 조건 충족**(스타듀식 — 연애 상태에서 조건이 부합하면 결혼).
- **티어별 추가 조건 (결정 2026-06-22):**
  - **주민(T1):** 정표 + *배우자 방*(저승 숲 목공방 집 업그레이드).
  - **메인(미호·멜·바나):** 정표 + 배우자 방 + ***속죄 아크 완료***(그들의 ♡ 속죄 비트 마무리 = *해방* 조건). 메인 결혼 = 해방(ADR-0014).
  - **옥자:** 척추 해결(B6) + *deed-track ♡max*(§6.1) + 정표(특별) + 배우자 방. 궁극·최난도 분기.
- **정표 = 옥자의 소울-바인딩 혼례 부적.** *마녀 = 영혼을 묶는 자* → 옥자가 *죄인을 종신계약(족쇄)으로 묶던* 능력을 *연인을 서약(선택)으로 묶는* 데 쓴다(속죄 거울, ADR-0004 정합 — 강제 결박 ↔ 사랑의 결합). 모든 결혼에 재사용, *마녀 서비스*라 스파인 불가침(residents.md 정합). **단 옥자 본인 결혼은 강림(차사)이 맺어준다** — 옥자를 *계약으로 묶었던 자*가 이번엔 *사랑으로 묶는다*(강림 속죄 거울의 완성, 시적). (정표의 구체 외형·획득 동선은 §8 서랍.)

### 6.4 결혼 경로별 척추 톤 (결정 2026-06-22)

**대전제: "본다"가 속죄, "맺는다"는 선택.** 플레이어의 속죄는 옥자를 *소유*하는 게 아니라 마침내 *보는* 것 → **척추 해결(진실·화해·봉인 해제·플레이어 자신의 해방)은 결혼과 독립**(조각+조연+공허로만 게이트, 누구와 결혼하든 무관).

| 경로 | 척추 | 옥자 | 톤 |
|---|---|---|---|
| **옥자와 결혼** | 해결 | *해방* + 결합 (궁극·최난도 분기) | 카타르시스 정점 |
| **타인과 결혼 (척추 해결 후)** | 해결 | 마침내 *보였기에* 슬픔 치유. 계약 해방·결합은 못 감(안 간 길) | **씁쓸한 축복** |
| **타인과 결혼 (척추 미해결)** | 미해결 | 끝내 안 보임 — 보이지 않게 지켜봄, 공허(프로스티) 잔존 | 조용한 영구 잔향 |
| **무결혼** | 선택 | 해결 여부 따라 | 코지 샌드박스 |

**씁쓸한 축복(A):** 옥자의 가장 깊은 상처 = *플레이어가 자신을 못 본 것*(외면). 척추 해결 = 마침내 *본다*("당신이었군요") → **그것만으로 옥자의 슬픔은 치유**된다(결혼 안 해도). 타인과 맺어도 옥자는 *벌하지 않고 축복* — *"네가 살아서, 누군가를 보고 사랑할 줄 아는 사람이 됐다면, 그걸로 됐다."* 단 *계약 해방·결합*은 옥자 분기 한정(ADR-0014 보존). 지브리식 멜랑콜리, 징벌 아님.

**해방의 두 층:** ㉠ *플레이어 자신의 해방* = 척추 해결(B6, 결혼 독립) — "머무름은 선택". ㉡ *옥자의 해방* = 옥자 결혼 한정(메인 결혼=해방, ADR-0014).

### 6.5 세트피스 연출 — 등급제 + B5 클라이맥스 시퀀스 (결정 2026-06-23)

**연출 형식 = 무게에 비례한 3등급** (제작비를 정서 정점에 집중):

| 등급 | 형식 | 비트 |
|---|---|---|
| **1 (최소)** | 대화박스 + 텍스트 (현 `dialogue.gd`) | B0 온보딩 복선 한 줄·일상 대화 |
| **2 (인게임 스크립트 이벤트)** | 캐릭터 이동·카메라·이펙트 + 대화박스 (스타듀 하트 이벤트 결, 인게임 도트체) | B1~B3 씨앗(♡3, 각 메인 활동 무대)·조연 11인 그날밤 고백·B4 공허 떡밥(프로스티·세레나) |
| **S (풀 컷신)** | 인스타툰체 원화 일러스트 + 텍스트 (원작 IP 룩, Phase 2 에셋) | **B6 귀환·B7 해방** — 게임의 정서 정점 |

**B5 재구성 = 하이브리드 클라이맥스 시퀀스** (장소로 무대 + 행동으로 빗장 + 비주얼로 카타르시스):

1. **발동(트리거):** 척추 해결 게이트 충족(§6.1 — 세 조각 ♡4 + B4 + 조연 11인 ♡3 그날밤 증거) → 척추 시퀀스 발동, 플레이어 발걸음이 **미혹의 숲 깊은 곳 옥자 집**(생전 대화재가 난 약방의 잔해)으로 향한다. ★ 옥자 집은 "숨겨진·게이트, 미스터리 해결 후 열림"(world-map·CONTEXT) → *척추 해결이 곧 옥자 집 도달*이라 지리-서사가 정합.
2. **적막·암전:** 옥자 집 진입 순간 모든 인게임 소음(BGM·환경음) 소거 → 적막. 플레이어 캐릭터 내부에서 스며 나오는 *공허 이펙트* → 오직 플레이어 내면 공간으로 화면 암전(Fade to Black). (B4가 게임 중반 깔아둔 공허의 *회수·정점* — B4 떡밥 ≠ B5 회수, 경계 유지.)
3. **능동 재구성(봉인의 법칙 메카닉화):** 암전된 공허 공간에 그동안 모은 파편(메인 세 조각 + 조연 11 증거 + 자신의 공허)이 **밤하늘 별자리처럼** 떠오르고, 플레이어가 이를 *직접 선으로 잇는* 최소 능동 메카닉을 수행. 마우스·키로 파편을 자기 공허에 연결하는 순간 **"내 봉인에 스스로 닿았다"는 감각이 손끝에 박힌다** → §2.2 봉인의 법칙("타인의 말로 안 풀린다")을 문자 그대로 정당화(자동 몽타주는 이 법칙 위배라 기각).
4. **소프트락 안전장치 = 강림 = 최후의 거울:** 못 풀거나 서사를 놓친 유저를 위해, 내면 공간 한편의 *강림 파편(♡3)* 이 침묵 속에서 올바른 궤적으로 *나직이 끄덕이거나 시선*으로 힌트. 말은 못 하지만(§5.2 🔴 금지) 자각을 유도(§5.2 🟡 확증선 "끄덕임만"의 정확한 구현).
5. **카타르시스 전환:** 마지막 연결 완료 → 가라앉아 있던 봉인이 파쇄되며 **인게임 도트체 → S등급 풀 컷신(B6 귀환)으로 급전환.** "아, 당신이었군요" + 대화재 속 숨겨진 연인의 진실 폭발.

→ **장소(무대) + 행동(빗장) + 비주얼(카타르시스)의 3중 구조.** B5 능동 메카닉의 그레이박스 표현·별자리 퍼즐 난도·B7 해방 연출 디테일은 구현(Phase 3 ④) 시 각론.

> ⚠️ **다음 grill 서랍(§8) 갱신(2026-06-23):** 세트피스 연출(§6.5)·도달 수치 게이팅 구조(§6.1)는 이 세션에서 확정. 남은 서랍 = *절대 수치 튜닝*(활동 경제 Phase 3 후) / 결혼 추가 조건 구체 사양 / 강림 "삼키는" 연출 디테일 / 각 비트 트리거 장소 매핑.
