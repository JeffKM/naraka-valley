# Dear My Naraka — 세계 지도 설계 (맵 확장)

> **상태:** 맵 확장 grill(2026-06-22) 산출물. 토폴로지·건물 로스터·구역별 환경 에셋의 청사진이다. 정체성·결정 근거는 [CONTEXT 지리](../../CONTEXT.md) + [ADR-0014](../adr/0014-relationship-marriage-liberation.md)(사회·결혼·해방)·[ADR-0015](../adr/0015-multi-region-world-map-expansion.md)(멀티 구역·순서). 구역 내부 상세 레이아웃·타일 배치는 빌드 시 구역별로 확정(여기는 넓이의 청사진).
>
> **원칙:** 설계는 8구역 다 그리되 **빌드는 한 구역씩**(빈 맵·번아웃 통제, ADR-0015). 그레이박스 우선(ADR-0001).

## 1. 구조

**분리 맵 + 가장자리/길 전환**(스타듀식). 구역마다 별도 맵, 현재 구역만 로드, 세이브가 현재 구역 추적. 기존 문 전환(`_transition_to`)·카메라 rect 패턴의 확장.

> **★[ADR-0044] 개정(2026-07-01) — 남향 물 축 + pseudo-Z 다단.** 인게임 "맵이 단조롭다" owner 피드백. ① **물 종축 남향화:** 물 구역을 **[북: 나루 마을] → [중앙: 삼도천] → [남: 황천해]** 종형 벡터로 재배치(마을에서 아래로 내려가면 강→바다). ② **마을 내부 강 폐지:** 마을을 동/서로 가르던 수직 강 → **최남단 배후 강**(동선 피로 해소·마을 통짜화, 다리로 낚시터 연결). ③ **전 구역 지형 = 2행 pseudo-Z 다단**([asset-ruleset §4.1]) — 절벽·강둑·해안선에 가짜 고저차 강제(z축 아님). 각 구역 실배치는 그 구역 Slice.

## 2. 토폴로지 (구역 그래프)

```
        [저승 숲]──(숲 안쪽)──[미혹의 숲]
           │                    · 마녀의 오두막 = 옥자 집
           │(숲길)              · 특수 채집
           │
        [업화 갱도]            (채광·혼력 / 갱도 끝 = 전투 던전·체력)
           │  · 갱도 입구 · 업화 대장간 · 모험가 길드
           │(산길)
[안식 농원]──(길)──[나루 마을 = 허브 · 통짜(내부 이분할 강 폐지)]
 · 내 집             서편: 카페(명소·축제) · 메인 집(미호·멜·바나)
 · 큰 밭             동편: 만물상 · 주민 집들
 · 창고             ═══ 최남단 배후 강(동서 횡단) ═══──다리──(나룻터·뱃사공)
 · 축사                         │(남향 하강)
                            [삼도천 강 낚시]   · 혼백관(유품 전시) · 강 낚시터 · 강둑 단차
                                 │(하구, 남향)
                            [황천해 바다 낚시]  · 부두·생선가게 · 바다 낚시터 · 해안 절벽 런
```

```
[나락] = 독립 전투 던전(탈주 잡귀의 근원) — 진입로는 빌드 시 확정(갱도/마을 경유 서랍)
```

**허브 = 나루 마을**(모든 길이 통과, 펠리컨 마을 대응). **홈베이스(안식 농원)는 검증된 기존 무대 보존** — 카페만 마을로 이주해 "출근 동선"을 만든다. **물 구역(삼도천·황천해)은 마을 남쪽으로 하강**([ADR-0044] §2).

## 3. 구역별 건물·피처 + 환경 에셋

스타듀 레퍼런스 스샷에서 각 구역이 쓰는 환경 에셋을 함께 기록(맵 확장 에셋 작업의 출발점).

| 구역 | 활동 | 건물·피처 | 환경 에셋(스타듀 참조) |
|---|---|---|---|
| **안식 농원** | 농사 + 목축[서랍] | 내 집(배우자 방 지원)·큰 밭·창고(저장)·축사 | 기존 도색(풀·길·밭흙) + 농장 건물 외관·울타리 |
| **나루 마을** (강+다리 동/서) | 명소·서비스·거주 | **서:** 카페(옥자·미호·멜·바나·서빙·밤바·축제) · 메인 집 3 **동:** 만물상 · 주민 집들[기존 집 에셋 재사용→본체별 외관 재도색] | 자갈 광장·분수·돌담·강변로·다리·벚꽃·간판·건물 외관(4.14.38) |
| **삼도천** | 강 낚시 | 나룻터·뱃사공 · **혼백관**(유품·기억 전시) · 강 낚시터 | 강물·강변·나무 부두·말뚝 |
| **황천해** | 바다 낚시 | 해변 · 부두·**생선가게**(도구·미끼·물고기) · 바다 낚시터 | 바다·모래·부두·조수웅덩이(4.16.02) |
| **저승 숲** | 채집 | 개방 채집지 · **목공방**(집·농장 업그레이드, 어귀) | 빽빽한 나무·그루터기·덤불·버섯·통나무(4.15.46) |
| **미혹의 숲** | 특수 채집 | **마녀의 오두막 = 옥자 집**(숨겨진·게이트) | 더 어둡고 깊은 숲·연못·이끼·고목(4.15.54) |
| **업화 갱도** | 채광 + 끝 전투 던전 | 갱도 입구 · **업화 대장간**(도구·무기) · **모험가 길드**(전투 장비) | 바위 절벽·동굴 입구·광차 레일·호수·다리(4.16.18) |
| **나락** | 전투(독립) | 본격 전투 던전 — 탈주 잡귀 근원 | 심연·업화·봉인 모티프(신규) |

## 4. 서비스 분산 (스타듀 대응)

카페 출하대(멜, 판매·씨앗)는 유지. 나머지는 분산 — 생선가게(황천해/윌리)·업화 대장간(갱도/클린트)·모험가 길드(갱도/말론)·만물상(마을)·목공방(숲/로빈)·혼백관(나룻터/박물관). 점주는 마을 주민 **T2**(가벼운 호감도 퍼크, 결혼 X).

## 5. 새로 따라 들어온 것 / 서랍

- **바다 낚시**(황천해) — 강 낚시(삼도천)와 별개 무대·어종.
- **목축**(축사) — 맵 확장에선 건물 자리만, 메카닉은 활동 Phase.
- **나락 진입로**, **각 구역 내부 상세 레이아웃**, **밤 바↔나락↔갱도 던전 전투 엔진 공유 구현**(ADR-0011), **결혼 서브시스템 MVP**, **미결의 죄 해결 메커니즘** — 모두 후속.
