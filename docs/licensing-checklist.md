# 상업 라이선스 점검 체크리스트 (Steam 출시 전 필수)

> 나라카 밸리는 **Steam 유료 상업 출시**가 목표다. 사용하는 모든 생성형/외부 에셋 도구의
> 상업적 사용 권리와 저작권 귀속을 **출시 전에 반드시 확인**한다. 약관은 수시로 바뀌므로
> 출시 직전 재확인한다.

## 점검 항목

- [ ] **미드저니 (Midjourney)** — 컨셉·컷신·홍보 일러스트. 상업 사용 가능 플랜인지, 생성물 소유권/재배포 조건 확인.
- [ ] **제미나이 (Gemini)** — 이미지 생성. 상업적 사용 약관 및 출력물 권리 확인.
- [ ] **PixelLab** — 인게임 스프라이트·타일·애니. 상업 라이선스 등급, 게임 내 사용·재배포 조건 확인.
- [ ] **Retro Diffusion** — 인게임 픽셀 에셋. 라이선스(개인/상업), 생성물 소유권 확인. (Replicate 경유 시 해당 약관도)
- [ ] **Suno (또는 Udio 등)** — BGM. **무료 플랜은 보통 상업 사용 불가** → 상업 플랜 필수, 곡 소유권/로열티 조건 확인.
- [ ] **SFX 에셋팩** — 각 팩의 라이선스(CC0 / 로열티프리 / 귀속 필요 여부) 개별 확인 및 출처 기록.
- [ ] **폰트** — UI·로고에 쓰는 폰트의 상업·임베딩 라이선스 확인.
- [ ] **Godot 엔진** — MIT 라이선스(상업 무료). 스플래시/크레딧 표기 요건만 확인.

## 기록 원칙

- 각 에셋의 **출처·라이선스·구매 영수증/스크린샷**을 한곳에 보관한다.
- 라이선스가 불명확한 에셋은 **출시 빌드에 넣지 않는다.**
- AI 생성물의 저작권 등록 가능 여부는 출시 국가 기준으로 별도 확인(예: 한국/미국 기준 상이).
