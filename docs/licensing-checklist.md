# 상업 라이선스 점검 체크리스트 (Steam 출시 전 필수)

> 나라카 밸리는 **Steam 유료 상업 출시**가 목표다. 사용하는 모든 생성형/외부 에셋 도구의
> 상업적 사용 권리와 저작권 귀속을 **출시 전에 반드시 확인**한다. 약관은 수시로 바뀌므로
> 출시 직전 재확인한다.

## ⚠️ 최우선 — 원작 IP·상표 (생성형 에셋보다 먼저)

> 생성형 에셋 약관보다 *먼저·크게* 걸리는 위험이다. 게임 메카닉·장르(스타듀형)는 저작권 대상이 아니라 안전하지만, **실재 브랜드·캐릭터의 권리**와 **상표**는 별개다.

- [ ] **★ 원작 나라카 컨셉카페 IP 라이선스 (최대 리스크 — 출시 전 필수, 이상적으론 추가 투자 전)**
  - **현황(2026-06-22 grill):** 카페 소유자와 **구두/협업 합의만** 있음 — *서면 권리 없음.*
  - **위험:** 옥자·미호·멜·바나·'나라카 컨셉카페' 브랜드·세계관은 카페 사업체 소유자의 저작권·상표. 이를 **유료 상업 게임**으로 각색·수익화 중. 구두 합의는 범위(상업 판매·영토·기간·수익분배·머천다이징·속편)가 불명이고 분쟁 시 약하다 → 카페 측이 수익 배분 요구·takedown·소송 시 **게임 출시·수익 전체가 흔들림**(수년·비용 투자 후 못 파는 최악).
  - **조치:** 출시 전 **서면 IP 라이선스/계약** 체결. 핵심 조항 — ㉠ 권리 범위(게임 각색·상업 판매·전 세계·기간) ㉡ 수익/로열티 분배 ㉢ **게임 코드·자작 에셋=내 소유**, 원작 IP는 라이선스 ㉣ 머천다이징·속편 ㉤ **카페 폐업·매각·관계 종료 시에도 게임 판매 존속(survival) 조항** ㉥ 크레딧·종료 조건. **게임/엔터 IP 변호사 검토 권장.**
  - **사용자 결정(2026-06-22):** 계약 *작업*은 **후순위로 미룸**(지금 안 함). ⚠️ 단 **상업 출시 전 게이트는 유지**(이 항목이 미해결인 채 유료 출시 금지). 참고: 서면화는 *이른 지금*이 협상 레버리지·비용 면에서 가장 유리(성공·관계 변화 후엔 불리).
- [ ] **게임 제목 상표 클리어런스** — 가제 **"My Dear Naraka"**('Naraka' 포함). NARAKA: BLADEPOINT(NetEase) 등과 *게임 카테고리* 충돌·상표 등록 가능성 확인(KIPRIS/USPTO/EUIPO). 'Naraka'는 일반 종교 용어라 단독 상표력은 약하나 출시·상표출원 전 확인. 로고·폰트는 스타듀를 안 흉내냄.
- [x] **스타듀밸리 참고 경계 (저작권)** — ✅ **안전 확인(2026-06-22 grill)**: 메카닉·장르·맵 *구조*(허브 마을·낚시터·광산)는 아이디어라 저작권 무관(스타듀형은 합법 시장). 금지선 = 스타듀의 *실제 에셋·정확한 맵 픽셀·BGM·대사·보호 명칭(Pelican Town·Junimo 등)* 복제. 현재 오리지널 세계관·AI 생성 에셋·오리지널 지명/제목이라 안전권. 가드레일: 펠리컨 마을 구체 배열 복제·타일 트레이싱 금지.

## 기록 원칙

- 각 에셋의 **출처·라이선스·구매 영수증/스크린샷**을 한곳에 보관한다.
- 라이선스가 불명확한 에셋은 **출시 빌드에 넣지 않는다.**
- AI 생성물의 저작권 등록 가능 여부는 출시 국가 기준으로 별도 확인(예: 한국/미국 기준 상이).
