## 맥락

안식 농원은 Slice 1에서 스타듀식 코어 루프(농사·목축·개간·집꾸미기)를 그레이박스로 완성했다. owner가 2026-07-22 라이브에서 "아직 스타듀 문법과 거리가 있다"고 지적하면서, Fable 5 캠페인이 **3축 격차 감사**(맵·농사·UI)와 **타일맵 생성 알고리즘 9계열 서베이**를 병렬 수행했다. 감사가 밝힌 핵심 격차:

- **맵 축(§5 재검증):** P2·P7·P8은 이미 해소(유지), **P3(물 유기형)·P4(절벽 유기형)·P5(덤불 벽)·P6(조닝) 미해소.** 신규 최대 격차 **C-1 = "북단 밀집/남단 공동" 비대칭** — 건물 6동·STARTER_PATCH·대형 밭이 맵 상위 30%에 몰리고 y20~65(맵 55%)에 설계 피처가 0. C-2(Forest Farm 나무 밀도 미달)·C-4(남향 스파인 목적지 없음)·C-5(연못이 활동존과 격리) 동반.
- **농사 축:** 수치·데이터 계층은 대부분 스타듀 정합(유지). 격차는 (1) 체감 루프(도구 스윙 애니 전무·물뿌리개 자원 통째 부재) (2) **에너지 철학 역전** — 스타듀는 괭이·물주기만 과금하고 파종·수확·시비는 무료인데, 우리는 5동사 균일 10 과금이라 "보람 액션에 과세"한다.
- **UI 축:** 정체성 에셋은 유지 대상. 재작업 대부분은 새 아트가 아니라 `inv_frame` 내부를 HanjiUi 문법으로 끌어올리는 배선 교정이며, 이는 이미 [ADR-0048]이 결정한 범위다.

알고리즘 서베이의 결론: **스타듀는 지상 지면에 WFC도 하드 인접-제약 솔버도 쓰지 않는다**([ADR-0058] 재확인). 자연스러움은 손설계 매크로 + 결정적 마이크로 알고리즘(오토타일·CA·저진폭 노이즈·스캐터 분포)에서 나온다. 서베이가 티어별 채택 세트를 산출했다.

이 grill은 그 감사·서베이 결과를 확정 결정으로 박제하고, 기존 시각 결정([ADR-0058] 불변식)과의 충돌을 정합화한다.

## 결정

**안식 농원을 스타듀 문법으로 리메이크하되, 손설계 매크로 불변·결정적 마이크로 알고리즘·흙-지배 base·관계=곱셈기 4대 불변식을 준수한다.** 6부.

### 결정 1 — 맵 남단 조닝 (C-1 해소) *[잠정 — owner 큐 검토 대상]*

y20~65 공동에 **목적존을 손설계**한다:

- **남서 = forage 숲** — 나무 밀집대(C-2)와 연속되는 채집 프레이밍.
- **중남 = 과수원 존** — 혼의 나무를 이전/확장.
- **남동 = overgrown 추가 개간지** — [ADR-0055] 차등 재점령 필드를 확장, 개간 테마 유지.
- **연못 주변 = 물가 활동존** — Slice 3 낚시 앵커. `SPIRIT_POND_RECT` 좌표 앵커의 **바이트를 보존**(save/앵커 불변, `_autotile_pond_siblings` 주석 요구).

십자 길은 **폭을 축소**하고, 남향 스파인을 목적지 없이 뻗던 것을 **과수원-연못에 연결**한다(C-4 해소). 일렬 덤불 12개(완벽 세로 벽)는 **불규칙 나무선 + 바위 클럼프로 대체**하되 SOLID 경계 기능은 유지한다(P5).

> **잠정 사유:** 조닝 좌표 배치는 owner의 레이아웃 취향(Forest Farm "숲에 안긴" 밀도)을 직접 반영해야 하므로, 본 ADR은 **방향과 존 목록만 확정**하고 실좌표는 owner 큐 검토 대상으로 남긴다.

### 결정 2 — 마이크로 알고리즘 채택 세트 (서베이 티어1·2)

**티어 1(즉시·저비용·결정적):**
- **외톨이 영역 flood-fill 정리** — '짙은 타일·노이즈 카펫' 반복 통증의 직접 소거. `_G16_GRASS_PATCHES`·스캐터 산출 직후 min-size 정리 패스.
- **지터드 그리드 스캐터** — 가중 테이블의 순수 랜덤 좌표를 셀당 해시 변위로(뭉침/공백 제거).
- **경계 디더 프린지** — 스캐터 테이블에 경계밴드 가중만 추가.

**티어 2(노이즈 유틸 신설):** `_mask_noise`(Godot 4 FastNoiseLite **OpenSimplex2**, seed=월드시드) 단일 진입점을 먼저 만들고 그 위에 얹는다.
- **저진폭(±1~2px) 도메인 워핑** — 원형 CA·격자 경계티의 정면 해법. `_bake_grass_dirt_wang` 코너·`_G16_GRASS_PATCHES` 문턱 **좌표에만**(픽셀 출력 아님).
- **fBm 2~3옥타브 주파수대 분리** — 존 틴트·변종 다양성.
- **다수결 평활 1~2패스** — CA 클럼프 경계 노이즈 매끈화.

**티어 3(존 빌드 필요분만):** Neyman-Scott 군집 스캐터 + density-mask 곱셈기(꽃패치·나무무리·잡초무리 군락화), Catmull-Rom 강/소로 중심선.

**원칙(불변식):** 전부 **결정적 해시**(`hash(cell,zone,seed)`, 난수 금지) · **매크로 손설계 불변** · **저진폭** · **밭/길/확정 물가엔 도메인 워핑 금지**(인공물 직선·손그림 물가 마스크 락 보존) · 픽셀 그리드 32px 스냅.

**연못 유기화(P3):** 손그림 형태 마스크를 4변으로 확장한다. 마스크는 **오프라인 생성 → 수동 검수 후 승격**(런타임 절차 후처리 아님 — [ADR-0058] 기각 정책 준수). Perlin/구형 value 노이즈는 축정렬 줄무늬로 배제, SIMPLEX_SMOOTH만 사용.

### 결정 3 — 에너지 모델 스타듀 정합 (ADR 개정 성격)

**괭이·물주기만 에너지 과금, 파종·수확·시비는 무과금으로 전환한다.** 현행 5동사(괭이·심기·물·비료·수확) 균일 10 과금을 폐지한다.

- 근거: 스타듀는 괭이·물주기만 스태미나를 쓰고 파종·수확·시비는 무료다. 수확에 과금하면 "보람 액션에 과세"하는 셈이라 스타듀 체감과 정면으로 충돌한다.
- **에너지 풀 그레이박스 압축(MAX 100 · 10/회)은 유지** — 정밀 밸런싱은 Phase 3.
- **안전망:** 과금 계약(괭이·물=과금 / 파종·수확·시비=무과금)을 고정하는 헤드리스 테스트를 신설한다(현재 파종·수확 에너지 계약 테스트 부재).

### 결정 4 — 물뿌리개 용량·리필 도입 *[잠정 — owner 큐 검토 대상]*

- **그레이박스 용량 20칸.** 혼우물(`WELL_RECT`)·연못에서 리필.
- 혼우물 실효화(현재 장식 배치만) + 저승 스프링클러(카탈로그 §1-B 기결정, 미구현)의 가치 상승.
- 도구 티어([ADR-0027](./0027-tool-tier-aoe-access-skill-efficiency.md)) = 물뿌리개 용량 상향 축과 정합.

> **잠정 사유:** 자원 압박(유한 물)의 도입 여부는 코지 톤과 긴장의 균형 결정이라 owner 큐 검토 대상. 용량 수치 20은 그레이박스 값.

### 결정 5 — 도구 스윙 애니메이션 defer 해소

캠페인 §2-2(아트=Claude PixelLab 전담 오버라이드)로 기존 owner-Gemini 대기 defer가 해소된다.

- **괭이·물뿌리개·낫·수확 4모션**을 PixelLab로 생성([ADR-0012] 규격·공통 `proportions` 고정값).
- Gemini 교체용 스펙카드 큐를 병행([ADR-0025] 스펙 게이트).
- 스윙 애니 도입 시 `skill.gd`의 `speed_factor`(현재 no-op) 숙련 실효화가 동시 해금된다.

### 결정 6 — UI는 ADR-0048 실행 (신규 ADR 불요)

UI 축 격차는 이미 [ADR-0048]이 결정한 범위이므로 **신규 ADR을 만들지 않고 0048을 실행**한다.

- **1순위:** `inv_frame` 내부 타이포 통일(`ThemeDB.fallback_font` → `HanjiUi.font()`+먹빛 외곽선) + 위젯 한지 스킨(raw `draw_rect` → `HanjiUi.draw_plate/draw_nine`).
- **2순위:** 상점 아이템 그리드 승격 · 인벤탭 정보패널+닫기X+휴지통 · 정산 아이템 아이콘 내역.
- **정정:** UI 감사 초안의 "도구 아이콘 5종 미존재" 판정은 **코드 검증으로 정정**됨 — `assets/tools/*.png` 실존, `main.gd:251-255`에서 핫바 배선 완료(PR#211). 색박스는 폴백 경로일 뿐. 1순위에서 제외.

## 대안 검토

- **lush 잔디-지배 피벗(base flip)** — **기각.** owner의 Forest Farm 크럭스 결정([homestead 감사 §1])이 흙-지배 base를 확정했고 [ADR-0053]이 유효. 잔디는 흙 위 유기 클럼프로만 부활(결정 2·[ADR-0058] 개정).
- **전 지형 blob 47타일 전환** — **보류.** 히어로 경계에서 owner가 곡률 부족을 특정 지적할 때만 조건부 승격(서베이 "보류"). 손그림 Wang 16타일(corner-matching V2)이 업계 표준 최적점이라 16타일로 충족되면 불요.
- **WFC / 듀얼그리드 / TileMapLayer / 물가 Wang base합성 / 밭 래그드 / 16px 논리** — **재론 금지 유지**(기존 메모리·ADR 박제). 우리 손그림 Wang이 adjacency 손저작 버전.
- **에너지 현행 유지(5동사 균일 과금)** — **기각.** "보람 액션 과세"가 스타듀 체감과 정면 충돌(결정 3).

## 결과

- **긍정:** 검증된 인프라(스캐터·CA·유효마스크·Wang·오토타일러) 위 증분이라 리스크 최저. 남단 조닝이 C-1 비대칭을 직격하고, 마이크로 알고리즘 세트가 P3·P4·P5 유기화를 결정적·저진폭으로 낸다. 에너지 정합이 스타듀 코어 체감을 회복하고, 도구 스윙 애니가 최대 체감 격차를 메운다.
- **부정/비용:** 남단 조닝·물뿌리개 용량은 잠정(owner 큐 검토 대상 — 캠페인 §2-1에 따라 **잠정 확정으로 즉시 착수**하고, owner가 나중에 뒤집으면 그 항목만 재작업). 좌표 이동이 좌표 의존 테스트 4종을 깨뜨린다.
- **영향(빌드 착수 시 동반 갱신):**
  - **좌표 의존 헤드리스 테스트 4종** — `home_expansion_test.gd`(최고 위험, HOUSE_EXT_RECT·STARTER_PATCH_RECT·워프칸)·`livestock_test.gd`(넋우릿간/넋둥우리/silo 좌표·pathing)·`reclaim_test.gd`(ENCROACH_SCAN_RECT·성역)·`well_test.gd`(WELL_RECT).
  - **rect 상수** — `_autotile_pond_siblings`·`_g16_near_building` 상수 동반 갱신.
  - **에너지 계약 테스트 신설**(결정 3 안전망).
- **불변 준수:** [ADR-0044] pseudo-Z·남향 물 축 · [ADR-0053] 흙-지배 base · [ADR-0055] 차등 재점령 · [ADR-0008] 관계=곱셈기 전부 불변.
- **되돌리기 비용:** 세이브 좌표(존 재배치)·에너지 계약·[ADR-0058] 불변식 개정을 수반해 번복 비쌈 → ADR로 박제.
- **후속(빌드):** 워크트리 격리 → 남단 조닝·마이크로 알고리즘·에너지·물뿌리개·도구 애니 슬라이스로 분해 → `game/run_tests.sh` 회귀(좌표 테스트 4종 동반 갱신). 잠정 항목(결정 1·4)도 즉시 착수(캠페인 §2-1) — owner 검토는 사후 일괄.
