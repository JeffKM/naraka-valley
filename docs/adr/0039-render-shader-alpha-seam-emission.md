# 렌더 안정성·특수 연출 규격 — 알파·전이·발광 마스크

고도 엔진 통합 시 헤일로(반투명 엣지)·전이 이음매 깨짐·야간 발광 재작업을 막기 위해 에셋 생성 단계에서 렌더/셰이더 규격을 잠근다. 근거 규칙: [asset-ruleset.md](../design/asset-ruleset.md) §8.

- **알파값 제어:** 스프라이트 픽셀은 풀 불투명 또는 풀 투명(하드 알파)만. 반투명 AA 엣지 금지(후처리 알파 임계). **플레이어 가림 투명화는 베이크 금지 — 엔진 Modulate/셰이더에 위임.** (§1.3 그림자 오버레이는 스프라이트가 아니라 예외.)
- **전이 픽셀(Seam):** 타일·절벽 면 한정, 외곽 1~2 논리px에 2px 디더(체커) 마진 강제(알파 블렌드 아님 — 청키·하드알파 정합). 독립 프롭 불필요.
- **발광 마스크(Emission):** 야간 발광 부위(창·영혼빛)는 생성 시 별도 레이어 `*_emit.png`로 분리 추출 → 야간 라이팅(Phase 3-6) 재작업 0.
