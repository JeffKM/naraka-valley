# 관계 모델 확장: 결혼 가능한 마을 + 해방 엔딩 (ADR-0005 개정)

맵 확장 grill에서 게임의 사회·서사 구조를 **순수 스타듀형(창발형·관계는 덤·엔딩 없음)에서 룬 팩토리/스토리 오브 시즌즈형(농사 + 결혼 + 해결되는 메인 스토리)으로 확장**하기로 했다. ADR-0005의 *미스터리 핵심*은 보존하되, *관계 넓이*(옥자 풀 트랙 없음·동료 개별 인연 없음·단골 가벼움)는 뒤집는다. 이 ADR이 ADR-0005의 해당 부분을 개정한다.

## 무엇이 바뀌나

- **결혼 도입.** 호감도 최상단에 결혼 캡스톤을 둔다. 대상 = 메인(미호·멜·바나) + 옥자(게이트) + 마을 주민 T1.
- **마을 주민 3티어.** 단일 "가벼운 단골"을 분화: **T1 결혼 가능**(풀 캐릭터: 초상화·하트·하트 이벤트·결혼; 목표 남요괴 5+·여요괴 3+·비인간 요괴 3+) / **T2 사귐만**(점주 — 퍼크, 결혼 X) / **T3 순수 단골**(호감도 0).
- **메인 결혼 = 해방.** 메인(옥자 제외) 결혼은 그들의 속죄를 완성해 종신계약에서 *해방*시키고 맺어지는 서사다(ADR-0004 정합 — damned soul을 사랑·속죄로 풀어준다).
- **옥자 = 미스터리 게이트.** 옥자의 호감도는 플레이어가 미결의 죄를 해결한 *후에야* 시작된다 — 생전의 죄→속죄→화해→사랑이 메인 스토리의 *궁극의 결혼 분기*(가장 어려운 엔드게임). 앵커 역할 보존.
- **해방 엔딩.** 종신계약은 기본 형기(강제력=족쇄)이고, 속죄 완성 시 *강제*가 풀린다. 단 해방 = 저승 탈출이 아니라 "머무름은 선택"(사랑하는 이·카페를 떠나지 않음). **스토리 엔딩 ≠ 게임 엔딩**(룬 팩토리형): 메인 스토리는 클라이맥스로 해결되고, 농사·카페 명소 일구기(ADR-0009)·결혼 생활은 계속된다.

## 무엇을 보존하나 (ADR-0005에서)

- 미스터리의 본질: **조용한 죄**(외면·망각·부재), **기억상실 아님**, **옥자 앵커**, **스스로 깨달아야 속죄**.
- **서사 조각·활동 곱셈기는 메인 4인 독점.** 주민은 결혼 *넓이*만 주고, 서사 조각도 곱셈기(미호=자동화·멜=마진·바나=보호, ADR-0008)도 못 준다. 깊이는 메인이 독점한다 → "메인이 특별한 이유" 보존.
- **서사 텍스트는 캐릭터에만.** 활동(낚시·채광 등)은 게이트/열쇠로만.

## Consequences

- **1순위 리스크 = 못 끝냄·번아웃.** T1 결혼 캐릭터 1명 = 프로젝트에서 가장 비싼 콘텐츠 단위(초상화 표정셋 + 하트 단계 대사 + 하트 이벤트 컷신 + 결혼 대사·배우자 행동). 스타듀는 팀이 수년+출시 후로 12명을 만들었다. **완화책: 맵엔 주민 집 *자리만* 그레이박스로 확보하고, T1 주민은 한 명씩 점진 추가**(코어 빌드가 캐릭터 제작에 안 막힘). "한 시스템 100% 끝내고 다음" 유지.
- ADR-0005는 *관계 넓이* 부분에서 이 ADR로 **개정(superseded-in-part)**. 미스터리 핵심부는 ADR-0005가 계속 유효.
- ADR-0009(카페 명소)와 정합: 축제 = 카페 이벤트 데이(별도 마을 광장 없음), 마을 = 카페로 사람이 오는 배경.
- **서랍:** 미결의 죄 *해결 메커니즘*, 결혼 서브시스템 MVP(연애→약혼→배우자 이주→자녀?), T1 주민 최종 수·개성, 각 메인 해방-결혼 서사의 구체 내용. 어느 쪽으로 가도 본 ADR을 깨지 않는다.
