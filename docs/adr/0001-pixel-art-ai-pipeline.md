# 에셋은 픽셀아트 특화 AI로 만들고, 자체 도트화 툴은 만들지 않는다

진짜 스타듀밸리형(탑다운 픽셀) 게임을 1인 + 생성형 에셋으로 만든다. 미드저니/제미나이는 게임 스프라이트(일관된 4방향 캐릭터·애니메이션·격자 타일)를 만들지 못하고 큰 단일 일러스트에만 강하므로, 역할을 **컨셉·컷신·홍보 일러스트**로 한정한다. 인게임 도트는 **PixelLab / Retro Diffusion** 등 픽셀아트 특화 AI로 생성하고 **Aseprite**로 손보정한다.

"미드저니 이미지를 도트화하는 자체 스튜디오 툴"은 만들지 않기로 했다. 이유: ① 단순 픽셀화 필터는 정적 룩만 줄 뿐 일관성·4방향·애니메이션·타일링이라는 진짜 난관을 못 푼다, ② 기성 도구가 이미 그 문제를 전용 모델로 해결했다, ③ 자체 툴 제작은 본 게임 대신 도구를 만드는 야크 셰이빙으로 1인 개발의 전형적 실패 패턴이다.

## Consequences

- 캐릭터 4명의 픽셀 단위 완벽 일관성은 AI가 100% 자동으로 주지 못하므로 Aseprite 수작업 보정이 남는다. 그래도 자체 툴 제작보다 압도적으로 빠르다.
- "에셋 정리·네이밍·엔진 임포트" 수준의 글루 스크립트는 허용한다(변환 엔진 제작과 구분).
