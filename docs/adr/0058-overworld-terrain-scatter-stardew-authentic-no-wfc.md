# ADR-0058 — 지상 지형 변주 = 스타듀-정통 스캐터 (WFC 폐기, 구역-키드 가중 테이블)

- **상태:** Accepted (owner 결정 2026-07-17 — deep-research 후 "피봇 수용")
- **정합/개정 대상:**
  - [ADR-0057](./0057-terrain-tiles-pixellab-lowcolor-crisp-regen-supersede-0043.md)(저색 crisp 베이스 + 디테일=스캐터 오브젝트 구조)를 **확장** — 그 "스캐터 오브젝트 레이어"의 *분포·풍성함* 규칙을 확정한다(생성 방식은 0057, 배치·변주는 본 ADR).
  - [ADR-0053](./0053-homestead-dirt-dominant-ground-overgrown-identity-sealed.md)(흙-지배 base 봉인)·[ADR-0055](./0055-homestead-reclamation-differential-respawn-weeds-cozy.md)(재점령 동적 레이어)와 **정합** — 본 ADR은 *정적 base 변주(스캐터 데칼)*만 다루고 base 흙-지배·재점령 동적 레이어를 건드리지 않는다.
  - [ADR-0031](./0031-mining-combat-woven-mine-graduation-naraku.md)(광산·나락)에 **forward 노트 완화** 부가(아래 §던전).

## 맥락

owner가 "지상 지형 변주를 더 풍성하게"를 WFC(파동함수붕괴, itch.io 맵 생성기 참조)로 시작하려 했다. 착수 전 스타듀 지형 taxonomy를 deep-research(위키 미러 1,999장 + 웹 리버스엔지니어링, 3-에이전트 독립 수렴)한 결과:

**스타듀밸리는 지상 지면에 WFC도 하드 인접-제약 솔버도 쓰지 않는다.** forage·debris는 전역 랜덤 좌표를 뽑아 유효 지형이면 놓는 **독립 롤 + 거부표집**(공간 상관 0)이고, 자연스러움은 [지형-타입 유효 마스크 + 구역·계절별 가중 스캐터 테이블 + 풀의 확률적 이웃-확산(유일한 정당한 인접 메커닉) + 넉넉한 변종]에서 나온다. **WFC는 스타듀식 지상 룩에 오버킬·오프모델.**

owner가 느낀 "부자연·심심함"의 정량 진단: 현재 `_GD_TABLES`가 **단일·전역**이라 구역마다 clutter 믹스가 비슷하고(변주 레버 #2 미사용), **변종 수 부족**(레버 #1·#4). 이미 있는 CA(풀 확산 `_g16_cluster_cleanup`)·유효마스크·클러스터는 스타듀 레시피와 정합.

## 결정

**지상 구역 지형 변주는 WFC 인접규칙 엔진을 짓지 않고, 스타듀-정통 스캐터로 낸다.** 3부:

1. **구역-키드 가중 스캐터 테이블** — `_GD_TABLES`/`_GD_SPARSE`를 구역별로 분화(폴백=전역, 회귀 0). 각 구역이 고유 clutter 정체성(숲=나뭇가지↑·물가=이끼·광산입구=돌↑). *심심함의 최대 레버.*
2. **풀 이웃-확산 CA 되살리기** — 유일한 정당한 인접 메커닉(만개 풀이 흙 이웃으로 확률 확산). 결정적·백트래킹 없음. **파괴적 확산(잡초 번식)은 [ADR-0055] 재점령 소관, 여기 안 넣음.**
3. **변종 확대 = 병렬 아트 트랙** — 로스터·스펙카드([ADR-0025])만 코드 트랙에서 산출, 아트 생산은 owner 병렬. 우선순위: 풀 tuft → 잡초 → twig/stone.

### 불변식
- **저작 맵 불가침**([ADR-0005/0015]): 스캐터는 GROUND/PATH 빈 셀만, 저작 셀 불침범.
- **결정성**: 좌표 해시 시드만(난수 금지) — save.dat·회귀 오탐 방지.
- **퍼포먼스**: 로드 시 1회 bake, 백트래킹 없음(홈빌드 17s·bana_test 행 회피).
- **흙-지배 base 유지**([ADR-0053]): "풀무리 증가"(owner 2026-07-17)는 **스캐터 tuft 클러스터 밀도·가중 ↑**이지 base 잔디패치 부활 아님. `_G16_GRASS_PATCHES=false` 유지.

### 스코프 경계
- **계절-키드 테이블 + 계절 타일시트 스왑**(변주 레버 #1) = **Slice 7**(계절 시스템 미존재라 지금 키잉 금지). 테이블 구조는 나중에 `{region:{season:...}}`로 한 겹 더 키잉 가능하게 설계.
- **던전(광산·나락) 절차 생성** = **Slice 5**. *ADR-0031 forward 완화:* 스타듀 광산조차 WFC가 아니라 **프리팹 방 조립**이다 → S5 던전은 "정식 WFC 확정"이 아니라 **"절차 생성 — WFC는 한 옵션, 프리팹-스티칭이 스타듀 실제"**. S5 착수 grill에서 확정.

## 결과

- **긍정:** 검증된 인프라(스캐터·CA·유효마스크·Wang) 위 증분이라 리스크 최저. 구역별 정체성 + 변종으로 "심심함" 직격. WFC 엔진 야크셰이빙 회피. 안식 농원에서 즉시 육안 개선.
- **부정/비용:** 구역별 테이블·변종은 지속 큐레이션 필요(각 구역 지어질 때 테이블 채움). 변종 아트는 owner 페이스 의존.
- **폐기:** WFC-lite 인접-제약 솔버(설계만 하고 미구현). 던전용 정식 WFC "확정"도 철회(S5 옵션으로 강등).
