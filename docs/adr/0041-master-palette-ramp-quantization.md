# 마스터 팔레트 램프 + 양자화 강제

에셋 통일을 *방향 가이드*가 아니라 *고정 공유 팔레트*로 강제한다. AI가 생성마다 제 색을 고르면 한 맵에 모았을 때 색이 따로 놀기 때문에, 재질별 마스터 램프를 잠그고 생성 후 그 팔레트로 양자화한다. 근거 규칙: [asset-ruleset.md](../design/asset-ruleset.md) §16.

- **재질별 램프**(청키·low detail라 램프당 3~5색): warm 베이스(풀녹·흙갈) / 저승 객체(슬레이트·목재·석재) / 영혼빛(`#60d8f0→#2068e8`) / 앰버 발광 / 외곽선 `#401818` / 그림자 반투명 블랙. §9(따뜻한 베이스+저승 객체)와 정합.
- **강제 = 미디엄 양자화**: 프롬프트 팔레트 명시 + 생성 후 PIL nearest 스냅(`tools/quantize_to_palette.py`). 형태 보존 + 색 통일(전 픽셀 하드 스냅은 디테일 뭉갬이라 미디엄).
- **단일 출처 = `docs/design/master-palette.md`**(스와치 png + hex). 실제 hex는 바닥 타일 warm 재생성 때 큐레이션 확정.
