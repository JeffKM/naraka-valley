# 건물 facade 접지(초록 사각 은폐) · 진입 성역 오버랩 규칙 · 개간지 밀도=clearable debris

> **Status:** accepted (2026-07-06 grill, owner 사인오프 rev3~rev5). **구현 = 다음 슬라이스**(본 ADR은 결정·스펙, 코드 미착수).
> **연관:** [ADR-0053](./0053-homestead-dirt-dominant-ground-overgrown-identity-sealed.md)(흙-지배 지면) · [ADR-0036](./0036-building-front-facade-southward-entry-invariant.md)(남향 진입 불변) · [ADR-0046](./0046-building-door-entry-width-two-tile-default-even-footprint.md)(문폭=진입로=트리거) · [ADR-0035](./0035-homestead-elevation-cliff-overgrown-redesign.md)(overgrown debris) · [ADR-0047](./0047-gemini-full-asset-regen-supersede-adr0001-scope.md)(Gemini facade) · [required-assets-roster.md](../design/required-assets-roster.md) §8.

## 맥락

[ADR-0053] 흙-flip으로 마당이 tan 맨흙 지배가 되자, **건물 facade 스프라이트에 구워진 초록 잔디 지면**이 tan 위에서 뜬금없는 초록 사각으로 튀어 건물이 "둥둥 떠" 보인다(전엔 초록 잔디밭이라 묻혔음). 스프라이트에 지면이 일체형이라 셰이더·단순 코드로 초록만 못 뗀다. 또한 흙-flip이 늘린 "훤한 여백"을 어떻게 채울지(가이드 §2 "빽빽함" vs [ADR-0053] "상흔의 황량함" vs [ADR-0035] "clearable debris")도 정해야 한다.

## 결정

### 1. 건물 접지 = Option C (코드 오버랩 트릭 + 잔디억제 패드), 근본은 아트 이원화

facade 초록 사각의 **근본 제거 = facade 지면을 tan/투명으로 리컬러 재출력**(owner-Gemini 트랙, roster §8 등록). 그 전까지 **코드 슬라이스로 즉시 완화**:
- **잔디억제 패드:** 건물 발치 1~2칸 링에 잔디 패치 생성을 억제해 *깨끗한 다져진 흙 패드* 보장(건물이 잔디에 안 닿음). ※세계 지면은 이미 tan이라 "흙 에이프런"은 무효 — 실효 레버는 잔디억제.
- **오버랩 트릭:** 초록 사각 경계 위 레이어에 데코(통나무·debris 바위·잡초)를 얹어 초록 면적을 시각적으로 가림.

### 2. 오버랩 배치 = 진입 불변식 성역화 (내비게이션 최우선)

시각 접지감보다 **진입 불변식 수호가 최우선**. 공간 제약:

```
        [ 북(N) : 오버랩 가능 ]
      ┌──────────────────────┐
 [W]  │     건물 facade      │  [E]
오버랩 ├─────[ 문 Door ]──────┤ 오버랩
 가능  │//// 진입 성역 ////│  가능
      └────[ 트리거 폭 ]─────┘
        [ 남(S) : 데코 금지 · 흙 패드만 ]
```

- **2.1 남향 성역(Zero-Collision):** 남쪽 정면(문폭 + 좌우 여유 1칸)에 **콜라이더 프롭(SOLID 덤불·바위·통나무) 배치 절대 금지**([ADR-0036]/[ADR-0046]). 오직 평면 흙 패드만.
- **2.2 3면 비대칭 소수(N/E/W):** 초록 은폐 오버랩은 후면·좌우 모서리에만. **정형 격자 빽빽 금지**(테스트 화면 인상 재발) → **비대칭 클럼프 소수**만. SOLID 프롭은 통행 방해 없게 **외벽에 밀착 스냅**.

### 3. 개간지 밀도 = clearable debris 중심 (영구 장식 지양)

빈 tan을 **영구 앰비언트 장식 타일로 메우지 않는다.** 모든 밀도의 기초 = 플레이어가 제거 가능한 **overgrown debris**([ADR-0035] 이승의 미련·업화석·석화 고목):
- 시작 빽빽함 = 서사적 미련·상흔이 뭉친 *제거 대상* → 치우면 그 밑 **상흔의 맨흙 노출**([ADR-0053]) → 가이드 밀도·[ADR-0035] 개간·[ADR-0053] 상흔이 한 서사로 합쳐짐.
- 영구·비-clearable 촘촘한 꽃/풀 스캐터는 **최소** — clearable 루프와 상흔 정서를 둘 다 흐린다.
- 시작 debris **밀도 수치(예: 75%)는 Phase 3 밸런싱 변수**([ADR-0035] 명시).

## 결과

- **내비게이션 안정성:** [ADR-0046] 트리거와 충돌 원천 차단.
- **비주얼 밸런스:** 남향 진입로의 깨끗함 ↔ 3면 거친 비대칭 debris 대비로 초록 사각이 자연스레 파쇄.
- **오버 엔지니어링 차단:** 밀도는 debris 재활용(신규 장식 시스템 발명 없음), 근본 접지는 아트 이원화로 분리.
