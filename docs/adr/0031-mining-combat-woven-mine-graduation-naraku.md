# 채광·전투 = woven 갱도(이중 시계) + 졸업 나락(보스), 갱도 전투는 관계-중립

스타듀 광산처럼 **채광지역에 전투를 다시 짜 넣고**(잡몹 woven), 나락을 **본격 전투(강몹+보스) 전용 졸업 무대**로 둔다. [ADR-0015](./0015-multi-region-world-map-expansion.md) #11의 "갱도 = 채광 위주 + **갱도 끝에** 전투 던전"을 개정하고, [ADR-0011](./0011-combat-health-resource.md)의 "갱도 = 혼력+체력 동시(이중 시계)" 무대 프로필을 *확정*한다.

## 배경 — 풀어야 했던 모순

문서가 갱도-전투 관계를 세 갈래로 엇갈리게 두고 있었다: ⓐ ADR-0011 무대 프로필 = "갱도 = 혼력+체력 *동시*(이중 시계)"(전투가 층층이 짜임) / ⓑ CONTEXT·ADR-0015 = "갱도 = 채광 위주 + 갱도 *끝에* 전투 던전"(끝에서만 전투) / ⓒ CONTEXT = "나락 = 갱도 끝 던전과 *별개*". ⓐ와 ⓑ가 정면으로 충돌했고, 전투 무대가 (가)갱도 본체·(나)갱도 끝 던전·(다)나락 셋으로 흩어져 있었다. 이 grill에서 ⓐ(woven)를 채택하고 (나)를 폐기해 무대를 둘로 정리했다.

## 결정

1. **갱도 = 채광 + 잡몹 woven(이중 시계 부활).** 채굴하며 잡귀와 싸우는 스타듀 광산 그 자체 — 곡괭이=혼력 시계, 몹=체력 시계, 후퇴 판단이 이중. ADR-0011 무대 프로필("갱도 = 혼력+체력 동시")을 확정한다. **"갱도 끝 전투 던전" 중간 층은 폐기**(ADR-0015 개정) — 나락과 위상이 겹치는 잉여 층이었다.

2. **나락 = 강몹+보스 순수 전투(체력만), 갱도 졸업 무대.** 폐기한 "끝 던전"의 역할("채광으로 뚫고 내려가 → 끝에서 전투")을 나락이 승격 흡수한다("채광으로 갱도를 뚫고 내려가 → 나락으로 졸업"). **나락 *최초 해금* = 채광 깊이**(활동 게이트, "평평≠막힘" — base 실력으로 도달) / **나락 *내부 진척 가속* = 바나 관계**(ADR-0021 매크로 당근 그대로). 게이트(채광)와 곱셈기(바나)를 분리한다.

3. **나락만 바나 도메인. 갱도 잡몹 전투 = 관계-중립 채광 곁다리.** 갱도 잡몹은 전투 스킬 base XP만 주고([ADR-0019](./0019-physical-skill-relationship-multiplier-two-axis.md) 2축) 바나 호감도를 키우지 않으며 바나 전투 XP 당근도 안 받는다(낚시의 관계-중립 base-only 루프, [ADR-0030](./0030-fishing-relationship-neutral-dave-reel-fight.md)과 대칭 — 전투판 곁다리). 나락에서만 바나 호감도가 자라고(활동→관계) 바나 당근이 가속한다. 이로써 도메인 일치([ADR-0029](./0029-cafe-relationship-stage-domain-aligned-rewards.md): 호감도 자라는 곳 = 보상 받는 곳)가 깔끔히 보존된다.

4. **깊이 구조는 정반대.** **갱도 = 스타듀 광산형**(영구 하강 + N층마다 엘리베이터 체크포인트, 도달 깊이 보존 — 안정적 채집 무대). **나락 = 해골동굴형**(매 진입 리셋 + 무한/심층, 엘리베이터 약함 — 고위험-고보상 단판). 두 정체성이 깊이 구조로 갈린다(안전·영구 ↔ 위험·리셋).

5. **나락 보스 = 디스크리트 관문 보스.** 깊이 마일스톤마다 보장 출현하는 named 보스(랜덤 엘리트 아님). 리셋되는 건 *그날의 런*이고 "최고 도달 보스" 마일스톤은 영구 보존(바나 매크로 진척의 눈금). 보스 드랍 = 나락 희귀 산출(카페 프리미엄 공급, CONTEXT 바나). **보스 전투 자체엔 메인 서사 텍스트 없음**(CONTEXT "활동=기계적 게이트로만") — 서사는 바나 ♡ 단계 컷신에만, 옥자 연결은 낚시식 *아이템 열쇠*가 아니라 *바나 관계 경유*(나락 진척 → 바나 ♡ → 바나의 옥자 '목격' 조각).

6. **조작 = 장착 전환(자동 공격 없음).** 핫바에서 곡괭이/무기를 골라 들고 마우스 클릭이 장착된 것으로 작동([ADR-0024](./0024-mouse-cursor-control-scheme.md)). 채굴 중 잡몹이 끊으면 무기로 전환해 처리 후 복귀 — 이 *전환 자체*가 이중 시계의 입력이자 "스타듀 광산 손맛". 자동 전투로 깔면 갱도가 "채광 + 배경 소음"으로 죽는다.

7. **무기=모험가 길드(말론) / 도구=업화 대장간(클린트), 겹침 제거.** CONTEXT의 "대장간이 도구·무기 벼림"에서 *무기*를 빼 길드로 통일(스타듀 정확 대응 — 길드=무기·반지·부츠 판매+토벌 보상, 대장간=도구 강화·지오드). 무기 티어 = 도구 티어와 완전 분리(전투 깊이 곡선, ADR-0011 #4). 갱도 잡몹으로 길드 무기를 올려 나락 보스 자격을 갖추는 게 졸업 램프.

## 폐기한 대안

- **"갱도 끝 던전" 유지** — (나) 갱도 끝 던전 + (다) 나락이 위상 중복. woven으로 갱도 자체가 전투를 품으면 끝 던전은 잉여. 무대를 둘로 줄이는 게 동선도 보존(끝 던전→나락 졸업으로 승격).
- **바나 전투 XP 당근을 갱도까지 확장** — 전투 스킬은 하나이니 갱도 잡몹에도 바나 당근을 줄 수 있으나, 도메인 일치(ADR-0029)가 어긋난다(호감도는 나락서 자라는데 보상은 갱도서도 받음). 갱도=관계-중립으로 못 박아 정체성(갱도=채광 노동 / 나락=바나 전투)을 선명히 했다.
- **나락 = 채광과 무관한 독립 병렬 무대** — 졸업 게이트(채광 깊이 해금)가 난이도 램프(갱도 입문 전투 → 나락 본격)를 자연스럽게 만들고 폐기한 끝 던전의 동선을 잇는다. 병렬이면 그 램프가 사라진다.
- **자동 공격 전투** — 이중 시계의 손맛(언제 칼 빼고 언제 캐느냐)을 죽인다. ADR-0024·0002 직접 조작과도 충돌.

## Consequences

- **ADR-0015 #11 개정.** "갱도 = 채광 위주 + 갱도 끝 전투 던전" → "갱도 = 채광+잡몹 woven + 나락 졸업 게이트". 전투 자원 정책(별도 HP)·8구역 지리는 불변.
- **ADR-0011 무대 프로필 확정.** "갱도 = 혼력+체력 동시(이중 시계)"가 모순 해소로 정식 확정됐다. 무효였던 옛 "밤 경비" 행은 그대로 폐기 상태.
- **서랍 유지(최후순위, Phase 3 ⑤ "가장 복잡"):** 무기 강화(Forge/Galaxy/Infinity)·반지·부츠·트링켓·바나 사역마 위상·적/보스 패턴·HP 풀 수치·엘리베이터 간격·바닥 깊이·층당 드랍 곡선·기절 페널티 무게·혼불 바람 정밀 위험-보상 곡선. 이 ADR은 *공간·진행·도메인·조작 구조*만 확정한다.
- **이 구조(woven 갱도 + 졸업 나락 + 갱도 관계-중립)를 되돌리려면** 새 ADR로 대체해야 한다.
