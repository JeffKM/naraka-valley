# 채집 = 관계-중립 발견 루프 + 비-스킬 품질 + 나무 작업 갈래

채집 시스템을 grill(2026-06-29)로 확정했다. 스타듀 채집 *스킬의 본체가 통째로 [ADR-0019](./0019-physical-skill-relationship-multiplier-two-axis.md) 배제 대상*(전부 +양·+질·+가치 = 멜 마진 충돌)이라, 품질·성장 모델이 스타듀와 의도적으로 갈린다 — 미래 독자가 "스타듀형인데 왜 채집 스킬엔 더블·이리듐이 없나"를 묻지 않도록 기록한다.

## 결정

1. **채집 = 두 번째 관계-중립 루프, 단 "앰비언트 줍기"(미니게임 없는 수동 발견).** 낚시(ADR-0030)와 함께 소유 메인이 없어 어떤 호감도 당근도 받지 않는 정직한 노동 루프다. 단 낚시가 *능동 미니게임 노동(긴장형)* 이라면 채집은 *이동 중 줍는 수동 발견(이완형)* — 두 관계-중립 루프의 질감을 가른다(은둔 활동파에게 긴장형+이완형 두 결). **줍기 = 혼력 0**(혼력을 전혀 안 먹는 유일 산출 루프 → "평평≠막힘" 안전판), **나무 작업(벌목·수액) = 혼력 소모**(명백한 노동).

2. **채집 스킬(0~10) = 얕은 2갈래, 스타듀 채집 스킬의 본체는 전부 배제.** ADR-0019 #1이 +가치/+양 특성을 막으므로, 스타듀 Gatherer(더블)·Botanist(이리듐)·Forester(+원목)·판매가↑는 **전부 배제**한다. 남는 건 둘뿐: ㉠ **줍기 갈래 = 혼 감지**(근처/화면 밖 채집물 인지·희귀종 예고 = 순수 *편의*, **미혹의 숲 "길 잃음" 테마를 메카닉으로 살림**) ㉡ **나무 작업 갈래 = 벌목 속도·벌목 혼력 절감**(ADR-0019 line 19 그대로, 피지컬). 성장 손맛은 +양이 아니라 **+발견**으로 채널링(ADR-0019 "새 동사 > 수치 반복" 정합).

3. **채집물 품질 = 비-스킬·portable.** 스킬(배제)·비료(야생이라 없음) 둘 다 품질을 못 주므로, 둘로 쪼갠다: ㉠ **장소 = 종(種) 희소도**(미혹의 숲=희소종 / 저승 숲=일반종 — *무엇을* 줍나) ㉡ **품질 등급 4단계 = 명부의 운(전역값=portable) + 혼 유인제 부적(제작 gear)** — *얼마나 좋은가*. 낚시 품질(크리=스킬 + 퀄리티보버=기어)과 대칭(운=전역 + 부적=기어). 품질이 portable해 **재배해도 깨지지 않는다**(아래 #4). 영향=선물 호감도·판매가·요리 회복 / 무영향=카페 메뉴·가공품(작물 규칙 동일).

4. **자생 씨앗 채택, 단 종 발견 게이트.** 채집물 재배(자급)를 허용하되, **희소종은 탐험으로 *먼저 발견*해야 씨앗을 만들 수 있다** → 발견(탐험) 정체성 보존 + 자급 양립. 재배해도 채집 XP·운기반 품질(스타듀 "밭에서 길러도 채집" 충실). 혼합 씨앗(무작위 절기 작물)도 채택(채집→농사 다리).

## 딸린 결정 (상세 = stardew-systems-catalog §1-C)

- **무대 = 구역 횡단, 숲이 본진**: 저승 숲(일반종·통과형)·미혹의 숲(희소종·막다른) 본진 + 황천해 해변 채집(조개·산호 결) + 업화 갱도 바닥 광물 줍기(채굴 노드와 별개, **줍기=혼력0·채집 XP**, 스타듀 충실).
- **벌목·원목·재성장·수액**: 나무(TREE)→도끼로 벌목→원목(제작+목공방 건축 자재), 코지 재성장(묘목 재심기+느린 자연 재생, 민둥산 죄책감 방지), 수액 채취=게잡이통식 패시브(혼력0). 도끼 티어=큰 나무 접근(ADR-0027).
- **미혹의 숲 게이트**: 진입은 자유(워프 기존), **특수 채집지(희소종)만 도끼 티어로 접근**(ADR-0027 "큰 장애물 접근"). 스토리 게이트(옥자 집=미결의 죄)와 메카닉 게이트(특수 채집지) 분리.
- **목공방 건축 서비스**(집·농장 업그레이드, 로빈 대응)·**목공방 주인** = 별도 결(건물 업그레이드 시스템·서랍). 채집 그릴은 *원목 공급*까지만.
- **산출처**: 카페 가공(멜 공급사슬)·선물(머신, ADR-0032)·카페 일구기 공물(채집 번들 리스킨)·요리·판매·제작. 새 시스템 0.
- **서랍**: 집 동굴 버섯 패시브·버섯 통나무(흡수)·채집물 도감(해방 후 롱테일). 미혹 특수 채집물 일부=[미결의 죄] 환경 단서(비밀 노트 결, 서사 텍스트는 캐릭터에만).

## 고려한 대안

- **채집 스킬 아예 없음(B)**: 도구·탐험만으로 성장. 스킬 5종 대칭이 깨지고 줍기가 영원히 안 늚 → 혼 감지(미혹 테마 정합)를 살리려 기각.
- **품질 = 장소만(초기 Q4)**: 깔끔하나 *재배 시 품질 레버 붕괴*(밭은 저승도 미혹도 아님) → 자생 씨앗과 충돌해 "종 희소도 / 운+부적 품질"로 정밀화.
- **채집물 재배 기각(초기 Q7)**: 발견 정체성은 보호되나 스타듀 자급 의도를 버림 → 품질 portable화로 "발견 게이트 하 재배 허용"이 양립 가능해져 채택.
- **+양/+질을 "편의"로 재해석해 두툼한 스킬(C)**: ADR-0019 정신(+% 그라인드 배제)과 경계선 마찰 → 기각.
