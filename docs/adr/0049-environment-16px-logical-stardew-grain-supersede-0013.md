# 환경 아트 16px 논리 해상도로 하향 — 스타듀 청키 그레인 (ADR-0013 supersede)

> **Status:** accepted (2026-07-04, 16px 실험 GO). **supersedes [ADR-0013](./0013-environment-art-32px-native.md).**

## 맥락

[ADR-0013](./0013-environment-art-32px-native.md)은 환경 아트를 **32px native**로 올려 캐릭터와 픽셀밀도를 맞췄다(ADR-0012의 16px×2 업스케일이 흐릿하다는 피드백 해소). 그러나 owner가 스타듀 실물과 대조하니 **32-native는 스타듀보다 픽셀이 곱고(덜 청키) 따뜻한 시골 농가 그레인이 안 난다**. 2026-07-04 타일 시스템 grill에서 이 긴장을 값싼 실험으로 검증했다([gemini-regen-batch.md §4.0](../design/gemini-regen-batch.md)·[tileset-ruleset.md](../design/tileset-ruleset.md)).

## 결정

**환경/전 인게임 아트의 논리 해상도를 16px로 내리고, 정수 4× 스케일로 렌더한다**(온스크린 타일 64px = 32-native 시절과 물리 크기·프레이밍 동일, *논리 픽셀만 절반=청키 그레인*). 16px 실험이 GO(owner 판정: 필드 128 다운스케일 + 무외곽선 소프트 베이스가 32-native보다 스타듀답다).

- **픽셀밀도는 여전히 전 자산 통일**(ADR-0013의 핵심은 보존) — 다만 32→16 논리로 함께 내린다. 타일만 16으로 내리면 픽셀 그레인이 섞이므로 **캐릭터(480×320→240×160)·건물·프롭·작물·UI도 16 밀도로** 맞춘다.
- **얼굴 가독성 우려 무효화:** ADR-0012가 32-native를 민 1순위 근거는 인게임 얼굴 가독성이었으나, **표정은 인게임 스프라이트가 아니라 대화 초상화가 전담**([ADR-0003])한다. 인월드 스프라이트는 실루엣·복장만 읽히면 되므로 16 밀도로 내려도 그 가독성은 유지된다.
- **아트 문법(동반 grill 결정):**
  - **외곽선 = 스코프 분리(Q1):** 걸어다니는 **베이스 지형(풀·흙길·밭·자갈·모래·숲바닥)=무외곽선·저대비·소프트** / 분리 객체(나무·바위·건물·debris·클럼프)=`single color outline`. [ADR-0035]를 지형 한정 개정.
  - **클럼프(Q5):** 큰 풀 덩어리는 베이스 타일에 박지 않고 **스캐터 프롭**(격자 반복 회피 — 디컴파일: 스타듀 tuft도 런타임 오버레이).
- **지형 생성 파이프라인:** Gemini 고해상 필드 텍스처 → `tools/gemini_grass_to_field.py`(워터마크 제거 + **필드 128px BOX 다운스케일**) → 월드좌표 모듈로 타일링 + 지터 디더 유기적 경계.

## 실행 (핵심 — 저리스크 경로)

**16px 전환은 코어 렌더 코드가 아니라 *아트만* 바꾼다.** 월드 `TILE=32`·좌표·충돌·세이브·
터레인 ID·`.tres` 구조·온스크린 크기(64px)를 **전부 불변**으로 두고, 각 자산의 *유효 해상도만* 절반으로:

- **객체(캐릭터·프롭·건물·실내·작물·UI) = "청키화" ÷2(BOX)→×2(NEAREST)** — 기존 잘 나온 32 자산을
  기계적으로 16 밀도에 맞춘다(드롭인 교체, 회귀 0). `tools/chunkify_asset.py`. ADR-0012 "흐릿함"과
  다른 이유 = crisp NEAREST + alpha 하드에지(뭉갬 방지). ADR-0047이 캐릭터를 선명 우선으로 뽑아둔 게
  이 1스텝 청키화를 깔끔하게 만든다. → **손 재생성 불필요**(96개 Gemini 재생성 ≠ GO의 조건).
- **지형 베이스(잔디·흙) = 새 소프트 Gemini 필드로 교체**(Q1 무외곽선·소프트 — 청키화 대상 아님).
  Gemini 필드 → `gemini_grass_to_field.py` 128 다운스케일 → 16px 타일.

## 결과 (downstream)

- **16px 전환 프로그램:** ①지형(새 Gemini, 파이프라인 확보) → ②객체 전부 `chunkify_asset.py` 일괄.
  **첫 수직 슬라이스 = 안식 농원만**(지형 16px + 농원 가시 자산 청키화 + `map_dump` 육안) → 롤아웃.
  전환 중 픽셀 그레인 일시 혼재 감수(실험 때 수용).
- 온스크린 64px·프레이밍(≈30타일 폭) 불변. ADR-0013 "환경=32 native"·tileset-ruleset §1은 이 ADR로 대체.
