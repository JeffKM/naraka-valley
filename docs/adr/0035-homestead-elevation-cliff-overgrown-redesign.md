# 안식 농원 — 높이 단차(절벽+계단) 도입 + Overgrown 개간 재설계

안식 농원을 **좌우 대칭 중앙 밭**에서 **비대칭 Overgrown 개간지**로 전면 재설계하며, 그 과정에서 **스타듀식 1단 높이 단차**(서쪽 고지 절벽 + 돌계단)를 도입한다. 이는 [ADR-0003](./0003-pixel-spec.md)/[ADR-0018](./0018-cozy-wide-camera-stardew-pivot.md)의 "평면 탑다운" 전제를 바꾸는 결정이라 기록한다.

## 무엇을 / 왜

- **중앙 직사각형 밭 삭제** → 오래 버려져 거칠어진 땅(CONTEXT [온보딩]의 overgrown 개간 시작상태). 본가/창고 남쪽 **5×5 스타터 패치(debris 0%)**만 즉시 경작, 나머지는 debris(이승의 미련·업화석·석화 고목)로 덮인 비대칭 마당.
- **비대칭 건물 재배치**: 본가+창고 북동 저지대 병렬(본채+별채, 자재 동선 최적화), 축사 서쪽 고지 남서 끝.
- **1단 높이**: 서쪽이 한 단 높은 고지(절벽면), 동·남이 저지. **창고 왼쪽 돌계단**이 유일한 연결. 절벽 단면 = 천연 울타리(방목 울타리 면제).
- **고지 = 도구 게이트 해금 마일스톤**: 돌계단을 **업화석(곡괭이)·석화 고목(도끼) 하드 게이트**로 막아, 도구 업그레이드 후 부수고 올라가 [하늘 목장]을 해금한다(개간 Slice 1 보상). CONTEXT "확장만 도구 티어 게이트"([ADR-0027](./0027-tool-tier-aoe-access-skill-efficiency.md)/[ADR-0033](./0033-foraging-discovery-loop-nonskill-quality-tree-work.md), 평평≠막힘 [ADR-0008](./0008-growth-model-relationship-multiplier.md))와 정합 — 저지 스타터+0티어로 농사는 처음부터 완전 작동.
- **영혼빛 연못**(중앙-약간 서쪽, 비정형): 휑한 중앙의 비주얼 앵커 + 물뿌리개 수급처 + 텐션 낚시(메카닉은 Slice 3).

## 높이의 구현 본질

z축 높이가 아니라 **2D 평면 + 절벽 타일(통과불가 충돌·시각 단면) + 계단(통과 가능)**이다. 좌표·카메라·세이브는 평면 그대로, 시각·충돌·동선만 추가된다(스타듀의 절벽도 동일 — 진짜 3D 아님).

## 트레이드오프

비주얼 풍성함 + 개간 게이트 마일스톤("농사→도구→영토 확장"의 상위 동기) ↔ 절벽 타일셋·절벽 충돌·계단·비대칭 빌드 로직의 복잡도. 되돌리기 비쌈(맵 빌드·아트·회귀 좌표에 깊이 박힘). 대안(평면 유지 + 시각적 둔덕)은 owner가 반려 — 진짜 단차의 공간감·게이트 낭만을 택함.

## 범위

지금(S0)은 **레이아웃 + 아트**만 — debris·게이트는 "치울 수 있는 객체" 구조로 배치하되 실제 파내기·해금 메카닉은 **Slice 1 개간 그레이박스**로 미룬다([ADR-0028](./0028-build-workflow-greybox-art-interleave-no-fun-gate.md) 인터리브). 높이 도입은 **안식 농원 한정**, 다른 구역 확장은 별도 판단. 아트는 [ADR-0026](./0026-stardew-art-style-pivot-slim-cast.md) 마스터 스타일(single color outline·low detail·거친 도트, tileset-ruleset.md 개정본)을 따른다.
