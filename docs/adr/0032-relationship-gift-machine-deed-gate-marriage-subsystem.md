# 관계 시스템 grill: 선물 머신 복원(활동→deed 게이트 이동) + 결혼 서브시스템(질투·동행 혼·룸메이트) + 옥자 엔드게임 트랙

> **갱신:** 2026-06-29 (관계 시스템 grill — 스타듀 전수 큐 ⑤).
> **개정:** [ADR-0017](./0017-main-affinity-activity-gate.md)("메인 호감도는 대화/선물이 아니라 활동" → 폐기, 선물 머신 복원 + 활동은 *척추 서사 deed 게이트*로 이동).
> **해소:** [ADR-0014](./0014-relationship-marriage-liberation.md)가 결혼 서브시스템 grill로 미룬 서랍 — 질투·자녀·룸메이트 확정.
> **정밀화:** [ADR-0014](./0014-relationship-marriage-liberation.md) 옥자 결혼 타이밍("미스터리 해결 후") → *미혼 완주자 전용 엔드게임 트랙*으로 못 박음.
> **불침범:** [ADR-0008](./0008-growth-model-relationship-multiplier.md) #1 "평평≠막힘", [ADR-0004](./0004-atonement-theme.md) 속죄 테마, [ADR-0021](./0021-relationship-reward-light-affinity-narrative-primary.md)/[ADR-0029](./0029-cafe-relationship-stage-domain-aligned-rewards.md) 도메인 일치 당근.

## 맥락

관계 시스템은 이미 ADR이 가장 두껍게 깔린 영역이다(0008→0017→0021→0029 동력 모델, 0014 결혼, 0022 이혼). grill ⑤는 새 설계보다 **(a) 흩어진 동력 정의의 잔재 정리 + (b) 🔧부분 디테일 채우기 + (c) ADR-0014가 서랍에 둔 결혼 서브시스템 확정**이 본체였다. 다섯 매듭이 풀렸다.

가장 큰 전환은 **IP 주인(옥자님)의 요청** — *메인 캐릭터들도 스타듀 선물 머신을 풀로 갖고, 선호/혐오는 캐릭터 뒤 **실제 인물**의 진짜 취향을 반영*. 이는 ADR-0017의 핵심 문구("메인 호감도는 대화/선물이 아니라 활동")를 정면으로 뒤집는다. 그러나 ADR-0017이 *지키던 벽* — **"통합 플레이 = 속죄"**(척추를 보려면 농사·카페·전투를 모두 해야 함) — 은 무너뜨릴 수 없다(ADR-0004 정합). 그래서 선물 머신을 들이되 그 벽을 *다른 위치에* 다시 세웠다.

## 결정

### 1. 선물 머신 복원 + 활동은 "척추 서사 deed 게이트"로 이동 (ADR-0017 개정)

- **메인 3인(미호·멜·바나) 하트 미터 = 스타듀식.** 선물·일일 대화·활동 실적이 *모두* 채운다. **선호/혐오는 실제 인물의 진짜 취향 반영**(옥자님 요청). 옛 "메인은 선물 아닌 활동" 폐기.
- **단 속죄 서사 진급(♡ 단계 관문 컷신·[미결의 죄] 조각)은 그 캐릭터의 활동 deed 문턱을 추가로 요구.** 선물로 하트는 차도 *미호 농사 / 멜 카페 / 바나 전투 deed* 없이는 속죄 비트가 안 열린다.
- 결과: **"하트=애정은 선물로, 속죄 서사는 행동으로만."** ADR-0017의 "활동→관계 동력"이 폐기가 아니라 *"활동→척추 서사 deed 게이트"로 이동* → 통합 플레이=속죄(ADR-0017 ★) 보존, 선물 머신과 양립.
- 일상 텍스처(선물·생일·선호/혐오)가 가장 공들인 4인에게 **부활**한다.

### 2. 옥자 = 활동 게이트의 예외, 미혼 완주자 전용 엔드게임 트랙 (ADR-0014 정밀화)

- 옥자에겐 *게임 중* 플레이어-구동 호감도 미터가 **없다**(미호·멜·바나의 활동 트랙과 다름). "옥자와 관계 쌓기" = 곧 **[미결의 죄] 미스터리 진척**(미호·멜·바나 ♡4 도달이 게이트, 옥자는 *보이지 않는 앵커* — ADR-0017 `AND(세 조각 ♡4)`에 옥자가 없는 것과 정합).
- 스타듀식 호감도 쌓기·선물·옥자 [결혼]은 **메인 스토리 클라이맥스(미스터리 해결) 이후 + *미혼 완주자*에게만** 열리는 궁극 트랙(미혹의 숲 옥자 오두막).
- 클라이맥스 시점에 이미 미호·멜·바나 중 누군가와 결혼했으면 → 옥자 트랙은 닫히고 [해방]의 *씁쓸한 축복*으로 귀결(ADR-0014와 양립 — 옥자 결혼은 *클라이맥스 선택지*가 아니라 *미혼 완주자 전용 진엔딩성 분기*).

### 3. 배타성 = 단일 ♡5 슬롯, 질투 = 카페 가족 서사 반응 (ADR-0014 질투 서랍 해소)

- **♡0~4 우정엔 배타성·질투 0**(척추가 셋 다 ♡4를 *요구*하므로 불가침).
- 배타성은 **연애(♡5) 슬롯이 단 1개**라는 *구조*로 강제 — 정표는 한 명에게만, 두 번째 연애는 [이혼/결별](ADR-0022) 후에만.
- **질투 = 카페 가족의 일회성 서사 반응**(메인들이 서로 친구·한 무대 출근·[그날 밤]으로 묶여 *연애를 안다* — 스타듀에 없는 디제시스 장치): ㉠연애 개시 순간 안 뽑힌 메인들의 **작고 회복되는 하트 찌릿함**(며칠 뒤 자동 회복 — 영구 손실·적대 0) + "서운함" 비트(곧 응원으로 정착 = 옥자 씁쓸한 축복의 소형 미러) ㉡이미 연애 중 두 번째 정표 시도 = 침묵 아닌 *인-픽션 거절*.
- **일상 선물엔 질투 0**(선물 머신 보존 — 질투는 *로맨틱 정표*에만 반응). 스타듀 "다중 꽃다발→하트 폭락 그라인드"는 기각(코지 톤·ADR-0022 "적대 없음" 정합).

### 4. 자녀 = 동행 혼 (ADR-0014 자녀 서랍 해소)

- 결혼 후 부부의 [혼례 부적] 결속에서 안식 농원에 *깃들어 태어나는* 작은 새 혼. **생물학적 출산 아님** — [그날 밤]의 모찌·네오 "탄생"·프로스티 "형상화"와 같은 *혼의 형상화* 법칙의 가정·소형판(죽은 영혼이 아기를 낳지 않는다는 망자 설정 정합).
- 결속에서 깃들므로 **성별·생물학 무관**(동성 부부 포함 모든 부부 동일 — 스타듀 입양 분기 불요).
- 속죄 테마: *파괴*(그 밤의 불)에서 났던 모찌·네오의 대구 — *사랑의 결속*에서 깃드는 혼("파괴→양육"의 가족판).
- 스타듀 구조 차용(최대 1~2 혼·아기→유아·저상호작용 코지 존재). [이혼]해도 나라카에 깃든 존재라 *잔류*(불가역·비극 없음). 옵션 엔드게임 코지 롱테일(코어 아님).

### 5. 룸메이트 = 비인간 요괴 플라토닉 동거 (ADR-0014 룸메이트 서랍 해소)

- [결혼]을 원치 않는 플레이어용 *플라토닉 동거* 분기(크로버스 대응). **대상 = 비인간 요괴 3인(모찌·네오·프로스티)만**(표정만·인간화 ✕인 이들이 무연애 동행의 결에 맞음 — 인간형 남5·여3 요괴는 결혼 전용).
- 같은 비인간 요괴를 **로맨스(결혼) OR 플라토닉 룸메이트로 택1**(♡ 트랙·반전 연출은 ♡4까지 공유하다 *서약 시점*에 갈림).
- 룸메이트 = *집 공유 + 동행*만, **연애·혼례 부적·동행 혼·해방 없음**(크로버스가 자녀 없는 것과 동형).
- **가정 동거 슬롯은 단일** = 배우자 XOR 룸메이트(택1).

### 6. 스타듀 디폴트 일괄 채택

선물 머신을 통째로 들였으므로 남은 §1-F 🔧부분이 표준 디폴트로 닫힌다 — **선물 주 2회+생일 제한 / NPC 생일(절기 달력 배치, ×배율) / 배우자 방·이주+가벼운 농장 잡일 / NPC 일정(동선)**. 되돌리기 쉽고 트레이드오프 없는 표준이라 본 ADR에 묶어 기록(개별 결정 아님). 생일은 *실제 인물 생일* 차용 가능(옥자님 협의, 콘텐츠 서랍).

## 폐기한 대안

- **(메인 선물 = 순수 분위기 0 ♡, 활동이 유일 동력)** — ADR-0017 원안. 옥자님이 "선물 머신 풀 + 실제 취향 반영"을 명시 요청해 폐기. 가장 공들인 4인의 일상 텍스처가 죽는 문제도 있었다.
- **(순수 스타듀 — 활동 문턱 없이 선물만으로 척추)** — 나락 한 번 안 가고 바나와 결혼·완주 가능 → "통합 플레이=속죄"(ADR-0004) 증발. deed 게이트로 회피.
- **(옥자 결혼 = 클라이맥스 선택지, 미호·멜·바나와 동급 타이밍)** — "호감도 쌓기는 메인 스토리 후 시작"(주인 의도)과 충돌. 미혼 완주자 전용 엔드게임 트랙으로 양립 해소.
- **(스타듀식 질투 = 다중 꽃다발 하트 폭락)** — 코지 톤·ADR-0022 "적대 없음" 위반. 카페 가족 서사 반응 + 회복되는 찌릿함으로 대체.
- **(자녀 생물학적 출산 / 도착한 어린 망자 입양)** — 망자 설정 충돌 / "어린이 죽음" 무게. 혼의 형상화(동행 혼)로 톤 안전·기존 lore 재사용.
- **(룸메이트 = 서랍/전용 인물 필요)** — 비인간 요괴가 이미 그 니치. 로맨스 XOR 플라토닉 택1로 전용 인물 비용 0.

## Consequences

- **[ADR-0017] 개정:** "메인 호감도 = 대화/선물 아닌 활동" 폐기. 활동은 *척추 서사 deed 게이트*로 이동(통합 플레이=속죄 ★ 보존). ADR-0017 본문의 잔재(바나 "밤 영업 방어"=ADR-0010 폐기·ADR-0031 나락으로 대체)도 함께 정리.
- **[ADR-0014] 서랍 셋 해소:** 질투·자녀·룸메이트 확정. 결혼 블록의 "(자녀?)"·"자녀-이혼 처리 서랍" 닫힘.
- **CONTEXT 갱신:** 호감도(선물 머신+deed 게이트·옥자 예외), 결혼(배타성·질투), 동행 혼(신규), 룸메이트(신규).
- **콘텐츠 부채:** 메인 4인 + T1 주민 11인의 선호/혐오 테이블(실제 인물 반영분은 옥자님 소싱) — Phase 3 콘텐츠. 동행 혼·룸메이트 연출 에셋은 후속.
- **밸런스 노브(설계-잠금 아님):** 선물↔활동의 하트 기여 비율, deed 게이트 문턱 수치, 질투 찌릿함 크기·회복일, 주간 선물 캡 — Phase 3 밸런싱.
- **되돌리려면** 이 ADR을 새 ADR로 대체해야 한다(특히 "선물 머신 복원 + 활동=deed 게이트 이동", "옥자 미혼 완주자 전용 트랙").
