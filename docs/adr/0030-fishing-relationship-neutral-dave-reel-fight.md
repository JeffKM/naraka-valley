# 낚시 = 관계-중립 노동 루프 + 데이브식 릴 격투

낚시 시스템을 grill(2026-06-29)로 확정했다. 두 결정이 스타듀 기본 경로에서 의도적으로 벗어나므로 기록한다 — 미래 독자가 "스타듀형인데 왜 낚시만 이렇게?"를 묻지 않도록.

## 결정

1. **낚시 = 유일한 관계-중립 base-only 루프.** ADR-0008은 모든 활동 메카닉에 *캐릭터별 관계 곱셈기*(미호=자동화·멜=마진·바나=보호, ADR-0021로 가벼운 호감도 당근)를 두지만, 낚시는 "순수 활동"이라 소유 메인이 없어 **어떤 호감도 당근도 받지 않는다.** 가속은 ㉠낚시 스킬(base) + ㉡기어(낚싯대·미끼·태클)뿐이고, 산출물(물고기)은 *하류*에서 멜(카페 재료·출하)·미호(혼의 짐승 양육결)로 합류해 곱셈기 경제와 간접적으로만 만난다. → ADR-0008 #1 "평평≠막힘"의 순수 쇼케이스(은둔 활동파가 관계 0으로 끝까지 파는 정직한 노동 루프 1개를 의도적으로 남김). 관계축을 포기한 대가로 *기어가 낚시의 빌드 표현*이 된다.

2. **핵심 메카닉 = 데이브 더 다이버식 릴 격투**(스타듀 "바 유지" 기각). 캐스팅→입질→'당김/풀기' 텐션 줄다리기(물고기 스태미나·발버둥 읽기). 작은 물고기=짧은 격투(노동 가벼움), 큰/전설어=점증 릴 파이트=미니보스(깊이·드라마). **탑다운·물가캐스팅 유지**(ADR-0002·0018) — 데이브의 *잠수 서브씬*(B안)은 카메라·스코프(ADR-0028 그레이박스·번아웃 1순위)와 충돌해 기각. 데이브에서 가져온 건 "사냥/격투 손맛"이지 "수중 탐험"이 아니다.

## 딸린 결정 (상세 = stardew-systems-catalog §1-D)

- **낚싯대 4티어**(생선가게 구매, 대장간 도구 트랙과 분리): 각 = 줄 강도(체급 게이트) + 슬롯(미끼/태클). T4=전설 체급. 스타듀 트레이닝(핸디캡)대 컷.
- **기어 = (B) 미디엄**: 미끼(소모품·입질) + 태클(장착·격투) 각 큐레이션 3~4종. base는 맨몸 가동.
- **혼력 = 격투당 체급 비례**(캐스팅·대기=시간만, 큰 놈 포획=혼력 대량). 데이브 "괴물과 싸우면 지친다" 정합.
- **뱃사공 = T2 점주**(황천해 생선가게, 윌리 1:1) — 기어 판매 + 물고기 환전 단일 창구, 속죄-중립.
- **옥자 게이트 열쇠 = 접근 가능한 각본 어종**(황천해 심해, 보장 포획·중티어 OK, 세레나 스레드 정합)이며 **전설 미니보스(선택 프레스티지·T4)와 분리** — 메인 엔딩이 낚시 숙련을 하드 요구하지 않음(은둔 보호·"무조건 출시").

## 고려한 대안

- **잠수 서브씬(풀 데이브)**: 가장 데이브답지만 낚시를 최중량 시스템으로 격상 → ADR-0002(탑다운)·0028(그레이박스·번아웃) 충돌로 기각.
- **낚시를 메인에 묶기**: 곱셈기 일관성은 생기나 "순수 활동" 정체성을 깨고 억지 연결 → 기각.
- **옥자 열쇠 = 전설 클라이맥스**: 극적이나 메인 엔딩이 낚시 숙련을 하드 요구 → 비-낚시 플레이어 벽 → 기각.
