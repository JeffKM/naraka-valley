# 까마귀 작물 피해 + 허수아비 방어 재도입 (2026-06-24 기각 뒤집기)

2026-06-24 grill에서 "허수아비 / 까마귀(작물 피해)"는 **❌ 기각**됐다 — "작물 피해 위협은 코지 톤과 충돌(스트레스)". 같은 날 바나의 [밤 경비] 옵트인 **손실-방어 루프도 폐기**됐고(ADR-0021), 그 결과 밤 지표에서 위협 계열이 일괄 정리됐다. owner가 2026-07-05에 이 결정을 **재고**해, 까마귀 위협(A)과 레어크로우 수집(B)을 **둘 다 되살리기로** 확정했다. 대신 영구 손실이 징벌적이지 않도록 스타듀의 3중 안전장치로 묶고, 저승 세계관으로 번역하며, 바나/전투 도메인과 완전히 분리한다.

## 결정

1. **A(까마귀 작물 피해) + B(레어크로우 수집) 둘 다 채택.** 순수 장식이던 허수아비를 *기능 프롭*으로 승격한다(현재 안식 농원의 통과-O 장식 배치는 유효, 그 위에 방어 기능 레이어를 얹는다).

2. **손실 = 스타듀 충실 "영구 삭제".** 무방비 작물은 밤새 까마귀에게 쪼여 **영구 소실**한다(씨앗·자란 날수 증발). 소프트/회복 손실로 완충하지 않는다 — owner는 "진짜 위협"의 농사 스테이크를 원한다.

3. **영구 손실을 공정하게 묶는 3중 안전장치(스타듀 값 그대로).** 이게 기각 사유(스트레스)를 무력화하는 핵심이다:
   - **① 발동 문턱** — 밭에 작물 **15개 이상** 심겨야 까마귀 등장. 초보·소규모 밭엔 위협 없음("평평 ≠ 막힘" 정합, 코지 온보딩 보존).
   - **② 손실 상한** — 한 밤 소실은 **소수로 캡**(심은 수 비례하되 최대 몇 개). 밭 전체가 하룻밤에 증발하지 않음.
   - **③ 자명·저렴한 예방** — 허수아비 하나로 반경 **8칸(248타일)** 영구 방어. 예방이 명확·저비용이라 위협이 "밭 커지면 허수아비 세워라"는 **일회성 크래프팅 넛지**로 수렴한다.
   - (정확한 수치 15·N·8은 그레이박스 서랍. ADR엔 *세 안전장치가 존재한다*는 형태만 못 박는다.)

4. **저승 번역 — 까마귀 = 저승 흉조, 잡귀와 완전 분리.** 날것 "crow"를 넣지 않고, **덜 여문 혼백 작물의 [미련]에 이끌려 쪼아먹는 저승 흉조**로 앉힌다(미호의 방화→양육 테마와 정합, 허수아비=기른 것을 지키는 파수). ⚠️ **잡귀와 별개다** — 까마귀는 전투 대상이 아니라 허수아비로 *접근을 차단*하는 순수 농사 해충이며, 바나·[나락]·전투(체력)와 무관하다. 이래야 "밤 지표엔 (잡귀) 위협 없음"(ADR-0021) 원칙과 공존한다. 명명(예: 미련까마귀)은 그레이박스 서랍.

5. **레어크로우 수집(B).** 8종 = **기능·반경 동일한 저승 테마 스킨**(순수 수집·전시 코스메틱, 서사 없음 — ADR-0005). **기존 서비스에 분산 획득**(혼백관·만물상·카페 이벤트 데이·저승식 전령 — 새 NPC·상점 안 만듦, ADR-0001 정신). **8종 완성 → 디럭스 허수아비 레시피(반경 2배=16칸)** 해금. 수집 추적·전시는 **혼백관**(박물관).

6. **도메인 = 미호/농사, 무게이트 base 인프라.** 허수아비는 스프링클러처럼 **♡0에서도 만드는 base 크래프팅**(재료 = 개간 드랍 원목 등)이다. 미호 호감도로 잠그지 않는다("평평 ≠ 막힘"). 미호 관계 보상(농사 XP 당근, ADR-0021)과 **직교** — 친해진다고 까마귀가 덜 오지 않으며, 허수아비가 유일한 답이다. 허수아비 자체엔 서사 없음(미호는 캐릭터 서사만).

## 고려한 대안

- **손실 성질 (나) 소프트/회복 손실** — 성장 지연·수확 감소만 주고 작물은 안 죽임. 기각 사유를 더 확실히 없애지만 owner가 "진짜 위협" 부재를 이유로 반려.
- **손실 성질 (다) 텔레그래프+예방만** — 손실은 진짜지만 경고+예방으로 일회성 넛지화. (가)+안전장치 ③에 흡수됨(별도 채택 불요).
- **B만 살리기(수집 메타만, 위협 없음)** — 가장 코지-안전하나 owner가 A의 농사 스테이크를 원해 반려.
