# 이혼: 해방은 불가역, 혼인(곁의 형태)만 해소 — ADR-0014 확장

> **확장:** [ADR-0014](./0014-relationship-marriage-liberation.md)(결혼·해방)가 결혼 캡스톤을 정의했으나 *이혼*은 서랍으로 남겼다. 이 ADR이 그 빈칸을 채운다.
> **정합:** [서사 바이블 §6.4](../design/narrative-bible.md)의 "본다=속죄(불가역) / 맺는다=선택" 분리 · [ADR-0004](./0004-atonement-theme.md) 속죄 테마 · [ADR-0021](./0021-relationship-reward-light-affinity-narrative-primary.md) 관계 보상 모델.

## 맥락 — 이혼이 나라카에선 그냥 "되돌리기"가 아니다

스타듀의 이혼(50,000g → 배우자 ♡0·적대화, Dark Shrine 기억 삭제·자녀 비둘기화)은 ㉠실수 무르기 ㉡경로 갈아타기 ㉢어두운 콘텐츠를 위한 QoL이다. 그런데 나라카 결혼은 **3티어**라 이혼의 *의미*가 티어마다 갈린다:

- **주민 T1** — 해방 없는 순수 스타듀식 넓이. 이혼은 코히어런트하다.
- **메인 3인(미호·멜·바나)** — 결혼 = 속죄 완성 = **[해방]**(종신계약 족쇄 해제, "머무름은 선택"). 이혼이 *속죄를 되돌린다(재결박)*면 ADR-0004 테마를 정면 파괴한다.
- **옥자** — 결혼 = 척추 해결 climax + "당신이었군요" 카타르시스 정점 + 강림이 맺어줌. 이혼이 *중심 미스터리 해결*을 무르면 정서 정점이 붕괴한다.

즉 메인·옥자 이혼을 스타듀식으로 그대로 들이면 게임의 척추와 충돌한다([stardew-systems-catalog](../design/stardew-systems-catalog.md) ⚠️ 충돌 항목). grill(2026-06-24)에서 이 가지를 따라 내려가 결론을 냈다.

## 결정

**이혼을 둔다(전원 — 주민·메인·옥자). 단 *해방·화해·진실은 불가역*이고, 해소되는 것은 *혼인(곁에 머무는 형태)*뿐이다.**

1. **해방 ≠ 혼인 (핵심 분리).** 서사 바이블 §6.4가 이미 "본다=속죄(불가역) / 맺는다=선택"으로 둘을 갈라 뒀다. 이혼은 *선택(혼인)*만 되돌리고 *성취(해방·화해·미스터리 해결)*는 건드리지 않는다. **메인은 해방된 영혼으로 카페에 그대로 잔류**한다 — 종신계약은 이미 풀렸고(사랑이 풀었음, 영구), 다만 혼인 관계가 끝날 뿐이다. *재결박은 없다.*

2. **푸는 주체 = 묶은 자의 거울.** 정표 = 옥자의 소울-바인딩 부적이므로, **메인 3인 혼인 해소 = 옥자가 부적을 푼다**(영혼을 *묶는* 마녀 ↔ *푸는* 마녀). **옥자 본인 혼인 해소 = 강림이 푼다**(맺어준 자 ↔ 푸는 자 — 강림 거울의 대칭). 별도 던전·전용 아이템 없이 기존 lore로 자족.

3. **전배우자 = 적대 없음(씁쓸·우호적) + 호감도 0 리셋.** 나라카 톤(코지·코믹 + 속죄)에 스타듀식 "차갑게 적대하는 전배우자"는 안 맞는다 — 메인은 *당신의 사랑이 풀어준* 영혼이라 고마움이 남는다. 단 관계 수치는 ♡0으로 리셋(이혼의 실질 페널티).

4. **재구애 가능, 단 authored 콘텐츠는 재탕 없음.** ♡를 다시 올릴 수 있으나 **이미 본 서사 조각(♡4)·하트 이벤트 컷신은 재지급하지 않는다**(이미 받은 보상). 기계적 보상(스킬 XP 당근 등 ADR-0021)은 정상 적용. **재혼(메인) 조건은 가볍다** — 해방·배우자 방은 이미 충족이므로 *♡5 연애 재도달 + 정표 재수여*만으로 재혼.

5. **비용 = 고액 골드(스타듀식).** ♡0 리셋과 함께 이중 마찰을 둬 이혼을 *가볍지 않게* 만든다(잦은 갈아타기 억제).

6. **Dark Shrine 기억 삭제는 기각.** 나라카의 [기억 봉인](강림 금기 계약 = 키스톤 세계 규칙, ADR-0016)은 메인 미스터리의 정서적 심장이다. 이혼 무르기 따위로 같은 장치를 재탕해 싸구려로 만들지 않는다.

## 폐기한 대안

- **(a) 이혼 전면 없음(결혼=영구 캡스톤).** 테마 순수하나 스타듀 QoL을 버리고 변심 플레이어를 리플레이로만 내몬다. 코지 장기전에서 배우자 변경에 *새 세이브 강제*는 과한 마찰.
- **(b) 주민 T1만 이혼 가능, 메인·옥자 영구.** 비대칭이 "메인은 무겁다"를 선언하나, §6.4의 해방≠혼인 분리를 쓰면 메인도 *재결박 없이* 이혼 가능하므로 굳이 막을 이유가 없다(자유 + 테마 둘 다 챙기는 길이 있는데 자유를 뺏음).
- **(c) 전면 스타듀식(메인 해방도 재결박).** 가장 자유롭지만 ADR-0004 속죄·[해방] 서사를 정면 파괴. 해방된 영혼을 다시 족쇄에 채우는 잔혹·부조화.

## Consequences

- **CONTEXT '결혼' 항목에 이혼 절 추가**(불가역 해방 + 혼인만 해소 + ♡0 리셋 + 거울 해소자).
- **[stardew-systems-catalog](../design/stardew-systems-catalog.md)** 이혼 행을 ⚠️ 충돌 → ✅ 결정으로 갱신(부록 A 우선순위 상에서 제거).
- **자녀-이혼 상호작용은 서랍** — 자녀 자체가 ADR-0014 서랍이라, 이혼 시 자녀 처리(스타듀 비둘기화 대응 등)는 자녀 서브시스템 grill로 미룬다.
- **구현은 Phase 3 ④(결혼 서브시스템)와 함께.** 그레이박스 단계에선 ♡ 리셋 + 골드 차감 + 부적 해소 텍스트면 충분(연출 디테일은 에셋 단계).
- **이 모델을 "이혼 없음"이나 "전면 스타듀식(재결박)"으로 바꾸려면** 이 ADR을 새 ADR로 대체해야 한다.
