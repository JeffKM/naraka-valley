# 애니메이션 바디 앵커 고정 + 드롭=UI 아이콘 공용화

애니메이션 지터(프레임마다 접지점이 흔들림)와 아이템 에셋 중복 제작을 막기 위한 애니메이션·UI 에셋 규격. 근거 규칙: [asset-ruleset.md](../design/asset-ruleset.md) §14·§15.

- **바디 앵커 고정:** 모든 워크/동작 시트는 프레임 변동과 무관하게 '발바닥 접지점' 중심 픽셀 좌표를 고정(지터 차단). char_sprite `FOOT_OFFSET_Y`가 구현 — 신규 시트도 발치 y를 동일 좌표에 export.
- **드롭=아이콘 공용화:** 수집/인벤 아이템은 단일 16×16 논리 격자, **필드 드롭 스프라이트 = UI 아이콘 텍스처 1:1 공용**(한 텍스처가 월드 드롭·인벤 슬롯 둘 다). ItemCatalog 아이콘이 공용 출처.
