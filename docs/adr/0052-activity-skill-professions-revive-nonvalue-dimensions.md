# 활동 스킬 전문직 부활(5스킬) — 비-가치 차원 한정 · ADR-0019/0033 개정

> **상태:** 채택(2026-07-05 grill, owner 사인오프 완료). 로스터 세부 수치는 그레이박스 밸런싱 서랍.
> **개정 대상:** [ADR-0019](./0019-physical-skill-relationship-multiplier-two-axis.md)(#1 "전문직 특성 배제" 완화) · [ADR-0033](./0033-foraging-discovery-loop-nonskill-quality-tree-work.md)(#2 채집 스킬 본체 배제 · #3 비-스킬 품질).
> **불변 유지:** ADR-0019 2축 골격 · "숫자 비충돌" 불변식(§13) · "평평≠막힘"([ADR-0008] #1) · 관계 곱셈기/보상 축([ADR-0021]/[ADR-0029]) · 서사=게이트([ADR-0005]).

## 맥락

"꽃 패치"를 **작물/야생 씨앗 시스템**(스타듀 야생 씨앗·꿀 시너지)으로 설계하려던 중, owner가 야생 씨앗 수확 등급을 **"채집 레벨 + 전문직"**으로 정하며 스타듀 전문직을 부활시키기로 했다. 그런데 [ADR-0019]가 전문직 특성을 **통째로 배제**한 이유가 바로 *스타듀 전문직의 핵심이 +판매가·마진이고, 그게 관계 곱셈기(멜=마진·미호=자동화·바나=보호)와 숫자가 겹치기 때문*이었다([ADR-0019] #1, [ADR-0033] 서두). [ADR-0033]도 같은 논리로 채집 스킬 본체(Gatherer 더블·Botanist 이리듐)를 배제하고 품질을 비-스킬(운/부적/희소도)로 돌렸다.

owner grill(2026-07-05)에서 이 배제의 *진짜 범위*를 다시 갈랐다:

- **배제의 본래 표적 = "+판매가/마진" 딱 하나.** 그게 멜의 *단위당 골드* 곱셈기와 같은 숫자 슬롯을 밟는다.
- **품질 등급·수량(2배·+1)·효율·편의는 다른 슬롯이다.** 품질/수량은 *생산 시점*에 "더 좋은/많은 물건"을 만들고, 마진은 *판매 시점*에 붙는다 — 곱해질 뿐 겹치지 않는다([ADR-0019] §13 "숫자가 겹치지 않는다"의 정확한 실현).

실증: 스타듀 전문직의 **약 80%가 이미 +판매가가 아니다** — Gatherer(2배)·Botanist(이리듐)·Miner(+1광석)·Prospector·Excavator(2배)·Pirate(보물2배)·Mariner·Luremaster·전투 6종(골드 0). 이들은 +가치 없이도 최고 인기다. +판매가/마진 계열(Tiller·Rancher·Artisan·Blacksmith·Gemologist·Fisher·Angler·Tapper·Forester)만 도려내면, 관계 곱셈기를 훼손하지 않고 전문직의 매력 핵심을 그대로 부활시킬 수 있다.

## 결정

**활동 스킬 5종(농사·채광·채집·낚시·전투)에 스타듀식 전문직 트리(lvl5 2갈래 → lvl10 각 2분기, 스킬당 선택 5개)를 부활시킨다. 단 퍼크는 *비-가치 4차원(품질 등급·수량·효율·편의)*으로만 한정하고, +판매가/마진은 관계 곱셈기(멜/미호/바나) 전용으로 남긴다.**

### 1. 허용/금지 차원 (숫자 슬롯 분리)

| 차원 | 담당 | 예 |
|---|---|---|
| **품질 등급** (일반→은→금→이리듐) | ✅ 전문직 | 약초학자=채집물 최고 등급 고정 |
| **수량** (2배 드롭·+1) | ✅ 전문직 | 채집꾼=20% 2배, 광부=+1 광석 |
| **효율** (혼력 감소·속도·성장 가속) | ✅ 전문직 | 재배가=성장 10%↑ |
| **편의/발견** (감지·미끼 면제·부화) | ✅ 전문직 | 추적자=채집물 위치 표시 |
| **+판매가 / 마진 / 단가** | ⛔ 전문직 금지 → 관계 곱셈기 전용 | 멜=서빙 마진, 미호=자동화, 바나=보호 |

**"더 좋게·많이·빠르게" = 전문직(내 노동의 숙련) / "더 부유하게" = 관계(캐릭터 곱셈기).** 두 축은 곱으로 복리(금등급 작물 × 멜 고마진 카페 = 둘 다 체감·중복 0)이며, 이게 [ADR-0019] "2축·숫자 비충돌"의 계승이다. 스타듀는 이 둘이 겹치지만(전문직=그냥 돈 더), 우리는 갈라 **둘 다 키울 이유**를 만든다.

### 2. 품질 등급 동인 = 스킬/전문직 주도 (replace, ADR-0033 #3 개정)

채집물·야생 씨앗·작물·어획·채광물의 **품질 등급(4단계)은 이제 스킬 레벨/전문직이 주도한다.** [ADR-0033]의 비-스킬 모델은 아래로 개정:

- **주 동인 = 스킬 레벨 + 전문직**(예: 채집 레벨↑ → 등급 롤↑, 약초학자 → 최고 등급 고정). 스타듀 충실.
- **명부의 운·혼 유인제 부적 = 잔존 미세 보정층으로 강등**(주도권 상실, 폐기는 아님 — portable 성질/전역 운의 세계관 정합은 얇게 유지).
- **종(種) 희소도 = 품질 등급과 분리 유지** — "*무엇을* 줍나"(품종 자체)는 여전히 장소가 결정([ADR-0033] #3 ㉠ 불변). 품질 등급은 "*얼마나 좋은 개체*냐"로, 종과 직교.
- **재배해도 채집 XP·스킬 기반 품질**(스타듀 "밭에서 길러도 채집" 충실 — [ADR-0033] #4 자생 씨앗 계승, 단 동인만 운→스킬).

### 3. 채집 스킬 = 얕은 2갈래 → 전문직 트리로 격상 (ADR-0033 #2 개정)

[ADR-0033] #2가 배제한 Gatherer(더블)·Botanist(이리듐)를 **부활**하되, 살아남았던 **혼 감지**(미혹의 숲 "길 잃음" 테마)·**벌목** 갈래를 트리에 흡수한다. 감지 = 채집꾼 아닌 감지자 갈래로 보존(테마 불변).

### 4. 전문직은 관계 곱셈기·속죄 서사와 직교

전문직 = **플레이어 자신의 노동 숙련**(순수 피지컬/스킬). 관계 곱셈기(캐릭터가 게임 경제를 바꿈)와도, 속죄 서사(죄의 반전=[ADR-0004] 미호 방화→양육 등, 캐릭터 전용)와도 **다른 슬롯**이다. 전투 전문직을 찍어도 바나의 보호 도메인을, 농사 전문직을 찍어도 미호의 자동화를 밟지 않는다(차원이 다름). 전문직은 서사 텍스트 0([ADR-0005] "서사는 캐릭터에만" 정합).

### 5. "평평 ≠ 막힘" 불변 ([ADR-0008] #1)

전문직은 벽이 아니다. 0레벨에서도 모든 활동이 base rate로 돈다. 전문직은 **순수 패시브 우상향**(쓰다 보면 늚)이고, lvl5/10 선택은 *접근*이 아니라 *편의·품질·수량*을 해금한다. 어떤 동작도 레벨로 게이팅하지 않는다.

## 5스킬 전문직 로스터 (나라카 번역)

> **규칙:** lvl5 = 2갈래 중 택1 → lvl10 = 그 갈래가 2분기(스킬당 총 선택 = lvl5 1개 + lvl10 1개). 스타듀 구조 충실, **+판매가/마진 퍼크는 전부 품질/수량/효율로 재조준**(⟲ 표시). 수치는 그레이박스 서랍(스타듀 크기 참고: 2배=20%, +1, 성장 10% 등 — 넉넉히).

### 농사 (Farming)
- **Lv5 · 경작자**(Tiller) — ⟲ *작물 품질 등급 확률↑* (스타듀 +10%가 → 품질로)
- **Lv5 · 목축가**(Rancher) — ⟲ *축산물 품질 등급↑* (스타듀 +20%가 → 품질로)
  - Lv10(경작자) · **장인**(Artisan) — ⟲ *가공품 품질 등급↑ + 가공 속도↑* (스타듀 +40%가 → 품질/효율)
  - Lv10(경작자) · **재배가**(Agriculturist) — 작물 성장 10% 빠름 *(효율, 그대로)*
  - Lv10(목축가) · **둥우리지기**(Coopmaster) — 가축 우정 가속 + 부화 시간 절반 *(편의, 그대로)*
  - Lv10(목축가) · **목자**(Shepherd) — 가축 우정 가속 + 산물 주기 단축 *(효율, 그대로)*

### 채광 (Mining)
- **Lv5 · 광부**(Miner) — 광맥당 광석 +1 *(수량, 그대로)*
- **Lv5 · 지질사**(Geologist) — 보석이 쌍으로 나올 확률 *(수량, 그대로)*
  - Lv10(광부) · **제련공**(Blacksmith) — ⟲ *제련 시간↓ + 잉곳 품질 티어* (스타듀 +50%바가치 → 효율/품질)
  - Lv10(광부) · **탐광자**(Prospector) — 석탄/혼탄 2배 확률 *(수량, 그대로)*
  - Lv10(지질사) · **발굴자**(Excavator) — 지오드 2배 확률 *(수량, 그대로)*
  - Lv10(지질사) · **보석사**(Gemologist) — ⟲ *보석 품질 등급↑* (스타듀 +30%가 → 품질)

### 채집 (Foraging) — ADR-0033 감지/벌목 흡수
- **Lv5 · 감지자**(Forester 위치, ⟲ 재조준) — 혼 감지 범위↑ + 벌목 원목 드롭 +1 *(발견+수량; 스타듀 Forester +50%목재가 → 원목 수량으로)*
- **Lv5 · 채집꾼**(Gatherer) — 채집물 20% 확률 2배 *(수량, 그대로)*
  - Lv10(감지자) · **벌목꾼**(Lumberjack) — 모든 나무에서 단단한 원목 확률 *(자원/수량, 그대로)*
  - Lv10(감지자) · **수액꾼**(Tapper) — ⟲ *수액 품질 등급↑ + 채취 주기 단축* (스타듀 +25%가 → 품질/효율)
  - Lv10(채집꾼) · **약초학자**(Botanist) — 모든 채집물 최고 등급(이리듐) 고정 *(품질, 그대로)*
  - Lv10(채집꾼) · **추적자**(Tracker) — 채집물 위치 화면 표시 *(발견, 그대로 — 미혹의 숲 테마)*

### 낚시 (Fishing)
- **Lv5 · 낚시꾼**(Fisher) — ⟲ *어획 품질 등급 확률↑* (스타듀 +25%가 → 품질)
- **Lv5 · 덫꾼**(Trapper) — 게잡이통 자원/미끼 소모↓ *(편의, 그대로)*
  - Lv10(낚시꾼) · **명조사**(Angler) — ⟲ *최고 등급 어획 확률 대폭↑* (스타듀 +50%가 → 품질)
  - Lv10(낚시꾼) · **보물잡이**(Pirate) — 낚시 보물 상자 2배 확률 *(수량, 그대로)*
  - Lv10(덫꾼) · **뱃사람**(Mariner) — 게잡이통에 잡동사니 안 걸림 *(편의, 그대로)*
  - Lv10(덫꾼) · **미끼장인**(Luremaster) — 게잡이통 미끼 불필요 *(편의, 그대로)*

### 전투 (Combat) — 스타듀에서도 골드 0, 전부 그대로
- **Lv5 · 투사**(Fighter) — 피해 +10% + 최대 체력 +15 *(전투력·[ADR-0011] HP는 곱셈기 밖)*
- **Lv5 · 척후**(Scout) — 크리 확률 +50% *(전투력)*
  - Lv10(투사) · **광전사**(Brute) — 피해 +15% *(전투력)*
  - Lv10(투사) · **수호자**(Defender) — 최대 체력 +25 *(생존)*
  - Lv10(척후) · **곡예사**(Acrobat) — 특수기 쿨다운 −50% *(효율)*
  - Lv10(척후) · **결사**(Desperado) — 크리 위력↑ *(전투력)*

> **재조준(⟲) 합계 = 9개**(Tiller·Rancher·Artisan·Blacksmith·Gemologist·Forester·Tapper·Fisher·Angler). 나머지 **16개는 스타듀 그대로** 부활 — 전문직 정체성의 매력 핵심(2배·이리듐·+1·전투력)이 +판매가 제거로 손상되지 않음의 증거.

## 고려한 대안

- **(A) 전문직 스타듀 그대로(+판매가 포함)** — 멜=마진 등 관계 곱셈기와 정면 충돌 → 관계의 기계적 보상 축을 축소/폐기하고 "관계 = 서사만"으로 → **순수 스타듀 회귀**([ADR-0008]/[ADR-0021] "관계=성장 엔진" 정체성 피벗). owner 기각(2026-07-05, "게임 정체성 피벗").
- **(B) 품질 동인 = 스킬·운/부적 병존(stack)** — 기존 [ADR-0033] 운/부적 층을 대등하게 유지. owner 기각 → **replace**(스킬 주도, 운/부적 강등)로 스타듀 근접·품질 레버 단일화.
- **(C) 채집만 지금 구체화, 나머지 4스킬은 원칙 상속** — "한 시스템씩"([CLAUDE.md]) 규율엔 맞으나, owner가 **5스킬 전면 재설계** 택 → 로스터 전체를 한 결로 잠가 스킬 간 대칭·중복을 지금 정리(개별 그릴 시 표류 방지).
- **(D) [ADR-0019] 유지(전문직 완전 배제)** — 관계 곱셈기 순수 보존이나, owner의 "노동한 만큼 강해지는 RPG 손맛"([ADR-0019] 착수 동기 자체) 요구를 못 채움 → 비-가치 차원 한정으로 양립.

## Consequences

- **[ADR-0019] #1 "전문직 특성 배제" 완화** — 품질/수량/효율/편의 차원은 허용, +판매가/마진만 배제. §13 "숫자 비충돌" 불변식은 *강화*(슬롯 분리를 명시 규칙으로). 2축 골격·"평평≠막힘"·혼력 풀 불변은 유효. → [ADR-0019] 상태 헤더에 "0052 개정" 표기.
- **[ADR-0033] #2·#3 개정** — 채집 스킬 본체(Gatherer/Botanist) 부활, 품질 동인 비-스킬→스킬 주도. #1(줍기=혼력0)·#4(자생 씨앗 발견 게이트)·무대 배정은 불변. → [ADR-0033] 상태 헤더에 "0052 개정" 표기.
- **관계 보상 모델([ADR-0021]/[ADR-0029]) 불변** — 전문직은 +판매가 슬롯을 안 밟으므로 멜=마진·미호=자동화·바나=보호와 직교. 재밸런싱은 "전문직 품질/수량 × 관계 마진"이 이중 폭발하지 않게 그레이박스에서 곡선 조율(우상향 엔진 수 증가 = [ADR-0019] Consequences 밸런싱 부담과 동일 성질).
- **품질 시스템 통일** — 5스킬 산출물(작물·축산·채집·어획·채광) 품질 등급이 공통 4단계·공통 동인(스킬/전문직 + 잔존 운/부적)으로 수렴. 카탈로그 §2-1(품질)·§2-2(채집) 품질 규칙 갱신 필요.
- **야생 씨앗/꽃 패치 표면**(안식 농원) — 야생 씨앗 심기·스프링클러 물주기는 기존 자생 씨앗([ADR-0033])·저승식 스프링클러로 커버. 수확 등급 = 이 ADR의 채집 레벨/전문직. **꿀·벌통 시너지는 지금 defer**(Phase 3 카페 아티산 — 카탈로그 §2-6 참고 기록).
- **그레이박스 스코프** — 전문직 트리 자체의 구현(레벨업·선택 UI·퍼크 적용)은 5스킬 각 시스템 빌드 슬라이스에서. 이 ADR은 *로스터·차원 규칙*을 잠글 뿐, 수치 곡선은 각 슬라이스 그레이박스.
- **되돌리기** — 전문직에 +판매가를 다시 허용하거나(대안 A), 전문직을 전면 폐기([ADR-0019] 원복)하려면 이 ADR을 새 ADR로 대체해야 한다.
